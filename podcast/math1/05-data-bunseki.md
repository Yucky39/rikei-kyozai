# 第5回：データの分析（数学I）

みなみ先生：はい、始まりました「みなみ先生の数学・機械学習ラジオ」。たくみくん、前回の三角比、少しは体に馴染んできた?

たくみ：（笑いながら）まだ頭の中で図を描かないと出てこない感じですけど、少しずつです。

みなみ先生：それでいいよ。さて今日はガラッと変わって「データの分析」。数学Iの最後の単元だよ。

たくみ：統計っぽいですね、図形とは全然違う感じがします。

みなみ先生：ここからは「たくさんの数字の集まりを、少ない数字で要約する」道具を作っていく。共通テストで毎年出るくらい大事な単元だよ。

たくみ：（気合を入れるように）それは気合入れないと。

みなみ先生：大丈夫、考え方はどれも素朴だから。まず代表値から。平均値は分かるよね、全部足して人数で割るやつ。

たくみ：それは分かります。

みなみ先生：それに加えて、中央値と最頻値も代表値の仲間。中央値は、小さい順に並べたときの真ん中の値。個数が偶数なら真ん中2つの平均。最頻値は一番多く出てくる値ね。

たくみ：平均だけで十分な気もするんですけど。

みなみ先生：例えば5人の年収、4人が300万円で1人だけ1億円だったら、平均は極端な1人に引っ張られて跳ね上がるよね。

たくみ：（驚いた声で）あ、1人のせいで変な結論になっちゃう。

みなみ先生：そう、平均値は外れ値の影響を受けやすい。中央値は順位だけ見るから動じにくい。この違いは大事だよ。

たくみ：平均だけ見て安心しちゃダメなんですね。

みなみ先生：次は「四分位数」。データを小さい順に並べて、全体の中央値で下半分・上半分に分ける。個数が奇数のときは、真ん中の値を除いてから分けるのがルール。

たくみ：真ん中は仲間はずれ、と。

みなみ先生：（笑って）そのイメージでいいよ。下半分の中央値が第1四分位数、全体の中央値が第2四分位数、上半分の中央値が第3四分位数。小さい方からキュー1、キュー2、キュー3と呼ぶ。キュー3引くキュー1が「四分位範囲」で、中央付近50パーセントの散らばりを表す。範囲より外れ値の影響を受けにくい指標だよ。

たくみ：最小値・キュー1・キュー2・キュー3・最大値のセット、聞いたことあります。

みなみ先生：それが「五数要約」。これを箱とひげの絵で表したのが箱ひげ図。外れ値の基準もあって、キュー1からキュー3引くキュー1の1.5倍を引いた値より小さいか、キュー3にその1.5倍を足した値より大きいものを外れ値とすることが多い。

たくみ：あの箱の外にポツンと点があるやつですね!

みなみ先生：正解。次は「分散」と「標準偏差」。各データの平均からのずれを偏差と呼ぶけど、偏差をそのまま平均すると必ずゼロになるって知ってた?

たくみ：（少し不安そうに）え、なんでですか?

みなみ先生：プラスとマイナスのずれが打ち消し合うから。だから偏差を2乗してから平均する。それが分散。マイナスを2乗すればプラスになるから、打ち消し合いが起きなくなる。

たくみ：2乗はマイナスを消すためなんですね。

みなみ先生：そう。分散のルートが標準偏差。分散は元のデータを2乗した単位になるから、同じ単位で散らばりを見たいときは標準偏差を使う。それと、分散にはもっと楽な計算式もあって、「2乗の平均」から「平均の2乗」を引いても求まる。偏差の2乗を展開して整理すると、ちゃんとこの形になるんだ。あとで一緒に計算して確かめよう。

たくみ：はい、お願いします。

みなみ先生：次は「変量の変換」。データエックスに対して、ワイイコールエーかけるエックス足すビー、と変換すると、ワイの平均はエーかける元の平均足すビー。ワイの分散はエーの2乗かける元の分散になる。

たくみ：（少し混乱した声で）足したビーは分散に影響しないんですか?

みなみ先生：しないんだ、ここみんなよく間違える。全データに同じ数を足しても間隔は変わらないから分散は不変。エー倍すると偏差自体がエー倍になって、2乗するからエーの2乗倍になる。標準偏差はエーの絶対値倍ね。

たくみ：足し算は平行移動、掛け算は散らばりの拡大縮小ってことですね。

みなみ先生：（嬉しそうに）いいたとえ!最後に2つの変量の関係。エックスとワイの組があるとき、「共分散」はエックスの偏差とワイの偏差を掛け合わせたものの平均。同じ方向に動けばプラス、逆ならマイナスになる。単位に左右されないよう、標準偏差同士で割って正規化したものが「相関係数」。必ずマイナス1からプラス1の間におさまって、1に近いほど強い正の相関、マイナス1に近いほど強い負の相関だよ。

たくみ：0に近いと相関が弱い、でしたっけ。

みなみ先生：正解。ただ大事な注意点があって、相関があっても因果関係があるとは限らない。アイスの売り上げと水の事故件数には相関があるけど、原因はどちらも「気温が高いこと」であって、アイスが事故を起こすわけじゃないよね。

たくみ：（驚いた声で）勝手に原因と結果だと思い込んじゃいけないんですね。

みなみ先生：最後にもうひとつ、「仮説検定」の考え方。ある主張と反対の仮定、「帰無仮説」を立てて、その仮定のもとで観察された結果が起こる確率を見積もる。その確率が基準、例えば5パーセントより小さければ、「めったに起きないことが起きた」と考えて仮定を捨てる。

たくみ：それって背理法っぽいですね。

みなみ先生：（嬉しそうに）まさにそう。「起こりにくいことが起きたなら前提を疑う」、よく似た発想だね。じゃあ数字で確認していこう。9人の小テストの得点、4、5、5、6、7、8、8、9、10点。平均は全部足すと62点、9人で割っておよそ6.9点。

たくみ：9個だから中央値は5番目、7点ですね。

みなみ先生：正解。下半分の4、5、5、6の中央値は5と5の平均で第1四分位数は5点。上半分の8、8、9、10は8と9の平均で第3四分位数は8.5点。じゃあ四分位範囲は?

たくみ：8.5引く5だから、3.5点です!

みなみ先生：正解!次は分散。データ、2、4、6、8、10。平均は6。偏差はマイナス4、マイナス2、0、2、4。

たくみ：2乗すると16、4、0、4、16になります。

みなみ先生：足すと40、5個だから分散は8。標準偏差はルート8、簡単にすると2ルート2。検算もしてみよう。2乗の値、4、16、36、64、100を足すと220、5で割って44。そこから平均の2乗、36を引くと8。一致したね。

たくみ：（驚いた声で）本当に同じ答えになりました!

みなみ先生：最後に相関係数。5人の数学の点、40、50、60、70、80と英語の点、50、40、60、70、80。どちらも平均60。偏差は数学がマイナス20、マイナス10、0、10、20。英語も同じくマイナス10、マイナス20、0、10、20。掛け合わせて足すと900、共分散は180。数学も英語も分散は200。相関係数は180を200で割って0.9。

たくみ：0.9はかなり強い正の相関ですね!

みなみ先生：（笑って）その通り。じゃあ仮説検定の例も。コインを30回投げたら表が21回。公正なコインで30回投げる実験を1000セットしたら、表が21回以上出たのは21セットだった。

たくみ：1000分の21だから、2.1パーセントですか。

みなみ先生：正解。基準の5パーセントより小さいから、「公正なコインでこの結果が起こるのは非常にまれ」と考えて、コインは公正だという仮定を捨てる。だからこのコインは表が出やすいと判断できる。

たくみ：（少し不安そうに）先生、僕、四分位数のところ、この前やらかしちゃって。データが偶数個なのに、奇数のときのやり方で真ん中を除いて計算しちゃったんです。

みなみ先生：ああ、それすごくよくある間違い。個数が奇数か偶数かで手順が変わるから、必ず最初に個数を数える癖をつけてね。あと変量の変換で、足したビーの部分まで分散に影響すると勘違いする人も多いから気をつけて。それと、いくつかのグループを合体させたデータの分散は、各グループの分散の単純な平均にはならない。必ず「2乗の平均引く平均の2乗」に立ち返る必要があるよ。

たくみ：グループごとの分散をただ平均しちゃダメなんですね。

みなみ先生：あと箱ひげ図からは、平均値や、山が2つあるみたいな分布の細かい形は読み取れない。それと仮説検定で、確率が基準より大きかった場合は「主張は間違い」じゃなくて「判断できない、保留する」が正しい言い方だよ。

たくみ：言い方、気をつけます。

みなみ先生：というわけで今日はデータの分析、代表値から四分位数、分散、標準偏差、変量の変換、相関係数、仮説検定まで見てきたね。

たくみ：平均だけじゃなく散らばり方まで見ることで、データの本当の姿が見えてくるんだなと思いました。

みなみ先生：（嬉しそうに）いい感想。そして、たくみくん、これで数学Iの単元、全部終わったんだよ。

たくみ：（驚いた声で）え、本当ですか!長かった……。

みなみ先生：（笑いながら）本当に。数と式から図形と計量、データの分析まで、よく頑張ったね。おめでとう!

たくみ：（照れたように）ありがとうございます、先生のおかげです。

みなみ先生：次からは数学A、新しい世界だよ。次回のテーマは「場合の数と確率」。サイコロやカードで、何通りあるかを数える単元だね。

たくみ：確率、日常でも聞く言葉ですけど、数学として勉強するのは初めてかもです。楽しみです!

みなみ先生：うん、また一緒に頑張ろうね。今日はこのへんで。

たくみ：ありがとうございました!
