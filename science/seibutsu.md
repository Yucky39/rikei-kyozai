# 第1部 生物基礎

生物基礎では、生命現象を「共通性」と「多様性」という視点でとらえ、細胞・代謝・遺伝子・体内環境・免疫・生態系という6つの大きなテーマを学びます。ミクロな分子レベルからマクロな生態系レベルまで、生命のしくみを一貫した論理で理解することが目標です。

## 生物の特徴と細胞

### この単元で学ぶこと
- すべての生物に共通する特徴（細胞・代謝・遺伝情報）を説明できる。
- 原核細胞と真核細胞の構造の違いを整理できる。
- 主な細胞小器官の名称とはたらきを対応づけられる。
- 顕微鏡の各部の名称と正しい操作手順を理解する。
- 細胞の大きさのスケール感（μm 単位）を把握する。

### 解説

**生物の共通性**
地球上の多様な生物は、共通の祖先から進化してきたため、次の共通性をもちます。

| 共通性 | 内容 |
|---|---|
| 細胞からなる | すべての生物のからだは細胞を基本単位とする |
| 遺伝情報として DNA をもつ | 形質を子孫に伝える物質は DNA |
| 代謝を行う | エネルギーを利用して物質を合成・分解する |
| ATP を利用する | エネルギーの受け渡しに ATP を用いる |
| 体内環境を保つ | 内部環境を一定に保とうとする（恒常性） |
| 進化する | 世代を重ねて変化する |

生物には**多様性**もあり、現在確認されている種は約190万種にのぼります。共通性は「共通の祖先」に、多様性は「進化・適応」に由来します。

**細胞の発見と細胞説**
- 1665年 **フック**：コルクの細胞（cell）を観察。
- **シュライデン**（植物, 1838）・**シュワン**（動物, 1839）：細胞説を提唱。
- **フィルヒョー**（1858）：「すべての細胞は細胞から生じる」。

**原核細胞と真核細胞**

| 項目 | 原核細胞 | 真核細胞 |
|---|---|---|
| 核膜に包まれた核 | ない | ある |
| DNA | 細胞質中に存在 | 核内に存在 |
| ミトコンドリア | ない | ある |
| 葉緑体 | ない | 植物細胞にある |
| 大きさ | 小さい（1〜数μm） | 大きい（10〜100μm） |
| 例 | 細菌（大腸菌）・シアノバクテリア | 動物・植物・菌類・原生生物 |

**主な細胞小器官のはたらき**

| 細胞小器官 | はたらき | 動物 | 植物 |
|---|---|:---:|:---:|
| 核 | DNA を含み、生命活動を統御 | ○ | ○ |
| ミトコンドリア | 呼吸によって ATP を合成 | ○ | ○ |
| 葉緑体 | 光合成を行う | × | ○ |
| 小胞体 | 物質の輸送 | ○ | ○ |
| ゴルジ体 | 物質の分泌・修飾 | ○ | ○ |
| リボソーム | タンパク質の合成 | ○ | ○ |
| 液胞 | 物質の貯蔵・浸透圧調節（発達） | △ | ○ |
| 細胞壁 | 細胞の保護・形の保持（セルロース） | × | ○ |
| 細胞膜 | 物質の出入りを調節（リン脂質二重層） | ○ | ○ |

ミトコンドリアと葉緑体は、独自の DNA をもち二重膜構造をとることから、かつて共生した好気性細菌・シアノバクテリアに由来するという**細胞内共生説**（マーグリス）が有力です。

**顕微鏡**
- **接眼レンズ → 対物レンズ** の順で総合倍率＝接眼倍率×対物倍率。
- ピント合わせは、対物レンズをプレパラートから**遠ざけながら**行う（衝突防止）。
- 低倍率で観察対象を視野の中央に入れてから高倍率にする。
- 倍率を上げると視野は狭く、暗くなる。

### 例題

**例題1** 次のうち、原核細胞にはなく真核細胞にのみ見られる構造をすべて選べ。
(ア) 細胞膜　(イ) 核膜　(ウ) ミトコンドリア　(エ) リボソーム　(オ) DNA

**解答** (イ)・(ウ)
リボソームは原核細胞にも存在する。DNA・細胞膜は両方にある。核膜とミトコンドリアは真核細胞にのみ存在する。

**例題2** 接眼レンズ10倍、対物レンズ40倍のときの総合倍率を求めよ。

**解答** 10 × 40 ＝ **400倍**

### 練習問題
1. 細胞説を動物について提唱した人物名を答えよ。
2. 光合成を行う細胞小器官の名称を答えよ。
3. 原核生物の例を1つ挙げよ。
4. 細胞内共生説によれば、ミトコンドリアはどのような生物に由来すると考えられるか。
5. 顕微鏡でピントを合わせるとき、対物レンズはプレパラートに近づけながら合わせるか、遠ざけながら合わせるか。
6. 植物細胞にあって動物細胞にない構造を2つ挙げよ。

### 解答と解説
1. **シュワン**。植物についてはシュライデンが提唱した。
2. **葉緑体**。内部のチラコイドで光を吸収する。
3. **大腸菌**（乳酸菌・シアノバクテリアなども可）。核膜をもたない。
4. **好気性細菌**。葉緑体はシアノバクテリアに由来する。いずれも独自の DNA と二重膜をもつ点が根拠。
5. **遠ざけながら**合わせる。近づけながらだとレンズがプレパラートに衝突する危険がある。
6. **細胞壁**と**葉緑体**（発達した液胞も可）。

### つまずきやすいポイント
- 「原核細胞に細胞膜はない」は誤り。**細胞膜はすべての細胞にある**。ないのは核膜。
- リボソームは細胞小器官だが膜をもたず、原核細胞にも存在する。
- 「細胞壁があるのは植物だけ」は不正確。細菌や菌類も細胞壁をもつ（成分は異なる）。植物のセルロースに対し、細菌はペプチドグリカン。
- μm（マイクロメートル）＝1/1000 mm。細胞の大きさの単位を mm と混同しない。

## 代謝とエネルギー

### この単元で学ぶこと
- 代謝を同化と異化に分けて説明できる。
- ATP の構造とエネルギー通貨としての役割を理解する。
- 酵素の性質（基質特異性・最適温度・最適 pH）を説明できる。
- 光合成と呼吸の全体像を反応式で書ける。

### 解説

**代謝**とは生体内の化学反応全体のこと。エネルギーの流れを伴い、次の2つに大別されます。

| 種類 | 内容 | エネルギー | 例 |
|---|---|---|---|
| 同化 | 単純な物質から複雑な物質を合成 | 吸収 | 光合成・窒素同化 |
| 異化 | 複雑な物質を単純な物質に分解 | 放出 | 呼吸・発酵 |

**ATP（アデノシン三リン酸）**
- 構造：アデノシン（アデニン＋リボース）＋ **リン酸3個**。
- リン酸どうしの結合を**高エネルギーリン酸結合**という。
- ATP ⇄ ADP ＋ リン酸（Pi）＋ エネルギー。この分解・合成によってエネルギーを受け渡す。
- ATP は「生体内のエネルギーの通貨」と呼ばれる。

**酵素**
- 生体内の化学反応を触媒する**タンパク質**。反応の活性化エネルギーを下げる。
- **基質特異性**：特定の基質にのみ作用する（鍵と鍵穴の関係）。
- **最適温度**：ヒトの酵素は約35〜40℃。高温では変性して失活する。
- **最適 pH**：ペプシン=約2（強酸性）、アミラーゼ=約7（中性）、トリプシン=約8（弱塩基性）。
- 触媒として反応の前後で自身は変化せず、繰り返しはたらく。

**光合成と呼吸の概観**

光合成（葉緑体）:
```
6CO₂ ＋ 12H₂O ＋ 光エネルギー → C₆H₁₂O₆ ＋ 6O₂ ＋ 6H₂O
```

呼吸（ミトコンドリア）:
```
C₆H₁₂O₆ ＋ 6O₂ ＋ 6H₂O → 6CO₂ ＋ 12H₂O ＋ エネルギー（ATP）
```

光合成は同化（エネルギー吸収）、呼吸は異化（エネルギー放出）で、互いにほぼ逆の反応です。

### 例題

**例題1** ATP を構成する3つの要素を答えよ。

**解答** アデニン（塩基）・リボース（糖）・リン酸。アデニンとリボースが結合したものをアデノシンといい、これにリン酸が3個結合している。

**例題2** 酵素が高温で失活するのはなぜか。20字程度で述べよ。

**解答** 主成分のタンパク質が熱で変性し、立体構造が壊れるため。

### 練習問題
1. 光合成は同化・異化のどちらか。
2. ATP が ADP に分解されるとき、放出されるか吸収されるかを答えよ。
3. 胃液に含まれる酵素ペプシンの最適 pH はおよそいくつか。
4. 酵素が特定の基質にのみはたらく性質を何というか。
5. 呼吸を行う細胞小器官の名称を答えよ。
6. 酵素の主成分は何という物質か。

### 解答と解説
1. **同化**。CO₂ と H₂O から有機物を合成し、光エネルギーを吸収する。
2. **放出される**。ATP → ADP ＋ Pi の分解でエネルギーが取り出される。
3. **約2**（強酸性）。胃内の強酸性環境で最もよくはたらく。
4. **基質特異性**。酵素の活性部位の立体構造が特定の基質と合致する。
5. **ミトコンドリア**。酸素を用いて有機物を分解し ATP を合成する。
6. **タンパク質**。そのため熱や pH で変性しやすい。

### つまずきやすいポイント
- 「ATP はエネルギーそのもの」ではなく、**エネルギーを一時的に蓄えて運ぶ分子**。エネルギーは結合に蓄えられている。
- 酵素は反応で消費されない（触媒）ため、少量で繰り返しはたらく。
- 最適温度を超えると反応速度は下がる。「温度が高いほど速い」は誤り。変性は不可逆。
- 光合成と呼吸は逆反応に見えるが、経路（反応の道すじ）はまったく異なる。

## 遺伝子とそのはたらき

### この単元で学ぶこと
- DNA の二重らせん構造と塩基の相補性を説明できる。
- 遺伝情報が DNA の塩基配列にあることを理解する。
- 半保存的複製のしくみを説明できる。
- 遺伝子の発現（転写・翻訳）の概要を理解する。
- 細胞周期と体細胞分裂の過程を順序立てて説明できる。

### 解説

**DNA の構造**
- **ヌクレオチド**（リン酸＋糖（デオキシリボース）＋塩基）が構成単位。
- 塩基は4種類：**アデニン(A)・チミン(T)・グアニン(G)・シトシン(C)**。
- **相補性**：A は T と、G は C と対をつくる（水素結合）。
- ワトソンとクリック（1953）が**二重らせん構造**を提唱。
- **シャルガフの規則**：A の量 ＝ T の量、G の量 ＝ C の量。

**遺伝情報**
遺伝情報は DNA の**塩基配列**として保存される。1つの遺伝子は原則1種類のタンパク質の情報をもつ。

**DNA の複製（半保存的複製）**
2本鎖がほどけ、それぞれが鋳型となって新しい鎖が合成される。新しい DNA は「もとの鎖1本＋新しい鎖1本」からなる。メセルソンとスタールが実験で証明。

**遺伝子とタンパク質（発現）**
```
DNA → (転写) → RNA → (翻訳) → タンパク質
```
これを**セントラルドグマ**という。RNA の塩基はチミン(T)の代わりに**ウラシル(U)**を用いる。連続する3つの塩基（コドン）が1つのアミノ酸を指定する。

**細胞周期**

| 時期 | 内容 |
|---|---|
| G₁期 | DNA 合成準備期 |
| S期 | DNA 合成（複製）期 |
| G₂期 | 分裂準備期 |
| M期 | 分裂期（前期・中期・後期・終期） |

G₁・S・G₂ をまとめて**間期**という。DNA 量は S 期で2倍になる。

**体細胞分裂の過程**

| 時期 | 特徴 |
|---|---|
| 前期 | 染色体が凝縮して太く短くなる。核膜が消える |
| 中期 | 染色体が赤道面に並ぶ |
| 後期 | 染色体が縦裂し、両極へ移動 |
| 終期 | 核膜が再形成され、細胞質が分裂 |

体細胞分裂では、分裂前後で染色体数は**変化しない**。

### 例題

**例題1** ある DNA の一方の鎖の塩基配列が 5'-ATGCCA-3' のとき、相補的な鎖の配列を答えよ（向きも示せ）。

**解答** 3'-TACGGT-5'。A↔T、G↔C の相補性による。

**例題2** DNA 中でアデニンが全体の30%を占めるとき、グアニンの割合は何%か。

**解答** A＝T＝30% なので A＋T＝60%。残り40%が G＋C で、G＝C だから **G＝20%**。

### 練習問題
1. DNA の二重らせん構造を提唱した2人の科学者名を答えよ。
2. RNA でチミンの代わりに用いられる塩基は何か。
3. DNA の複製様式を何というか。
4. 細胞周期のうち DNA が合成される時期を何というか。
5. 体細胞分裂で、染色体が赤道面に並ぶのは前期・中期・後期のどれか。
6. アミノ酸1個を指定する、mRNA 上の連続した3つの塩基の並びを何というか。

### 解答と解説
1. **ワトソンとクリック**。1953年にX線回折データをもとに提唱した。
2. **ウラシル(U)**。RNA では T のかわりに U が A と対をつくる。
3. **半保存的複製**。もとの1本鎖を鋳型に新しい鎖をつくるため、複製後は片方がもとの鎖。
4. **S期**。この時期に DNA 量が2倍になる。
5. **中期**。前期で凝縮し、中期で赤道面に整列、後期で両極へ分かれる。
6. **コドン**。64通りのコドンが20種類のアミノ酸などを指定する。

### つまずきやすいポイント
- 「DNA の複製で新しい DNA は全部新しい鎖でできる」は誤り。半保存的なので**必ず片方はもとの鎖**。
- 転写でできるのは RNA、翻訳でできるのはタンパク質。順序と生成物を混同しない。
- 体細胞分裂では染色体数は変わらない。数が半分になるのは減数分裂（第2部）。
- 「間期は細胞が休んでいる時期」ではない。DNA 複製など活発な準備が行われている。

## 体内環境の維持

### この単元で学ぶこと
- 体液の種類と恒常性（ホメオスタシス）の意味を理解する。
- 腎臓と肝臓のはたらきを説明できる。
- 血糖濃度・体温の調節のしくみを説明できる。
- 自律神経系とホルモンの協調的なはたらきを理解する。
- フィードバック調節のしくみを説明できる。

### 解説

**体液と恒常性**
体液は**血液・組織液・リンパ液**の3つ。細胞をとりまく液体環境を**体内環境（内部環境）**という。これを一定に保つ性質を**恒常性（ホメオスタシス）**という。調節の中心は**間脳の視床下部**。

**腎臓**
- 血液をろ過して尿をつくり、老廃物（尿素など）を排出。
- 構造単位は**ネフロン（腎単位）**＝腎小体（糸球体＋ボーマンのう）＋細尿管。
- **ろ過**（糸球体→ボーマンのう）と**再吸収**（細尿管・集合管）で尿を生成。グルコースは通常すべて再吸収される。

**肝臓**
- 有害なアンモニアを毒性の低い**尿素**につくりかえる（尿素回路）。
- グルコースを**グリコーゲン**として貯蔵・分解。
- **胆汁**を生成（脂肪の消化を助ける）。
- 解毒作用、血しょうタンパク質の合成、体温の維持。

**血糖濃度の調節**
血糖濃度は約0.1%（100 mg/100 mL）に保たれる。

| 状態 | ホルモン | 分泌器官 | はたらき |
|---|---|---|---|
| 高血糖 | インスリン | すい臓ランゲルハンス島B細胞 | 血糖を下げる（グリコーゲン合成促進） |
| 低血糖 | グルカゴン | すい臓ランゲルハンス島A細胞 | 血糖を上げる（グリコーゲン分解） |
| 低血糖 | アドレナリン | 副腎髄質 | 血糖を上げる |
| 低血糖 | 糖質コルチコイド | 副腎皮質 | 血糖を上げる（タンパク質から糖新生） |

血糖を下げるホルモンは**インスリンのみ**である点に注意。

**体温の調節（寒いとき）**
視床下部が感知 → 交感神経・ホルモン（チロキシン、アドレナリン）→ 代謝促進・発熱量増加、皮膚血管収縮で放熱抑制。

**自律神経系**

| 神経 | はたらき（例） |
|---|---|
| 交感神経 | 興奮・緊張時に優位。心拍数↑、瞳孔散大、消化抑制 |
| 副交感神経 | 安静・休息時に優位。心拍数↓、瞳孔縮小、消化促進 |

**フィードバック**
最終産物や結果が前の段階にはたらきかける調節。特に結果を打ち消す方向にはたらくものを**負のフィードバック**という（例：チロキシンが増えると視床下部・脳下垂体を抑制）。

### 例題

**例題1** 血糖濃度を低下させるホルモンの名称と分泌部位を答えよ。

**解答** インスリン。すい臓ランゲルハンス島のB細胞から分泌される。

**例題2** 肝臓が有害なアンモニアをつくりかえる、毒性の低い物質は何か。

**解答** 尿素。

### 練習問題
1. 恒常性の調節中枢がある脳の部位を答えよ。
2. 腎臓の構造上・機能上の単位を何というか。
3. 血糖を上昇させるホルモンを2つ挙げよ。
4. 安静時に優位になる自律神経は何か。
5. チロキシンを分泌する内分泌腺はどこか。
6. 尿の生成過程で、グルコースはどうなるか。

### 解答と解説
1. **間脳の視床下部**。自律神経系と内分泌系の両方を統御する。
2. **ネフロン（腎単位）**。腎小体と細尿管からなる。
3. **グルカゴン・アドレナリン**（糖質コルチコイドも可）。血糖を下げるのはインスリンのみ。
4. **副交感神経**。心拍を抑え消化を促進する。
5. **甲状腺**。チロキシンは代謝を促進する。
6. **すべて再吸収される**ため、健康なら尿中には出ない。糖が出る場合は糖尿病が疑われる。

### つまずきやすいポイント
- 血糖を**下げる**ホルモンはインスリンだけ。上げるホルモンは複数ある（からだにとって低血糖の方が危険なため）。
- 交感神経＝興奮、副交感神経＝休息。心拍数への作用が逆であることに注意。
- 尿素をつくるのは肝臓、排出するのは腎臓。役割を混同しない。
- 負のフィードバックは「結果を打ち消す」方向。増えたら抑え、減ったら促す。

## 免疫

### この単元で学ぶこと
- 生体防御の3段階と自然免疫・獲得免疫の違いを説明できる。
- 体液性免疫と細胞性免疫のしくみを区別できる。
- 免疫記憶と予防接種の関係を理解する。
- 免疫にかかわる医療とアレルギー・免疫の異常を説明できる。

### 解説

**生体防御の段階**
1. **物理的・化学的防御**：皮膚、粘膜、涙・だ液のリゾチーム、胃酸など。
2. **自然免疫**：食細胞（好中球・マクロファージ・樹状細胞）による食作用、NK細胞。
3. **獲得免疫（適応免疫）**：リンパ球（T細胞・B細胞）が特定の異物（抗原）を認識して攻撃。

**自然免疫と獲得免疫**

| 項目 | 自然免疫 | 獲得免疫 |
|---|---|---|
| 反応の速さ | 速い | 遅い（数日） |
| 特異性 | 低い（幅広く対応） | 高い（抗原特異的） |
| 免疫記憶 | ない | ある |
| 主な細胞 | 食細胞・NK細胞 | T細胞・B細胞 |

**獲得免疫の2つの型**

| 型 | 中心となる細胞 | しくみ |
|---|---|---|
| 体液性免疫 | B細胞 | 抗体（免疫グロブリン）を産生し抗原と結合（抗原抗体反応） |
| 細胞性免疫 | キラーT細胞 | 感染細胞やがん細胞を直接攻撃 |

**流れ（概略）**
樹状細胞が抗原を取り込み、その情報を**ヘルパーT細胞**に提示（抗原提示）→ ヘルパーT細胞が B細胞やキラーT細胞を活性化。B細胞は**形質細胞**に分化して抗体を放出する。

**免疫記憶**
一度侵入した抗原の情報は**記憶細胞**として保存される。同じ抗原が再び侵入すると、より速く強い反応（**二次応答**）が起こる。これを利用したのが**予防接種（ワクチン）**。

**免疫と医療・異常**

| 事項 | 内容 |
|---|---|
| 予防接種 | 弱毒化した抗原を接種し記憶細胞をつくる |
| 血清療法 | 抗体を含む血清を注射（即効性、治療） |
| アレルギー | 免疫が過剰に反応（花粉症など）。重篤なものをアナフィラキシー |
| 自己免疫疾患 | 自分の組織を攻撃（関節リウマチなど） |
| エイズ(AIDS) | HIV がヘルパーT細胞を破壊し免疫不全に |

### 例題

**例題1** 体液性免疫で、抗体を産生する細胞は何か。

**解答** B細胞が分化した**形質細胞（抗体産生細胞）**。

**例題2** 予防接種と血清療法の違いを、「記憶細胞」「抗体」の語を用いて説明せよ。

**解答** 予防接種は弱毒化抗原を接種して自分自身に記憶細胞をつくらせ、将来の感染を予防する。血清療法は他の生物がつくった抗体を直接注射し、すでに侵入した病原体を治療する。

### 練習問題
1. 皮膚や粘膜による防御は、免疫の第何段階の防御か。
2. 抗体の別名（タンパク質としての名称）を答えよ。
3. 感染細胞を直接攻撃するリンパ球を何というか。
4. 一度侵入した抗原に対して速く強く起こる免疫反応を何というか。
5. HIV が破壊する免疫細胞は何か。
6. 花粉症のように免疫が過剰に反応する現象を何というか。

### 解答と解説
1. **第1段階（物理的・化学的防御）**。免疫の前の、体内への侵入を防ぐ段階。
2. **免疫グロブリン**。抗原と特異的に結合する（抗原抗体反応）。
3. **キラーT細胞**。細胞性免疫の中心。
4. **二次応答**。記憶細胞によって起こる。予防接種はこれを利用する。
5. **ヘルパーT細胞**。破壊されると獲得免疫全体が機能しなくなる（免疫不全）。
6. **アレルギー**。重篤な全身反応をアナフィラキシーという。

### つまずきやすいポイント
- 抗原（侵入した異物）と抗体（それに結合するタンパク質）を混同しない。
- 自然免疫には**免疫記憶がない**。記憶があるのは獲得免疫。
- 予防接種は「抗原」を入れる（自分で抗体をつくる）、血清療法は「抗体」を入れる。
- ヘルパーT細胞は自分では攻撃しないが、司令塔として全体を活性化する要。

## 植生と生態系

### この単元で学ぶこと
- 植生の遷移の過程を順序立てて説明できる。
- 気候とバイオームの対応を理解する。
- 生態系における物質循環とエネルギーの流れを区別できる。
- 炭素循環・窒素循環のあらましを説明できる。
- 生態系のバランスと保全の重要性を理解する。

### 解説

**植生の遷移**
ある場所の植生が時間とともに移り変わること。

一次遷移（裸地から）:
```
裸地 → 荒原（地衣類・コケ）→ 草原 → 低木林 → 陽樹林 → 混交林 → 陰樹林（極相）
```
- **先駆種（パイオニア）**：地衣類・コケ植物など、栄養の乏しい環境に定着。
- **極相（クライマックス）**：遷移の最終的に安定した状態。日本の多くでは陰樹林。
- 火山噴火跡などからの遷移が一次遷移、山火事跡・耕作放棄地からのものが二次遷移（土壌や種子が残るため速い）。

**バイオーム（生物群系）**
気温と降水量によって決まる植生と動物のまとまり。

| 気候帯 | バイオーム | 例 |
|---|---|---|
| 熱帯・多雨 | 熱帯多雨林 | 常緑広葉樹 |
| 温帯・多雨 | 照葉樹林 | シイ・カシ（日本の暖温帯） |
| 温帯 | 夏緑樹林 | ブナ・ミズナラ（冷温帯） |
| 亜寒帯 | 針葉樹林 | トウヒ・モミ |
| 乾燥 | 砂漠・ステップ | サボテン・イネ科草本 |
| 寒帯 | ツンドラ | 地衣類・コケ |

**物質循環とエネルギーの流れ**
- **物質（炭素・窒素など）は循環する**（生産者→消費者→分解者→無機物→生産者…）。
- **エネルギーは循環せず一方向に流れる**（太陽光 → 化学エネルギー → 最終的に熱として放出）。

**炭素循環**
大気中の CO₂ → 生産者が光合成で有機物に固定 → 食物連鎖で移動 → 呼吸・分解で CO₂ として大気へ戻る。化石燃料の燃焼で大気 CO₂ が増加し、地球温暖化の原因となる。

**窒素循環**
- **窒素固定**：大気中の N₂ を根粒菌・アゾトバクターなどがアンモニウムイオンに変える。
- 植物が窒素同化でタンパク質などを合成。
- 遺体・排出物は分解され、硝化菌が硝酸イオンに変える。
- **脱窒**：脱窒素細菌が硝酸イオンなどを N₂ に戻し大気へ。

**生態系のバランスと保全**
生態系は多少の変動があっても回復する力（復元力）をもつが、限界を超えると崩れる。外来生物の侵入、生息地の破壊、地球温暖化などが生物多様性を脅かす。**キーストーン種**の減少は生態系全体に大きな影響を与える。

### 例題

**例題1** 一次遷移で最初に定着する生物（先駆種）の例を1つ挙げよ。

**解答** 地衣類（またはコケ植物）。栄養や水の乏しい裸地でも生育できる。

**例題2** 「物質は循環するが、エネルギーは循環しない」。エネルギーは最終的に何になって生態系から出ていくか。

**解答** 熱（熱エネルギー）として放出される。

### 練習問題
1. 遷移の最終的に安定した状態を何というか。
2. 日本の暖温帯に発達する、シイやカシからなるバイオームを何というか。
3. 大気中の窒素をアンモニウムイオンに変えるはたらきを何というか。
4. 脱窒素細菌が硝酸イオンなどを最終的に変える気体は何か。
5. 生態系で、生産者が無機物から有機物を合成するはたらきは何か。
6. 森林の遷移で、極相林を構成しやすいのは陽樹・陰樹のどちらか。

### 解答と解説
1. **極相（クライマックス）**。日本では多くが陰樹林になる。
2. **照葉樹林**。冬でも葉を落とさない常緑広葉樹が中心。
3. **窒素固定**。根粒菌やシアノバクテリアなどが行う。
4. **窒素（N₂）**。脱窒により大気へ戻る。
5. **光合成**（同化）。生態系の物質・エネルギーの入口。
6. **陰樹**。日陰でも芽生えが育つため、最終的に優占する。

### つまずきやすいポイント
- 「エネルギーも循環する」は誤り。物質は循環するが**エネルギーは一方向**（最終的に熱で失われる）。
- 一次遷移と二次遷移の違いは、土壌や種子が残っているかどうか。二次遷移の方が速い。
- 窒素固定と窒素同化を混同しない。固定は N₂→NH₄⁺、同化は無機窒素→有機窒素化合物。
- 極相は「変化が止まる」のではなく、全体として構成が安定している状態を指す。

# 第2部 生物

生物では、生物基礎で学んだ内容をより分子・細胞レベルで深く掘り下げ、生命現象のしくみを化学反応や遺伝情報の視点で詳しく学びます。さらに、発生・環境応答・生態・進化と系統といった、生命の営みと歴史全体を体系的に理解します。

## 生命現象と物質

### この単元で学ぶこと
- タンパク質の構造（一次〜四次）を説明できる。
- 酵素の性質と反応速度の関係を理解する。
- 呼吸を解糖系・クエン酸回路・電子伝達系の3段階で説明できる。
- 発酵のしくみと種類を理解する。
- 光合成をチラコイド反応とカルビン回路に分けて説明できる。
- 窒素同化のしくみを説明できる。

### 解説

**タンパク質の構造**

| 構造 | 内容 |
|---|---|
| 一次構造 | アミノ酸の配列順序（ペプチド結合による） |
| 二次構造 | αヘリックス、βシート（水素結合による） |
| 三次構造 | ポリペプチド鎖全体の立体構造（S-S結合など） |
| 四次構造 | 複数のポリペプチド鎖（サブユニット）の集合（例：ヘモグロビン） |

タンパク質は約20種類のアミノ酸がペプチド結合でつながったもの。立体構造が壊れることを**変性**という。

**酵素の反応速度**
- 基質濃度を上げると反応速度は増すが、やがて頭打ちになる（すべての酵素が基質と結合＝飽和）。
- **競争的阻害**：基質に似た物質が活性部位を奪う（基質濃度を上げると阻害が弱まる）。
- **非競争的阻害**：活性部位以外に結合し立体構造を変える。
- **アロステリック酵素**：最終産物が結合して活性を調節（フィードバック調節）。

**呼吸の3段階**

| 段階 | 場所 | 反応の概要 | ATP |
|---|---|---|---|
| 解糖系 | 細胞質基質 | グルコース→ピルビン酸2分子 | 差引2 |
| クエン酸回路 | ミトコンドリアマトリックス | ピルビン酸→CO₂、NADH等生成 | 2 |
| 電子伝達系 | ミトコンドリア内膜 | 酸素を用いて多量の ATP を合成 | 最大34 |

グルコース1分子から最大約**38分子（教科書により約34）**の ATP が生じる。電子伝達系で酸素が最終的に電子を受け取り水になる。

**発酵**（酸素を使わない異化）

| 種類 | 生成物 | 例 |
|---|---|---|
| アルコール発酵 | エタノール＋CO₂ | 酵母 |
| 乳酸発酵 | 乳酸 | 乳酸菌・筋肉（無酸素時） |

発酵の ATP は解糖系由来の**わずか2分子**。

**光合成の2段階**

| 段階 | 場所 | 内容 |
|---|---|---|
| チラコイド反応（明反応） | チラコイド膜 | 光で水を分解し O₂ 発生、ATP・NADPH 生成 |
| カルビン回路（暗反応） | ストロマ | ATP・NADPH を使い CO₂ を固定して有機物合成 |

カルビン回路では CO₂ が RuBP と結合し、最終的にグルコースがつくられる。

**窒素同化**
植物が無機窒素（NO₃⁻・NH₄⁺）を取り込み、アミノ酸・タンパク質・核酸などの有機窒素化合物を合成する過程。

### 例題

**例題1** 呼吸で最も多くの ATP を生成する段階と、その場所を答えよ。

**解答** 電子伝達系。ミトコンドリアの内膜で行われる。

**例題2** 酵母によるアルコール発酵の生成物を2つ答えよ。

**解答** エタノールと二酸化炭素（CO₂）。

### 練習問題
1. タンパク質のアミノ酸配列順序を示す構造を何というか。
2. 解糖系が行われる細胞内の場所はどこか。
3. 光合成で酸素が発生するのは、水が分解される反応による。この反応が起こる場所はどこか。
4. 基質に似た物質が活性部位に結合して起こる酵素の阻害を何というか。
5. カルビン回路で CO₂ を最初に固定する物質は何か。
6. 乳酸発酵で生成される物質は何か。

### 解答と解説
1. **一次構造**。ペプチド結合による配列で、これが立体構造を決める。
2. **細胞質基質（サイトゾル）**。ミトコンドリアの外で酸素を使わずに進む。
3. **チラコイド膜**（チラコイド反応）。光エネルギーで水が分解され O₂ が発生。
4. **競争的阻害**。基質濃度を高めると阻害の影響は小さくなる。
5. **RuBP（リブロースビスリン酸）**。CO₂ と結合して固定が始まる。
6. **乳酸**。激しい運動時の筋肉でも起こる。

### つまずきやすいポイント
- 光合成のチラコイド反応で発生する酸素は、**CO₂ ではなく水（H₂O）**が分解されてできる。
- 呼吸のうち多量の ATP を生むのは電子伝達系。解糖系・クエン酸回路だけでは少ない。
- 発酵は解糖系までしか進まないので ATP 収量が少ない（2分子）。
- 「暗反応」は光がなくても進むという意味だが、実際には明反応の産物（ATP・NADPH）を必要とするので、光がないと続かない。

## 遺伝情報の発現と調節

### この単元で学ぶこと
- 転写と翻訳の詳しいしくみを説明できる。
- 原核生物と真核生物の遺伝子発現調節の違いを理解する。
- オペロン説と選択的スプライシングを説明できる。
- 主なバイオテクノロジー（PCR・遺伝子組換え・電気泳動）のしくみを理解する。

### 解説

**転写**
DNA の塩基配列を鋳型に、RNA ポリメラーゼが mRNA を合成する。DNA の A・T・G・C に対し、RNA では U・A・C・G が対応する（T→A ではなく A→U）。

**翻訳**
mRNA のコドンに従って、リボソームでアミノ酸がつながりタンパク質ができる。**tRNA（転移RNA）**が対応するアミノ酸を運ぶ。tRNA の**アンチコドン**が mRNA のコドンと相補的に結合する。開始コドンは AUG（メチオニン）。

**原核生物の発現調節：オペロン**
関連する複数の遺伝子がまとまって調節される単位を**オペロン**という。**ラクトースオペロン**（ジャコブとモノー）が有名。ラクトースがないときは調節タンパク質（リプレッサー）がオペレーターに結合して転写を抑制。ラクトースがあると抑制が外れて分解酵素の遺伝子が発現する。

**真核生物の発現調節**
- DNA はヒストンに巻きついて**クロマチン**を形成し、その凝縮状態で発現が調節される。
- **転写調節領域**に調節タンパク質（転写調節因子）が結合。
- **選択的スプライシング**：転写後、mRNA 前駆体から**イントロン**が除かれ**エキソン**がつながる。エキソンの選び方を変えることで、1つの遺伝子から複数種類のタンパク質ができる。

**バイオテクノロジー**

| 技術 | 内容 |
|---|---|
| PCR法 | DNA を試験管内で大量に増幅。加熱で2本鎖を解離→プライマー結合→DNAポリメラーゼで伸長を繰り返す。耐熱性ポリメラーゼを使用 |
| 制限酵素 | DNA を特定の塩基配列で切断する「はさみ」 |
| DNAリガーゼ | DNA 断片をつなぐ「のり」 |
| 遺伝子組換え | ベクター（プラスミドなど）に目的遺伝子を組み込み、細胞に導入 |
| 電気泳動 | DNA 断片を大きさ（長さ）で分離。DNA は負に帯電し陽極へ移動、短いものほど速く進む |

### 例題

**例題1** DNA の鋳型鎖が 3'-TACGGA-5' のとき、転写でできる mRNA の配列を5'→3'で答えよ。

**解答** 5'-AUGCCU-3'。T→A、A→U、C→G、G→C。RNA なので T ではなく U。

**例題2** PCR法で DNA を1サイクルごとに2倍に増やすとき、3サイクル後には理論上何倍になるか。

**解答** 2³ ＝ **8倍**。

### 練習問題
1. mRNA のコドンに対応してアミノ酸を運ぶ RNA を何というか。
2. タンパク質合成の開始を指定する開始コドンは何か。
3. 真核生物の mRNA 前駆体から除かれる部分を何というか。
4. ラクトースオペロンの研究で知られる2人の科学者名を答えよ。
5. DNA を特定の配列で切断する酵素を何というか。
6. 電気泳動で DNA が移動する電極はどちらか（正極・負極）。

### 解答と解説
1. **tRNA（転移RNA）**。アンチコドンでコドンと結合する。
2. **AUG**。メチオニンを指定する。
3. **イントロン**。残ってつながる部分がエキソン。
4. **ジャコブとモノー**。オペロン説を提唱した。
5. **制限酵素**。特定の塩基配列を認識して切る。
6. **正極（陽極）**。DNA はリン酸により負に帯電しているため陽極へ引かれる。

### つまずきやすいポイント
- 転写での対応は A→U（T ではない）。DNA の複製と混同しない。
- スプライシングで残るのは**エキソン**、除かれるのが**イントロン**。名前が紛らわしい。
- PCR は指数関数的に増える（2ⁿ）。n サイクルで理論上2ⁿ倍。
- オペロンは主に原核生物のしくみ。真核生物ではそのままの形では見られない。

## 生殖と発生

### この単元で学ぶこと
- 減数分裂と配偶子形成のしくみを説明できる。
- 受精と初期発生（卵割）の過程を理解する。
- 誘導と形成体のはたらきを説明できる。
- 母性因子とホメオティック遺伝子の役割を理解する。
- 植物の発生の特徴を知る。

### 解説

**減数分裂**
生殖細胞（配偶子）をつくる分裂。連続する2回の分裂（第一分裂・第二分裂）で、染色体数が**半分（2n→n）**になる。第一分裂前期に**相同染色体が対合**し、**乗換え**によって遺伝子の組換えが起こる（多様性の源）。

| 項目 | 体細胞分裂 | 減数分裂 |
|---|---|---|
| 分裂回数 | 1回 | 2回 |
| 娘細胞数 | 2個 | 4個 |
| 染色体数 | 変化なし（2n） | 半分（n） |
| 相同染色体の対合 | なし | あり |

**配偶子形成**
- 精子：精原細胞→一次精母細胞→二次精母細胞→精細胞→精子（1個から4個）。
- 卵：卵原細胞→一次卵母細胞→二次卵母細胞→卵（1個から1個の卵と極体）。

**受精と発生**
受精卵は**卵割**（体細胞分裂だが細胞が大きくならない特殊な分裂）を繰り返す。
```
受精卵 → 桑実胚 → 胞胚 → 原腸胚 → 神経胚 → 幼生
```
カエル・ウニの発生が代表例。原腸胚期に外胚葉・中胚葉・内胚葉の3つの胚葉ができる。

| 胚葉 | 分化する器官（例） |
|---|---|
| 外胚葉 | 表皮・神経（脳・脊髄） |
| 中胚葉 | 骨格・筋肉・腎臓・血管・心臓 |
| 内胚葉 | 消化管の内壁・肝臓・肺 |

**誘導と形成体**
シュペーマンの実験で、原口背唇部が接する外胚葉を神経に**誘導**することが示された。他の部分の分化を促すこの領域を**形成体（オーガナイザー）**という。誘導は連鎖的に起こる（例：眼の形成）。

**母性因子**
卵の細胞質にあらかじめ蓄えられ、発生初期の体軸などを決める物質（例：ショウジョウバエのビコイド mRNA）。

**ホメオティック遺伝子**
からだの部域（どこに何をつくるか）を決める遺伝子群（ホックス遺伝子など）。変異すると、触角の位置に脚ができるなど、器官の位置が入れかわる（ホメオティック突然変異）。

**植物の発生**
受精卵から胚が発生し、種子ができる。植物は分裂組織（**茎頂・根端の分裂組織**）で一生を通じて器官をつくり続ける。花の形成には**ABCモデル**（A・B・Cクラスの遺伝子の組合せでがく・花弁・おしべ・めしべが決まる）が知られる。

### 例題

**例題1** 減数分裂で1個の一次精母細胞から最終的に何個の精子ができるか。

**解答** 4個。第一・第二分裂を経て4個の精細胞ができ、それぞれ精子になる。

**例題2** シュペーマンが発見した、他の細胞の分化を促す領域を何というか。

**解答** 形成体（オーガナイザー）。原口背唇部がこれにあたる。

### 練習問題
1. 減数分裂で染色体数は 2n からどうなるか。
2. 第一分裂前期に相同染色体どうしが並ぶ現象を何というか。
3. 神経や表皮に分化する胚葉は何か。
4. 卵割が普通の体細胞分裂と異なる点を1つ述べよ。
5. からだの部域を決め、変異すると器官の位置が入れかわる遺伝子群を何というか。
6. 中胚葉から分化する器官を1つ挙げよ。

### 解答と解説
1. **n（半分）になる**。配偶子が受精で再び 2n に戻る。
2. **対合（シナプシス）**。この時に乗換えが起こり遺伝子の組換えが生じる。
3. **外胚葉**。神経（脳・脊髄）と表皮になる。
4. **細胞が成長せず（大きくならず）分裂を繰り返す**ため、割球はしだいに小さくなる。
5. **ホメオティック遺伝子**（ホックス遺伝子）。ショウジョウバエで有名。
6. **筋肉・骨格・腎臓・心臓・血管**などから1つ。

### つまずきやすいポイント
- 減数分裂は「2回分裂で4個・染色体半減」、体細胞分裂は「1回分裂で2個・染色体不変」。この対比を必ず押さえる。
- 卵割は体細胞分裂の一種だが、割球が大きくならないため胚全体の大きさはあまり変わらない。
- 誘導は「一方が他方の分化を促す」現象。形成体は誘導する側。
- 母性因子は「卵にあらかじめ蓄えられた」もの。受精後につくられるのではない。

## 生物の環境応答

### この単元で学ぶこと
- ニューロンの構造と興奮の伝導・伝達を説明できる。
- 受容器と効果器のはたらきを理解する。
- 筋収縮のしくみ（滑り説）を説明できる。
- 動物の行動と植物ホルモンの作用を理解する。
- 光周性と屈性を説明できる。

### 解説

**ニューロン（神経細胞）**
細胞体・樹状突起・軸索からなる。軸索が髄鞘で覆われた**有髄神経繊維**では、興奮がとびとびに伝わる**跳躍伝導**が起こり速い。

**興奮の伝導と伝達**
- 静止時は細胞内が負（**静止電位**）。刺激で内外の電位が逆転し**活動電位**が生じる。
- **伝導**：1本のニューロン内を興奮が伝わること。両方向に伝わりうる。
- **伝達**：ニューロン間（**シナプス**）で神経伝達物質（アセチルコリンなど）を介して伝わること。一方向のみ。

**受容器と効果器**

| 受容器 | 適刺激 |
|---|---|
| 眼（網膜） | 光 |
| 耳（うずまき管） | 音 |
| 耳（前庭・半規管） | 傾き・回転 |
| 鼻 | 化学物質（におい） |
| 舌 | 化学物質（味） |

効果器は筋肉・分泌腺など、命令を受けて反応する器官。

**筋収縮（滑り説）**
筋原繊維は**アクチンフィラメント**と**ミオシンフィラメント**からなる。Ca²⁺の作用でミオシンがアクチンをたぐり寄せ、**アクチンがミオシンの間に滑り込む**ことで**サルコメア（筋節）が短くなる**。ミオシンフィラメント自体の長さは変わらない。ATP がエネルギー源。

**動物の行動**
- **生得的行動**：生まれつき備わる（かぎ刺激による定型的行動、走性など）。
- **学習行動**：経験で獲得（慣れ、条件づけ、刷り込みなど）。ミツバチのダンスによる情報伝達も有名。

**植物ホルモン**

| ホルモン | 主なはたらき |
|---|---|
| オーキシン | 成長促進、屈性、頂芽優勢 |
| ジベレリン | 種子発芽・果実肥大の促進 |
| サイトカイニン | 細胞分裂の促進 |
| アブシシン酸 | 気孔を閉じる、休眠の維持 |
| エチレン | 果実の成熟促進 |

**屈性と光周性**
- **屈性**：刺激の方向に対して曲がる性質。光屈性はオーキシンが光の当たらない側に移動して伸長を促し、茎が光の方へ曲がる。
- **光周性**：日長（連続した暗期の長さ）に応じた反応。**短日植物**（アサガオ・キク）は暗期が一定以上で花芽形成、**長日植物**（アブラナ）は暗期が短いと花芽形成。花芽形成にはフロリゲン（花成ホルモン）が関与。

### 例題

**例題1** シナプスで興奮が伝わるときにはたらく物質の総称を答えよ。

**解答** 神経伝達物質（例：アセチルコリン）。

**例題2** 筋収縮でサルコメアが短くなるとき、ミオシンフィラメントの長さは変化するか。

**解答** 変化しない。アクチンがミオシンの間に滑り込むため、フィラメント自体は短くならない（滑り説）。

### 練習問題
1. 有髄神経繊維で興奮がとびとびに伝わる現象を何というか。
2. 静止時のニューロン内外に見られる電位を何というか。
3. 茎が光の方へ曲がる性質にかかわる植物ホルモンは何か。
4. 短日植物の花芽形成を決めるのは、明期・暗期のどちらの長さか。
5. 刷り込み（インプリンティング）は生得的行動・学習行動のどちらに分類されるか。
6. 気孔を閉じるはたらきをもつ植物ホルモンは何か。

### 解答と解説
1. **跳躍伝導**。髄鞘のすき間（ランビエ絞輪）を跳んで伝わるため速い。
2. **静止電位**。細胞内が外に対して負になっている。
3. **オーキシン**。光の当たらない側に移動し、その側の成長を促す。
4. **暗期（連続した暗期）**の長さ。短日植物は暗期が一定以上必要。
5. **学習行動**。生後の限られた時期の経験で獲得される。
6. **アブシシン酸**。乾燥時に気孔を閉じ、水分の損失を防ぐ。

### つまずきやすいポイント
- 伝導（1本のニューロン内、両方向可）と伝達（シナプス間、一方向のみ）を区別する。
- 滑り説では**フィラメント自体は縮まない**。アクチンが滑り込むことでサルコメアが短くなる。
- 光周性を決めるのは日照時間（明期）ではなく、**連続した暗期の長さ**。
- オーキシンは低濃度で成長促進、高濃度では逆に抑制することがある（部位で最適濃度が違う）。

## 生態と環境

### この単元で学ぶこと
- 個体群と個体群密度の概念を理解する。
- 生命表と生存曲線を読み取れる。
- 種間関係（競争・捕食・共生など）を説明できる。
- 生態系の物質生産のしくみを理解する。
- 生物多様性の意味と保全を説明できる。

### 解説

**個体群と個体群密度**
同じ地域に生活する同種の個体の集まりを**個体群**という。個体群密度＝個体数 ÷ 生活空間の面積（体積）。密度が高くなると成長や繁殖に影響が出る（**密度効果**）。

個体群の成長は理論上は指数関数的（J字型）だが、資源の制限で実際は**S字型（ロジスティック曲線）**になり、環境収容力で頭打ちになる。

**生命表と生存曲線**
生命表は各発育段階での生存・死亡数をまとめた表。生存曲線には3型ある。

| 型 | 特徴 | 例 |
|---|---|---|
| 早死型（凸型） | 親の保護が厚く、幼齢の死亡が少ない | 大型ほ乳類・ヒト |
| 平均型（直線型） | 各時期で死亡率がほぼ一定 | 小型鳥類・ヒドラ |
| 晩死型（凹型） | 産卵数が多く、幼齢で多く死ぬ | 魚・多くの無脊椎動物 |

**種間関係**

| 関係 | 内容 |
|---|---|
| 競争（種間競争） | 資源をめぐって不利益（ゾウリムシの実験） |
| 捕食・被食 | 食う・食われる（周期的な個体数変動） |
| 相利共生 | 互いに利益（アリとアブラムシ、マメ科と根粒菌） |
| 片利共生 | 一方だけ利益 |
| 寄生 | 一方が利益、他方が不利益 |

生態的地位（ニッチ）が似た種は競争が激しく、**すみわけ・食いわけ**で共存することがある（競争的排除）。

**生態系の物質生産**
- **総生産量**：生産者が光合成でつくる有機物の総量。
- **純生産量**＝総生産量 − 呼吸量。
- **成長量**＝純生産量 −（被食量＋枯死量）。
栄養段階を上がるにつれ利用できるエネルギーは減る（エネルギー効率は数〜十数%程度）。

**生物多様性**
遺伝的多様性・種多様性・生態系多様性の3つの階層でとらえる。生息地破壊・外来生物・乱獲・地球温暖化などが脅威。小さな個体群では近親交配や遺伝的浮動により**絶滅の渦**に陥りやすい。

### 例題

**例題1** 純生産量を、総生産量と呼吸量を用いて式で表せ。

**解答** 純生産量 ＝ 総生産量 − 呼吸量。

**例題2** 産卵数が多く幼齢期に多くが死ぬ、魚などに見られる生存曲線の型を何というか。

**解答** 晩死型（凹型）。

### 練習問題
1. 個体群密度が高くなることで成長や繁殖が影響を受ける現象を何というか。
2. 個体群の成長を表すS字型の曲線を何というか。
3. 生態的地位（にすむ場所や食物）を分け合って共存することを何というか。
4. マメ科植物と根粒菌の関係は、種間関係のうちどれにあたるか。
5. 成長量は、純生産量から何を差し引いた量か。
6. 生物多様性の3つの階層をすべて答えよ。

### 解答と解説
1. **密度効果**。過密になると個体の成長が遅れるなどする。
2. **ロジスティック曲線**。環境収容力に近づくと頭打ちになる。
3. **すみわけ・食いわけ**。競争的排除を避けて共存する。
4. **相利共生**。植物は窒素化合物を得て、根粒菌は有機物を得る。
5. **被食量と枯死量**。成長量＝純生産量−(被食量＋枯死量)。
6. **遺伝的多様性・種多様性・生態系多様性**。

### つまずきやすいポイント
- 純生産量には呼吸量が含まれない（総生産量から呼吸量を引く）。用語の関係を式で覚える。
- 生存曲線の「早死型」「晩死型」は名称と実際の死に方が紛らわしい。早死型＝早くは死なない（幼齢の死亡が少ない）と押さえる。
- 相利共生（両方得）と片利共生（片方だけ得）、寄生（一方が損）を区別する。
- 個体群密度の分母は「面積・体積」。単なる個体数ではない。

## 生物の進化と系統

### この単元で学ぶこと
- 生命の起源と化学進化の道すじを理解する。
- 進化のしくみ（突然変異・自然選択・遺伝的浮動）を説明できる。
- ハーディ・ワインベルグの法則を用いて遺伝子頻度を計算できる。
- 種分化と分子系統のしくみを理解する。
- 生物の分類体系（3ドメイン・五界説）と系統樹を説明できる。

### 解説

**生命の起源と化学進化**
原始地球で無機物から有機物（アミノ酸など）が生じ、しだいに複雑化した過程を**化学進化**という。ミラーの実験（原始大気を模した放電）でアミノ酸の生成が示された。最初の生命は約40億年前に現れたと考えられ、やがて光合成を行うシアノバクテリアが酸素を放出し大気環境を変えた。RNA が遺伝と触媒の両方を担った「RNAワールド」仮説がある。

**進化のしくみ**

| 要因 | 内容 |
|---|---|
| 突然変異 | DNA の変化。進化の材料（変異）を生む |
| 自然選択 | 環境に適した個体が多く子孫を残す（適応進化） |
| 遺伝的浮動 | 偶然による遺伝子頻度の変化（小集団で大きい） |
| 隔離 | 集団間の交流が絶たれ、独自に進化 |

**ハーディ・ワインベルグの法則**
一定の条件（十分大きい集団・突然変異なし・自由な交配・自然選択なし・移出入なし）が満たされるとき、遺伝子頻度は世代を経ても変化しない。

対立遺伝子 A の頻度を p、a の頻度を q（p + q = 1）とすると、遺伝子型の頻度は：
```
p² ＋ 2pq ＋ q² ＝ 1
（AA）  （Aa）  （aa）
```

**種分化**
1つの種が2つ以上に分かれること。地理的隔離による**異所的種分化**が代表例。隔離が続くと生殖的隔離が生じ、交配できなくなる。

**分子系統**
DNA やタンパク質のアミノ酸配列の違い（変化の蓄積）を比較して類縁関係を推定する。違いが大きいほど分岐が古いと考える（**分子時計**）。

**生物の分類**
- 分類の階級：ドメイン > 界 > 門 > 綱 > 目 > 科 > 属 > 種。
- 種の学名は**二名法**（属名＋種小名、リンネ）。
- **3ドメイン説**（ウーズ）：細菌（バクテリア）・古細菌（アーキア）・真核生物。
- **五界説**（ホイタッカー）：モネラ（原核）・原生生物・菌類・植物・動物。

**系統樹**
生物の進化的な類縁関係を枝分かれの図で表したもの。分子データに基づくものが分子系統樹。

### 例題

**例題1** ある遺伝子について、劣性ホモ接合(aa)の個体が集団の16%を占める。ハーディ・ワインベルグの法則が成り立つとき、a の遺伝子頻度 q を求めよ。

**解答** q² ＝ 0.16 より q ＝ 0.4。

**例題2** 例題1の集団で、ヘテロ接合(Aa)の割合を求めよ。

**解答** p ＝ 1 − 0.4 ＝ 0.6。2pq ＝ 2 × 0.6 × 0.4 ＝ 0.48、すなわち **48%**。

### 練習問題
1. 原始地球で無機物から有機物が生じた過程を何というか。
2. 偶然によって遺伝子頻度が変化する現象を何というか（特に小集団で大きい）。
3. ハーディ・ワインベルグの法則が成り立つ条件を2つ挙げよ。
4. リンネが確立した、属名と種小名で学名をつける方法を何というか。
5. ウーズが提唱した、生物を3つの大きなグループに分ける説を何というか。
6. ある集団で A の遺伝子頻度が 0.7 のとき、a の頻度を求めよ。

### 解答と解説
1. **化学進化**。ミラーの実験がその可能性を示した。
2. **遺伝的浮動**。集団が小さいほど影響が大きい。
3. 「十分大きい集団」「突然変異がない」「自由な交配」「自然選択がない」「移出入がない」から2つ。
4. **二名法**。ラテン語で表記する。
5. **3ドメイン説**。細菌・古細菌・真核生物に分ける。rRNA の解析にもとづく。
6. p + q = 1 より q ＝ 1 − 0.7 ＝ **0.3**。

### つまずきやすいポイント
- ハーディ・ワインベルグの法則は「遺伝子頻度が変化しない」条件。この条件が崩れることが進化。
- p² + 2pq + q² = 1 で、ヘテロ(Aa)は 2pq（2倍を忘れやすい）。
- 自然選択は「環境に適したものが残る」ことであり、生物が意図的に変化するのではない。
- 3ドメインの「古細菌（アーキア）」は名前に反して細菌より真核生物に近いとされる。「古い細菌」ではない点に注意。
