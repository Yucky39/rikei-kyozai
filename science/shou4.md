# 小学4年 理科

小学4年生の理科では、1年間を通した生き物のようすや天気の変化、電気や空気・水のせいしつ、月や星の動き、そして自分の体のつくりなど、身のまわりの自然をくわしく調べます。観察や実験をたくさん行い、「くらべること」「記録すること」「理由を考えること」を大切にして学んでいきましょう。

---

## 季節と生き物

### この単元で学ぶこと

- 春・夏・秋・冬のそれぞれの季節で、植物や動物のようすがどう変わるかを観察して記録できる。
- あたたかい季節には生き物の活動が活発になり、寒い季節には活動がにぶくなることを理解する。
- 1年間、同じ植物(サクラ、ヘチマなど)や同じ場所の動物を続けて観察し、変化を季節と関係づけて考えられる。

### 解説

生き物のようすは、季節によって大きく変わります。学校の校庭にあるサクラの木や、花だんにまいたヘチマのたねを、1年間ずっと観察してみましょう。同じ木、同じ場所を決めて、気温もいっしょに記録するのがポイントです。

サクラの1年間のようすを表にまとめると、次のようになります。

| 季節 | サクラのようす |
| --- | --- |
| 春 | 花がさく。花が散ったあと、緑色の葉が出てくる。 |
| 夏 | 葉がこい緑色になり、たくさんしげる。 |
| 秋 | 葉が赤や黄色に色づき、やがて落ちる。 |
| 冬 | 葉がすっかり落ちるが、えだの先には小さな芽(冬芽)がついていて、春をまっている。 |

冬のサクラは、かれてしまったように見えます。でも、えだをよく見ると、小さくてかたい芽がついています。この芽が、春になって花や葉になるのです。「冬の木はかれているのではなく、春のじゅんびをしている」ということが、続けて観察するとわかります。

ヘチマは、春にたねをまいて育てるつる植物です。ヘチマの育ち方は次のとおりです。

| 季節 | ヘチマのようす |
| --- | --- |
| 春 | たねをまくと、めが出て、子葉(しよう)が開く。 |
| 夏 | くきがぐんぐんのびて、まきひげで支柱にまきつく。黄色い花がさく。 |
| 秋 | 実が大きく育つ。実の中にたくさんの黒いたねができる。 |
| 冬 | 実はかれて茶色になり、ふると中のたねがカラカラと音を立てる。かれた実からたねをとる。 |

ヘチマのくきは、気温が高い夏にいちばんよくのびます。1週間ごとにくきの長さをはかって記録すると、「気温が高いとき、ヘチマはよく育つ」ということがグラフからも読み取れます。

動物のようすも季節で変わります。

| 季節 | 動物のようす(例) |
| --- | --- |
| 春 | ツバメが南の国からやって来て、巣を作り始める。オオカマキリやナナホシテントウが活動を始める。 |
| 夏 | ツバメのひなが育つ。カブトムシの成虫が見られる。セミが鳴く。動物の活動がいちばん活発になる。 |
| 秋 | エンマコオロギやスズムシが鳴く。カマキリがたまごを産む。ツバメは南の国へ帰っていく。 |
| 冬 | 見られる動物の数がへる。ナナホシテントウは落ち葉の下などで、ヒキガエルは土の中で冬をこす。オオカマキリはたまごのすがたで冬をこす。 |

このように、生き物のようすの変化は、気温の変化と深く関係しています。あたたかくなると活動が活発になり、寒くなると活動がにぶくなったり、すがたを変えて冬をこしたりするのです。

### やってみよう(観察・実験)

**1年間の定点観察**

- 用意するもの:観察カード、えんぴつ、色えんぴつ、温度計、ものさし(ヘチマ用)、虫めがね
- 手順
  1. 観察するサクラの木と、観察する場所(花だんや草むら)を1つ決める。
  2. 月に1〜2回、同じ場所で観察し、日づけ・天気・気温を必ず記録する。
  3. 見たままの形や色をスケッチし、気づいたことを言葉でも書く。
  4. ヘチマは、くきの長さをものさしではかり、記録を折れ線グラフにまとめる。
- 結果の見方:気温の変化のグラフと、生き物のようすの記録をならべてくらべると、気温と生き物の活動の関係が見えてきます。
- 安全上の注意:ハチやかぶれる植物(ウルシなど)には近づかない。観察のときは、生き物をむやみにつかまえたり、すみかをこわしたりしない。

### 確認問題

1. 【選択】冬のサクラのえだの先についている小さなものは何ですか。ア:たね イ:芽 ウ:実
2. 【穴埋め】ヘチマのくきがいちばんよくのびるのは、気温が(   )季節である。
3. 【選択】春に南の国から日本にやって来て、秋に南の国へ帰る鳥はどれですか。ア:スズメ イ:ツバメ ウ:カラス
4. 【記述】冬になると、見られる動物の数がへるのはなぜですか。気温という言葉を使って説明しましょう。
5. 【選択】オオカマキリはどのすがたで冬をこしますか。ア:成虫 イ:よう虫 ウ:たまご
6. 【記述】生き物の観察カードには、日づけのほかに、何を記録しておくと季節との関係を調べやすくなりますか。

### 解答と解説

1. **答え:イ(芽)** 冬のサクラはかれているように見えますが、えだの先には冬芽がついていて、春に花や葉になります。
2. **答え:高い(あたたかい)** ヘチマは気温が高い夏に、くきが1日で数cm〜10cm以上のびることもあります。植物の成長は気温と関係しています。
3. **答え:イ(ツバメ)** ツバメはわたり鳥で、春に日本へ来て子育てをし、秋に南のあたたかい国へ帰ります。スズメやカラスは1年中日本にいます。
4. **答え(例):冬は気温が低くなるので、動物は活動がにぶくなり、土の中や落ち葉の下などでじっとして冬をこすものが多いから。** 動物がいなくなったのではなく、すがたが見えにくい場所で冬ごししていることが大切なポイントです。
5. **答え:ウ(たまご)** オオカマキリは秋にあわのかたまりのような「らんのう」にたまごを産み、たまごのすがたで冬をこして、春によう虫がかえります。
6. **答え(例):天気と気温。** 生き物のようすの変化は気温と深く関係しているので、気温を記録しておくと、変化の理由を考えるときに役立ちます。

### つまずきやすいポイント

- 「冬の木はかれてしまった」と考えてしまいがちですが、落葉樹は葉を落として冬をこしているだけで、生きています。冬芽を実際に観察すると、まちがいに気づけます。
- 「冬に虫がいなくなるのは、みんな死んでしまうから」という誤解があります。たまご・よう虫・さなぎ・成虫と、こん虫によって冬ごしのすがたがちがうことをおさえましょう。
- 観察カードに絵だけをかいて、日づけや気温を書き忘れると、あとで季節と関係づけられません。「いつ・どこで・気温は何度か」をセットで記録する習かんをつけましょう。

## 天気と気温

### この単元で学ぶこと

- 1日の気温の変化を、時こくを決めてはかり、折れ線グラフに表すことができる。
- 晴れの日は気温の変化が大きく、くもりや雨の日は変化が小さいことを理解する。
- 気温の正しいはかり方(百葉箱の役わり)を知る。

### 解説

「気温」とは、空気の温度のことです。気温は、はかり方を決めておかないと正しくくらべられません。気温をはかるときの約束は次の3つです。

- 風通しのよい場所ではかる。
- 温度計に直せつ日光が当たらないようにする。
- 地面から1.2m〜1.5mの高さではかる。

学校の校庭には、白い箱のような「百葉箱(ひゃくようばこ)」が置いてあります。百葉箱は、この3つの約束を守って気温をはかるための箱です。

| 百葉箱のつくり | 理由 |
| --- | --- |
| 白くぬられている | 日光をはね返して、箱の中が熱くなりすぎないようにするため。 |
| よろい戸(すきまのある戸)になっている | 風を通して、外の空気と同じ温度にするため。 |
| とびらが北向きについている | とびらを開けたとき、直せつ日光が中の温度計に当たらないようにするため。 |
| しばふの上に立っている | 地面の照り返しのえいきょうを受けにくくするため。 |

1日の気温を1時間ごとにはかって折れ線グラフにすると、天気によってグラフの形がちがうことがわかります。

| 天気 | 1日の気温の変化 |
| --- | --- |
| 晴れの日 | 変化が大きい。朝は低く、昼すぎ(午後2時ごろ)にいちばん高くなり、夕方から夜にかけて下がる。グラフは山の形になる。 |
| くもりの日 | 変化が小さい。グラフはなだらかになる。 |
| 雨の日 | 変化がとても小さい。1日中ほとんど同じくらいの気温のこともある。 |

ここで大切なのは、気温がいちばん高くなるのが「正午(昼の12時)」ではなく「午後2時ごろ」だということです。太陽がいちばん高くなるのは正午ごろですが、太陽の熱はまず地面をあたため、あたためられた地面が空気をあたためます。地面があたたまってから空気があたたまるまでに時間がかかるので、気温の最高は正午より少しおくれて午後2時ごろになるのです。

くもりや雨の日に気温の変化が小さいのは、雲が日光をさえぎって地面があまりあたためられないことと、夜も雲がふとんのような役わりをして熱がにげにくいことが理由です。

### やってみよう(観察・実験)

**1日の気温の変化を調べよう**

- 用意するもの:温度計(または百葉箱の温度計)、記録用紙、グラフ用紙、時計
- 手順
  1. 晴れの日を選び、午前9時から午後3時ごろまで、1時間ごとに気温をはかる。
  2. はかるときは、風通しのよい日かげ(または温度計に厚紙などで日よけをする)で、地面から約1.2m〜1.5mの高さではかる。
  3. 記録を折れ線グラフにする。横のじくは時こく、たてのじくは気温にする。
  4. べつの日(くもりや雨の日)にも同じようにはかり、グラフを重ねてくらべる。
- 結果の見方:晴れの日のグラフは午後2時ごろを頂上とする山の形、くもりや雨の日はなだらかな形になれば、天気と気温の変化の関係がつかめています。
- 安全上の注意:温度計はガラスでできているので、落としてわらないようにていねいにあつかう。えきだめ(温度計の下のふくらみ)を手でにぎると体温でくるってしまうので、目もりを読むときはえきだめにさわらない。目もりは温度計と目を直角にして読む。

### 確認問題

1. 【選択】晴れの日、1日のうちで気温がいちばん高くなるのはいつごろですか。ア:正午ごろ イ:午後2時ごろ ウ:午後4時ごろ
2. 【穴埋め】気温は、風通しのよい場所で、温度計に直せつ(   )が当たらないようにしてはかる。
3. 【選択】百葉箱のとびらはどの方角に向いてついていますか。ア:東 イ:南 ウ:北
4. 【記述】雨の日は、晴れの日とくらべて1日の気温の変化がどうなりますか。理由もつけて説明しましょう。
5. 【選択】百葉箱が白くぬられている理由として正しいものはどれですか。ア:よごれが目立つように イ:日光をはね返すため ウ:遠くから見えるように
6. 【記述】太陽がいちばん高いのは正午ごろなのに、気温がいちばん高くなるのが午後2時ごろなのはなぜですか。

### 解答と解説

1. **答え:イ(午後2時ごろ)** 太陽の高さがいちばん高いのは正午ごろですが、気温の最高はそれより少しおくれて午後2時ごろになります。
2. **答え:日光** 温度計に日光が直せつ当たると、空気の温度ではなく、日光であたためられた温度計自体の温度をはかることになってしまいます。
3. **答え:ウ(北)** 太陽は東から南の空を通って西へ動くので、北向きにしておけば、とびらを開けても中の温度計に日光が当たりません。
4. **答え(例):雨の日は気温の変化が小さくなる。雲が日光をさえぎるので、地面や空気があまりあたためられないから。** グラフにすると、晴れの日は山の形、雨の日はほぼ平らな形になります。
5. **答え:イ(日光をはね返すため)** 白い色は日光(熱)をはね返しやすいので、箱の中の温度が上がりすぎるのをふせぎます。
6. **答え(例):太陽の熱は、まず地面をあたため、あたためられた地面が空気をあたためる。空気があたたまるまでに時間がかかるので、気温の最高は正午より少しおくれるから。** 「太陽→地面→空気」の順にあたたまることがポイントです。

### つまずきやすいポイント

- 「太陽がいちばん高い正午に気温も最高になる」と考えやすいですが、実さいは午後2時ごろです。地面があたたまってから空気があたたまる、という順番を図でおさえましょう。
- 日なたで温度計をかざしてはかってしまうミスが多いです。それでは「気温」ではなく「日光に当たった温度計の温度」になってしまいます。
- 温度計のえきだめを指でつまんで持つと、体温で目もりが上がってしまいます。持つときは上の方を持ちましょう。
- 折れ線グラフで、点と点を定規で結ばずにフリーハンドでぐにゃぐにゃにかいてしまうと、変化のようすが読み取りにくくなります。

## 電流のはたらき

### この単元で学ぶこと

- 回路に流れる電気の流れを「電流」といい、電流には向きがあることを理解する。
- かん電池の向きを反対にすると、電流の向きが反対になり、モーターの回る向きも反対になることを知る。
- かん電池2この「直列つなぎ」と「へい列つなぎ」のちがいと、電流の大きさ・モーターの速さの関係を理解する。
- けん流計(かんいけん流計)を使って、電流の向きと大きさを調べられる。

### 解説

かん電池とモーターを導線でつないで輪のようにすると、電気が流れてモーターが回ります。この電気の通り道を「回路」、回路を流れる電気の流れを「電流」といいます。

電流には向きがあります。電流は、かん電池の**+極(プラスきょく)から出て、回路を通り、−極(マイナスきょく)へ**流れます。かん電池の向きを反対にすると電流の向きが反対になり、モーターの回る向きも反対になります。プロペラをつけたモーターなら、風の向きが変わるのでよくわかります。

電流の向きや大きさを調べる道具が「けん流計(かんいけん流計)」です。

- はりのふれる**向き**で、電流の**向き**がわかる。
- はりのふれる**大きさ(目もり)**で、電流の**大きさ**がわかる。
- けん流計は、回路の中に(直列に)つないで使う。
- **けん流計をかん電池だけに直せつつないではいけません。** 大きな電流が流れて、けん流計がこわれてしまいます。必ずモーターや豆電球といっしょに回路に入れます。

かん電池を2こ使うつなぎ方には、「直列つなぎ」と「へい列つなぎ」があります。

| つなぎ方 | つなぎ方のようす | 電流の大きさ | モーターの回り方 | 電池の持ち |
| --- | --- | --- | --- | --- |
| 直列つなぎ | かん電池の+極と、もう1この−極を順につなぐ(1本道になる) | かん電池1このときより大きい | 速く回る | 早くなくなる |
| へい列つなぎ | かん電池の+極どうし、−極どうしをそろえてつなぐ(道が2本に分かれる) | かん電池1このときとほぼ同じ | 1このときとほぼ同じ速さ | 長持ちする |

直列つなぎは、かん電池を2こ分の力で電流をおし出すので、電流が大きくなり、モーターは速く回り、豆電球は明るくつきます。そのかわり、電池は早く使い切ってしまいます。

へい列つなぎは、電流の大きさが1このときとほぼ同じなので、モーターの速さも豆電球の明るさもほぼ同じです。そのかわり、2この電池で分たんするので長持ちします。

へい列つなぎで注意することがあります。+極と−極の向きをそろえてつながないと、電流がうまく流れません。図をかいて、電流の通り道を指でなぞってたしかめるとよいでしょう。

この学習は、光電池(太陽電池)にもつながります。光電池に日光を当てると電流が流れ、当てる光を強くすると電流が大きくなり、モーターが速く回ります。

### やってみよう(観察・実験)

**かん電池のつなぎ方とモーターの速さを調べよう**

- 用意するもの:かん電池2こ、かん電池ボックス、モーター、プロペラ、導線(どうせん)、スイッチ、かんいけん流計
- 手順
  1. かん電池1こ・モーター・スイッチ・けん流計を1つの輪になるようにつなぎ、スイッチを入れて、けん流計のはりのふれ(電流の大きさ)とプロペラの回る速さを記録する。
  2. かん電池の向きを反対にして、はりのふれる向きとプロペラの回る向きを調べる。
  3. かん電池2こを直列つなぎにして、同じように電流の大きさとプロペラの速さを調べる。
  4. かん電池2こをへい列つなぎにして、同じように調べる。
  5. 結果を表にまとめ、1このとき・直列・へい列でくらべる。
- 結果の見方:直列つなぎのときだけ、けん流計のはりが大きくふれ、プロペラが速く回れば成功です。へい列つなぎは1このときとほぼ同じになります。
- 安全上の注意:導線をかん電池の+極と−極に直せつつなぐ「ショート回路」は、導線やかん電池が熱くなってとても危険なので、ぜったいにしない。けん流計をかん電池だけにつながない。使い終わったらスイッチを切り、かん電池を外しておく。

### 確認問題

1. 【選択】電流は、かん電池のどちらの極から流れ出ますか。ア:+極 イ:−極 ウ:どちらでもない
2. 【穴埋め】かん電池2こを(   )つなぎにすると、電流が大きくなり、モーターが速く回る。
3. 【選択】かん電池2こをへい列つなぎにしたとき、モーターの回る速さはかん電池1このときとくらべてどうなりますか。ア:速くなる イ:ほぼ同じ ウ:おそくなる
4. 【記述】かん電池の向きを反対にすると、モーターの回り方はどうなりますか。理由もつけて答えましょう。
5. 【選択】けん流計の使い方として正しいものはどれですか。ア:かん電池だけに直せつつなぐ イ:モーターなどといっしょに回路の中につなぐ ウ:回路の外に置いて近づける
6. 【記述】直列つなぎとへい列つなぎのよい点を、それぞれ1つずつ書きましょう。

### 解答と解説

1. **答え:ア(+極)** 電流は+極から出て、導線・モーターを通って−極へ流れると決められています。
2. **答え:直列** 直列つなぎは電池2こ分の力で電流をおし出すので、電流が大きくなります。
3. **答え:イ(ほぼ同じ)** へい列つなぎでは、回路に流れる電流の大きさが1このときとほぼ同じなので、モーターの速さもほぼ同じです。
4. **答え(例):反対向きに回る。かん電池の向きを反対にすると、回路を流れる電流の向きが反対になるから。** けん流計のはりのふれる向きも反対になります。
5. **答え:イ** けん流計をかん電池だけにつなぐと、大きすぎる電流が流れてこわれます。必ずモーターや豆電球のある回路の中に入れて使います。
6. **答え(例):直列つなぎのよい点は、電流が大きくなってモーターが速く回る(豆電球が明るくつく)こと。へい列つなぎのよい点は、かん電池が長持ちすること。** 目的に合わせてつなぎ方を選ぶことが大切です。

### つまずきやすいポイント

- 「電池を2こ使えば、どんなつなぎ方でもモーターは速く回る」と思いこみやすいです。へい列つなぎでは1このときとほぼ同じ速さです。実験でたしかめて、ちがいを表にまとめましょう。
- へい列つなぎで+極と−極の向きをそろえずにつないでしまい、「電流が流れない」となるミスがよくあります。回路の図をかいて、電流の道すじを指でなぞってたしかめましょう。
- ショート回路(電池の+極と−極を導線だけで直せつつなぐこと)は、熱くなって危険です。「電気の通り道に必ずモーターや豆電球を入れる」と覚えましょう。
- 「電流は+極と−極の両方から出て、モーターのところでぶつかる」という誤解があります。電流は+極から−極へ一方通行で流れます。

## 月と星

### この単元で学ぶこと

- 月は日によって形が変わって見えるが、どの形の月も、太陽と同じように東から出て南の空を通り西へしずむことを理解する。
- 星には明るさや色のちがいがあることを知る。
- 星ざの星のならび方は変わらないが、時間がたつと星ざの位置が動くことを理解する。

### 解説

**月の動きと形**

月を同じ場所から続けて観察すると、月は時間とともに動いて見えることがわかります。月は、**東の空からのぼり、南の空の高いところを通って、西の空にしずみます**。これは太陽の1日の動きと同じ向きです。

月は日によって形が変わって見えます。

| 月の形 | 見え方 |
| --- | --- |
| 満月 | まん丸の月。夕方、東の空からのぼり、真夜中に南の空高くを通り、明け方に西へしずむ。 |
| 半月(上げんの月) | 右半分が光って見える月。夕方に南の空に見え、真夜中に西へしずむ。 |
| 三日月 | 細い月。夕方、西の低い空に見える。 |

形はちがっても、動く向きは「東→南→西」でどれも同じです。月の形は、新月→三日月→半月→満月と、およそ1か月かけてもとの形にもどります。

月の位置を記録するときは、木や建物など動かない目印といっしょにスケッチし、時こくを書いておきます。うでをのばしてにぎりこぶしを重ねて数えると、高さ(角度)のおよそのめやすになります(こぶし1つ分が約10度)。

**星の明るさと色**

夜空の星には、明るさのちがいがあります。明るい順に「1等星、2等星、3等星…」と分けられています。また、星には色のちがいもあります。

| 星の名前 | 星ざ | 明るさ | 色 |
| --- | --- | --- | --- |
| ベガ(おりひめ星) | こと座 | 1等星 | 白っぽい |
| アルタイル(ひこ星) | わし座 | 1等星 | 白っぽい |
| デネブ | はくちょう座 | 1等星 | 白っぽい |
| アンタレス | さそり座 | 1等星 | 赤っぽい |
| ベテルギウス | オリオン座 | 1等星 | 赤っぽい |
| リゲル | オリオン座 | 1等星 | 青白い |

夏の夜空では、ベガ・アルタイル・デネブの3つの1等星を結んだ「夏の大三角」が見つけやすい目印です。冬にはオリオン座がよく見えます。

**星の動き**

星も、時間がたつと見える位置が変わります。多くの星は、月や太陽と同じように**東から南の空を通って西へ**動きます。ただし、北の空の星は、**北極星を中心に反時計回りに回る**ように動きます。北極星はほとんど動かないので、昔から方角を知る目印にされてきました。北極星は、カシオペヤ座や北と七星(ほくとしちせい)を手がかりにして見つけられます。

大切なのは、**星ざの星のならび方(形)は変わらない**ということです。動くのは星ざ全体の位置で、星どうしのならび方はくずれません。

### やってみよう(観察・実験)

**月と星の動きを観察しよう**

- 用意するもの:観察カード、方位じしん、時計、星ざ早見、かい中電灯(赤いセロハンをかぶせると目が慣れたままで見やすい)
- 手順
  1. 観察する場所を決め、方位じしんで東・西・南・北をたしかめる。
  2. 月(または夏の大三角などの星ざ)の位置を、建物や木などの目印といっしょにスケッチし、時こくを記録する。
  3. 1〜2時間後、同じ場所・同じ向きでもう一度観察し、位置がどう変わったかを記録する。
  4. 星ざは、星のならび方が変わっていないかもたしかめる。
- 結果の見方:月や星ざが、時間とともに東から南、南から西の方へ動いていれば、太陽と同じ向きに動いていることがたしかめられます。星ざの形は変わりません。
- 安全上の注意:夜の観察は必ず大人の人といっしょに行う。車の通る道路の近くはさける。寒い季節は防寒をしっかりする。観察は同じ場所・同じ向きで行うこと。

### 確認問題

1. 【選択】満月は、どの方角からのぼりますか。ア:東 イ:西 ウ:北
2. 【穴埋め】月や多くの星は、東から(   )の空を通って西へ動く。
3. 【選択】時間がたつと、星ざはどうなりますか。ア:位置もならび方も変わる イ:位置は変わるがならび方は変わらない ウ:位置もならび方も変わらない
4. 【記述】夏の大三角をつくる3つの星の名前を書きましょう。
5. 【選択】ほとんど動かず、方角を知る目印になる星はどれですか。ア:北極星 イ:ベテルギウス ウ:アンタレス
6. 【記述】月の位置を観察して記録するとき、気をつけることを2つ書きましょう。

### 解答と解説

1. **答え:ア(東)** 満月も三日月も半月も、月は東からのぼり、南の空を通って西へしずみます。太陽と同じ向きです。
2. **答え:南** 東→南→西という動きは、太陽・月・多くの星に共通しています。ただし北の空の星は北極星を中心に回るように動きます。
3. **答え:イ** 星ざ全体の位置は時間とともに動きますが、星どうしのならび方(星ざの形)は変わりません。
4. **答え:ベガ(こと座)、アルタイル(わし座)、デネブ(はくちょう座)** どれも1等星で、夏の夜空で見つけやすい三角形をつくります。ベガはおりひめ星、アルタイルはひこ星ともよばれます。
5. **答え:ア(北極星)** 北の空の星は北極星を中心に回るように動きますが、北極星自体はほとんど動きません。
6. **答え(例):同じ場所・同じ向きで観察すること。建物や木など動かない目印といっしょに、時こくも記録すること。** 場所や向きが変わると、月がどれだけ動いたか正しくくらべられません。

### つまずきやすいポイント

- 「月の形がちがうと、動き方もちがう」と考えがちですが、満月でも三日月でも動く向きは東→南→西で同じです。見える時こくと位置がちがうだけです。
- 「星は動かない」という思いこみがあります。星も時間とともに位置が変わります。1〜2時間あけて2回観察すると、動きが実感できます。
- 「星ざの形も時間とともに変わる」という誤解も多いです。動くのは位置だけで、ならび方は変わりません。
- 観察のたびに場所や向きを変えてしまうと、記録がくらべられなくなります。「同じ場所・同じ向き・目印といっしょ」が観察の鉄則です。

## 空気と水のせいしつ

### この単元で学ぶこと

- とじこめた空気は、おすと体積が小さくなり(おしちぢめられ)、おし返す力(手ごたえ)が大きくなることを理解する。
- とじこめた水は、おしても体積が変わらない(おしちぢめられない)ことを理解する。
- 空気と水のせいしつのちがいが、身のまわりの道具に利用されていることを知る。

### 解説

空気は目に見えませんが、たしかにそこにあります。ふくろに空気を集めてとじこめると、ふくろはパンパンになり、おすと手ごたえがあります。この「とじこめた空気」のせいしつを、注しゃ器(ちゅうしゃき)を使って調べてみましょう。

注しゃ器に空気をとじこめて、ピストンを指でおすと──

- ピストンは下がり、中の空気の**体積は小さくなる**(空気はおしちぢめられる)。
- おせばおすほど、**手ごたえ(おし返す力)は大きくなる**。
- 指をはなすと、ピストンは**もとの位置までもどる**。おしちぢめられた空気には、もとの体積にもどろうとする力があるからです。

今度は、注しゃ器に水をとじこめて、同じようにピストンをおすと──

- ピストンは**ほとんど下がらない**。水は、おしても**体積が変わらない**(おしちぢめられない)のです。

| くらべること | とじこめた空気 | とじこめた水 |
| --- | --- | --- |
| おしたときの体積 | 小さくなる(おしちぢめられる) | 変わらない(おしちぢめられない) |
| 手ごたえ | おすほど大きくなる | 初めからかたく、おしこめない |
| 指をはなすと | ピストンがもとにもどる | 変わらない |

このせいしつは、空気でっぽうでたしかめられます。つつの前と後ろに玉をつめ、間に空気をとじこめておし棒でおすと、「ポン!」と音がして前の玉が勢いよく飛び出します。これは、とじこめられた空気がおしちぢめられ、もとにもどろうとする力で前玉をおし出すからです。玉が飛ぶしゅん間、後玉はまだ前玉にとどいていません。空気の力が玉を飛ばしているのです。

つつに水をつめて水でっぽうのようにおすと、玉は「ポン」と飛びません。水はおしちぢめられないので、おし棒でおした分だけ前玉がそのままおし出されるだけです。

身のまわりでは、このせいしつがいろいろな物に利用されています。

- 自転車のタイヤ、ボール:とじこめた空気がおしちぢめられて、はずむ・しょうげきをやわらげる。
- きりふき、ペットボトルロケット:空気や水をおし出す力を利用している。
- エアクッション(かんしょう材):とじこめた空気がしょうげきを受け止める。

### やってみよう(観察・実験)

**注しゃ器で空気と水をおしくらべよう**

- 用意するもの:プラスチックの注しゃ器(先にゴム板やキャップでふたをする)2本、水、空気でっぽう、玉(スポンジ玉など)
- 手順
  1. 注しゃ器のピストンを引いて空気を入れ、先をゴム板におし当ててふさぐ。
  2. ピストンを指でゆっくりおし、体積の変化(目もり)と手ごたえを記録する。
  3. 指をはなして、ピストンがどうなるかを見る。
  4. もう1本の注しゃ器に水を入れ(空気が入らないように注意)、同じようにおして、体積と手ごたえをくらべる。
  5. 空気でっぽうで玉を飛ばし、前玉が飛ぶときの後玉の位置を観察する。
- 結果の見方:空気は目もりが小さくなり、水は目もりがほとんど変わらなければ、「空気はおしちぢめられるが、水はおしちぢめられない」ことがたしかめられます。
- 安全上の注意:空気でっぽうの玉を、人や動物に向けてぜったいに飛ばさない。注しゃ器のピストンを力まかせにおして、指をはさんだり注しゃ器をこわしたりしないようにする。

### 確認問題

1. 【選択】注しゃ器にとじこめた空気をおすと、空気の体積はどうなりますか。ア:大きくなる イ:小さくなる ウ:変わらない
2. 【選択】注しゃ器にとじこめた水をおすと、水の体積はどうなりますか。ア:大きくなる イ:小さくなる ウ:変わらない
3. 【穴埋め】とじこめた空気をおしちぢめると、手ごたえはだんだん(   )なる。
4. 【記述】空気でっぽうの前玉が飛び出すのはなぜですか。「おしちぢめられた空気」という言葉を使って説明しましょう。
5. 【選択】とじこめた空気のせいしつを利用しているものはどれですか。ア:ガラスのコップ イ:自転車のタイヤ ウ:鉄のくぎ
6. 【記述】空気でっぽうのつつに、玉と玉の間に空気のかわりに水をつめておすと、どうなりますか。理由もつけて答えましょう。

### 解答と解説

1. **答え:イ(小さくなる)** 空気はおしちぢめられるせいしつがあるので、ピストンをおすと体積が小さくなります。
2. **答え:ウ(変わらない)** 水はおしちぢめられないので、どんなに強くおしても体積は変わらず、ピストンはほとんど動きません。
3. **答え:大きく** おしちぢめられた空気は、もとの体積にもどろうとしておし返します。ちぢめるほど、おし返す力は大きくなります。
4. **答え(例):おし棒でおされて、とじこめられた空気がおしちぢめられる。おしちぢめられた空気がもとにもどろうとする力で、前玉がおし出されて飛ぶ。** 後玉が前玉に直せつぶつかって飛ばしているのではない、というところが大切です。
5. **答え:イ(自転車のタイヤ)** タイヤの中の空気がおしちぢめられて、地面からのしょうげきをやわらげています。ボールがはずむのも同じせいしつです。
6. **答え(例):前玉は「ポン」と勢いよく飛ばず、おした分だけおし出される。水はおしちぢめられないので、もとにもどろうとする力がはたらかないから。** 空気と水のせいしつのちがいがそのまま結果のちがいになります。

### つまずきやすいポイント

- 「空気でっぽうは、後玉が前玉にぶつかって飛ばしている」という誤解がとても多いです。前玉が飛ぶしゅん間、後玉はまだはなれた位置にあります。間の空気の力で飛ぶことをおさえましょう。
- 「水も強くおせば少しはちぢむはず」と考えがちですが、この学習では「水はおしちぢめられない」と理解すれば十分です。実験で手ごたえのちがいを体感することが一番の近道です。
- 注しゃ器に水を入れるとき、空気のあわが入っていると「水も少しちぢんだ」ように見えてしまいます。空気を入れないように注意して実験しましょう。
- 「空気は物ではない(何もない)」という誤概念があります。ふくろに集めるとおし返してくる体験を通して、空気も体積のあるれっきとした物だと理解しましょう。

## 物の温度と体積

### この単元で学ぶこと

- 空気・水・金ぞくは、あたためると体積が大きくなり、冷やすと体積が小さくなることを理解する。
- 体積の変化の大きさは、空気がいちばん大きく、次に水、金ぞくはとても小さいことを知る。
- 金ぞくは熱した部分から順にあたたまり、水や空気はあたためられた部分が上へ動いて全体があたたまることを理解する。

### 解説

**温度と体積の変化**

物は、温度が変わると体積が変わります。空気・水・金ぞくのそれぞれで調べてみましょう。

- **空気**:試験管の口にせっけん水のまくを作り、試験管を手やお湯であたためると、まくがふくらみます。氷水で冷やすと、まくはへこみます。空気はあたためると体積が大きくなり、冷やすと小さくなるのです。へこんだピンポン玉をお湯につけるともとにもどるのも、中の空気があたためられて体積が大きくなるからです。
- **水**:水をいっぱいに入れたフラスコに細いガラス管をつけてお湯であたためると、水面がガラス管の中を少し上がります。冷やすと下がります。水も、あたためると体積が大きくなり、冷やすと小さくなります。ただし、その変化は空気よりずっと小さいです。
- **金ぞく**:金ぞくの球が輪をぎりぎり通りぬける「金ぞく球実験器」を使います。球を熱すると、体積が大きくなって輪を通りぬけられなくなります。冷やすと、また通りぬけられるようになります。金ぞくも温度で体積が変わりますが、その変化は目で見てもわからないほど小さいのです。

| 物 | あたためると | 冷やすと | 変化の大きさ |
| --- | --- | --- | --- |
| 空気 | 体積が大きくなる | 体積が小さくなる | いちばん大きい |
| 水 | 体積が大きくなる | 体積が小さくなる | 空気より小さい |
| 金ぞく | 体積が大きくなる | 体積が小さくなる | とても小さい(道具を使わないとわからない) |

この体積変化は、身のまわりでも使われています。温度計は、中のえき(灯油に色をつけたものなど)が温度によって体積を変えることを利用した道具です。また、鉄道のレールのつなぎ目にすき間があけてあるのは、夏にレールがあたたまってのびてもぶつからないようにするためです。

**物のあたたまり方**

物のあたたまり方は、金ぞくと、水・空気とで大きくちがいます。

- **金ぞく**:熱した部分から順に、熱が伝わってあたたまっていきます。金ぞくの棒や板に「ろう」または示温(しおん)テープをつけて熱すると、熱したところから近い順にろうがとけて(テープの色が変わって)いくのが見えます。棒をななめにしても、熱した部分から順に伝わります。
- **水**:熱した部分の水があたためられて上へ動き、上にあった冷たい水が下へ下りてきて、ぐるぐる回りながら全体があたたまります。試験管の水を上の方だけ熱すると、上は熱くなっても下はなかなかあたたまりません。あたたまった水は上へ動くからです。おがくずや示温インクを入れると、水の動きが目に見えます。
- **空気**:水と同じように、あたためられた空気は上へ動き、冷たい空気は下へ動いて、全体があたたまります。だんぼうした部屋で、天じょう近くの温度が床(ゆか)近くより高くなるのはこのためです。

| 物 | あたたまり方 |
| --- | --- |
| 金ぞく | 熱した部分から順に伝わってあたたまる |
| 水 | あたためられた水が上へ動き、動きながら全体があたたまる |
| 空気 | あたためられた空気が上へ動き、動きながら全体があたたまる |

### やってみよう(観察・実験)

**空気・水・金ぞくの体積の変化を調べよう**

- 用意するもの:試験管、せっけん水、丸底フラスコ、細いガラス管つきゴムせん、金ぞく球実験器、実験用ガスコンロ(またはアルコールランプ)、お湯、氷水、ぬれぞうきん
- 手順
  1. 試験管の口にせっけん水のまくを作り、試験管を両手で包むようにあたためて、まくのようすを見る。次に氷水につけて冷やし、まくのようすを見る。
  2. 水をいっぱいに入れたフラスコにガラス管つきゴムせんをして、水面の位置に印をつける。お湯につけたときと氷水につけたときの水面の位置をくらべる。
  3. 金ぞく球実験器の球が輪を通ることをたしかめてから、球を熱して輪を通るかどうか調べる。そのあと水で冷やして、もう一度通るか調べる。
- 結果の見方:空気(まく)は大きくふくらみ、水(水面)は少しだけ上がり、金ぞくは道具を使ってやっと変化がわかる──この「変化の大きさのちがい」に注目しましょう。
- 安全上の注意:火を使う実験は、必ずぬれぞうきんを用意し、まどを開けてかん気する。熱した金ぞく球は見た目では熱いとわからないので、ぜったいに素手でさわらない。かた足立ちせず、いすをしまって立って実験する。やけどをしたら、すぐに流水で冷やして先生に知らせる。

### 確認問題

1. 【選択】空気をあたためると、体積はどうなりますか。ア:大きくなる イ:小さくなる ウ:変わらない
2. 【選択】温度による体積の変化がいちばん大きいのはどれですか。ア:金ぞく イ:水 ウ:空気
3. 【穴埋め】金ぞくは、(   )した部分から順にあたたまっていく。
4. 【記述】だんぼうしている部屋では、天じょう近くと床近くのどちらの空気の温度が高いですか。理由もつけて答えましょう。
5. 【選択】試験管に入れた水の真ん中あたりを熱したとき、先にあたたまるのはどこですか。ア:熱したところより上 イ:熱したところより下 ウ:上も下も同時
6. 【記述】鉄道のレールのつなぎ目に、すき間があけてあるのはなぜですか。
7. 【選択】へこんだピンポン玉をお湯につけるともとにもどるのはなぜですか。ア:プラスチックがとけるから イ:中の空気があたためられて体積が大きくなるから ウ:お湯がしみこむから

### 解答と解説

1. **答え:ア(大きくなる)** 空気はあたためると体積が大きくなり、冷やすと小さくなります。せっけん水のまくがふくらむ実験でたしかめられます。
2. **答え:ウ(空気)** 変化の大きさは、空気>水>金ぞくの順です。金ぞくの変化は金ぞく球実験器のような道具を使わないとわかりません。
3. **答え:熱** 金ぞくは、熱した部分から近い順に熱が伝わっていきます。ろうのとける順番で目に見えます。
4. **答え:天じょう近く。あたためられた空気は上へ動くから。** だから、だんぼうのときは風を下向きにしたり、サーキュレーターで空気をかきまぜたりすると、部屋全体があたたまりやすくなります。
5. **答え:ア(熱したところより上)** あたためられた水は上へ動くので、熱したところより上が先にあたたまります。下の方はなかなかあたたまりません。
6. **答え(例):夏にレールがあたためられて体積が大きくなり(のびて)、レールどうしがぶつかって曲がってしまわないようにするため。** 金ぞくの体積変化は小さいですが、レールのように長い物では無視できない大きさになります。
7. **答え:イ** とじこめられた中の空気があたためられて体積が大きくなり、へこみを内側からおし返すのでもとにもどります。

### つまずきやすいポイント

- 「金ぞくは温度が変わっても体積は変わらない」と思いがちです。変化が小さくて見えないだけで、金ぞくも体積が変わります。金ぞく球実験器で「輪を通らなくなる」体験が効果的です。
- 「水も金ぞくと同じように、熱したところから順にあたたまる」という誤解が多いです。水と空気は「あたためられた部分が上へ動く」ことで全体があたたまります。金ぞくとのちがいをはっきり区別しましょう。
- 「あたためると空気(の量)がふえる」と考えてしまうことがあります。ふえたのは体積であって、空気の量そのものはふえていません。
- 前の単元「とじこめた空気」の学習とまざって、「おしちぢめる」と「あたためる」を混同することがあります。体積を変える原因が「力」なのか「温度」なのかを整理しましょう。

## 水のすがた

### この単元で学ぶこと

- 水は、温度によって「氷(固体)・水(えき体)・水じょう気(気体)」とすがたを変えることを理解する。
- 水は熱すると約100℃でふっとうし、冷やすと0℃でこおり始めることを知る。
- ふっとうしているときのあわの正体は水じょう気であり、「ゆげ」は水じょう気が冷えてできた小さな水のつぶであることを理解する。
- 水は熱しなくても、水面からじょう発して水じょう気になって空気中に出ていくことを知る(自然の中の水)。

### 解説

**水を熱すると**

ビーカーの水を実験用ガスコンロで熱しながら、1分ごとに温度をはかってみましょう。

- 温度はどんどん上がっていきますが、**約100℃になると、それ以上は上がらなくなります**。
- このとき、水の中からさかんに大きなあわが出ています。この状態を「**ふっとう**」といいます。
- ふっとうしている間、熱し続けても温度は約100℃のまま変わりません。
- あわの正体は、水が目に見えないすがたに変わった「**水じょう気**」です。空気ではありません。

やかんの口の少し先に見える白いもやもやは「**ゆげ**」です。ゆげは気体ではなく、水じょう気が空気中で冷やされてできた**小さな水のつぶ(えき体)**です。やかんの口のすぐ近くに何も見えない部分がありますが、そこには目に見えない水じょう気があります。水じょう気(見えない)→ゆげ(見える)という順番に注意しましょう。

**水を冷やすと**

試験管の水を、食塩をまぜた氷水につけて冷やしていくと──

- 水の温度は下がっていき、**0℃になるとこおり始めます**。
- こおっている間は、温度は**0℃のまま**変わりません。
- 全部こおってしまうと、温度はさらに0℃より下がっていきます。
- 水が氷になると、**体積が少し大きくなります**。試験管の水がこおると、氷の表面がもり上がるのはこのためです。

**水の3つのすがた**

| すがた | よび方 | 例 | 温度 |
| --- | --- | --- | --- |
| 固体 | 氷 | 氷、雪、しも | 0℃以下でこおる |
| えき体 | 水 | 水、雨、ゆげ(小さな水のつぶ) | 0℃〜100℃ |
| 気体 | 水じょう気 | 空気中の水じょう気(目に見えない) | 約100℃でふっとう。ただしふっとうしなくても水面からじょう発する |

**自然の中の水(じょう発と結露)**

水は、100℃まで熱しなくても、水面から少しずつ水じょう気になって空気中へ出ていきます。これを「**じょう発**」といいます。せんたく物がかわくのも、水たまりがいつの間にか消えるのも、じょう発のおかげです。日なたに置いた入れ物の水は、日かげよりも速くへります。入れ物にラップでふたをしておくと水はほとんどへらず、ラップの内側に水てきがつきます。これは、じょう発した水じょう気がラップで外に出られず、冷えて水にもどったものです。

反対に、空気中の水じょう気が冷たい物にふれると、冷やされて水てきにもどります。冷たい飲み物を入れたコップの外側に水てきがつく「**結露(けつろ)**」がその例です。コップの中の水がしみ出したのではなく、空気中の水じょう気が変わったものです。

このように、水は「じょう発して空気中へ→冷やされて水にもどる」というじゅんかんを、自然の中でくり返しています。

### やってみよう(観察・実験)

**水を熱したときの温度とようすを調べよう**

- 用意するもの:ビーカー、ふっとう石、温度計、スタンド、実験用ガスコンロ、時計、記録用紙、ぬれぞうきん、(冷やす実験用に)試験管・氷・食塩・ビーカー
- 手順
  1. ビーカーに水とふっとう石を入れ、温度計を水の中ほどに固定する(温度計の先がビーカーの底につかないようにする)。
  2. 熱しながら1分ごとに温度とようす(小さなあわ、大きなあわ、ゆげなど)を記録する。
  3. ふっとうしてからも数分間熱し続け、温度が変わらないことをたしかめる。
  4. 結果を折れ線グラフにまとめる。
  5. 冷やす実験では、試験管の水を「氷+食塩+少しの水」につけ、1分ごとに温度とようすを記録し、0℃でこおり始めること、こおる間0℃のままであることをたしかめる。
- 結果の見方:グラフが100℃近くで水平になれば「ふっとう中は温度が変わらない」こと、0℃で水平になれば「こおる間は温度が変わらない」ことが読み取れます。
- 安全上の注意:ふっとう石を必ず入れる(急に熱い水が飛び出す「とっぷつ」をふせぐため)。熱いビーカーやゆげにさわらない。ゆげでもやけどをすることがある。安全めがねをかけ、立って実験する。火のそばにぬれぞうきんを用意しておく。

### 確認問題

1. 【穴埋め】水は熱すると約(   )℃でふっとうし、冷やすと(   )℃でこおり始める。
2. 【選択】ふっとうしている水の中から出る大きなあわの正体は何ですか。ア:空気 イ:水じょう気 ウ:ちっ素
3. 【選択】やかんの口の先に見える白い「ゆげ」の正体は何ですか。ア:目に見える水じょう気 イ:小さな水のつぶ ウ:けむり
4. 【選択】水がこおると、体積はどうなりますか。ア:大きくなる イ:小さくなる ウ:変わらない
5. 【記述】冷たいジュースを入れたコップの外側に水てきがつくのはなぜですか。「水じょう気」という言葉を使って説明しましょう。
6. 【記述】せんたく物が、火であぶらなくてもかわくのはなぜですか。
7. 【穴埋め】水がふっとうしている間、熱し続けても温度は(   )。

### 解答と解説

1. **答え:100、0** ふっとうする温度(約100℃)とこおり始める温度(0℃)は、水のせいしつとして必ず覚えましょう。
2. **答え:イ(水じょう気)** あわの正体は、水がすがたを変えた水じょう気です。あわをふくろに集めると、冷えて水てきになることからたしかめられます。空気ではありません。
3. **答え:イ(小さな水のつぶ)** ゆげは、目に見えない水じょう気が空気中で冷やされて、小さな水のつぶ(えき体)になったものです。見えるものはもう気体ではありません。
4. **答え:ア(大きくなる)** 水は氷になると体積が約1割大きくなります。寒い日に水道管がわれることがあるのはこのためです。
5. **答え(例):空気中の水じょう気が、冷たいコップにふれて冷やされ、水てきにもどったから。** コップの中のジュースがしみ出したのではありません。
6. **答え(例):せんたく物の水が、水面(ぬれた布の表面)から少しずつじょう発して、水じょう気になって空気中へ出ていくから。** 水は100℃にならなくても、じょう発します。風通しがよく日光の当たる場所ほど速くかわきます。
7. **答え:変わらない(約100℃のまま)** 加えられた熱は、温度を上げるのではなく、水を水じょう気に変えるのに使われるので、温度は上がりません。

### つまずきやすいポイント

- 「ふっとうのあわ=空気」という誤解が最も多いです。あわの正体は水じょう気で、集めて冷やすと水にもどることでたしかめられます。
- 「ゆげ=水じょう気」も定番のまちがいです。水じょう気は気体で目に見えません。見えている白いゆげは、冷えてできた小さな水のつぶ(えき体)です。
- 「熱し続ければ温度は100℃をこえてどんどん上がる」と考えがちですが、ふっとう中は約100℃のままです。グラフの水平な部分に注目しましょう。
- 「コップの外側の水てきは、中の飲み物がしみ出したもの」という誤概念があります。空気中の水じょう気が冷やされたものです。
- 「水は熱しないとじょう発しない」と思いこみやすいですが、ふだんの温度でも水面からじょう発は起きています。水たまりが消えることがその証拠です。

## 体のつくりと運動

### この単元で学ぶこと

- 人の体には、かたいほねと、やわらかいきん肉があることを理解する。
- ほねとほねのつなぎ目である「関節」で、体を曲げられることを知る。
- きん肉がちぢんだりゆるんだりすることで、体が動くしくみを理解する。
- 動物にも、ほね・きん肉・関節があり、体を動かしていることを知る。

### 解説

自分のうでや足をさわってみましょう。かたいところと、やわらかいところがあります。かたいところが「**ほね**」、やわらかいところが「**きん肉**」です。人の体には、大きさや形のちがうほねが全身で約200こもあります。

ほねには、大切な役わりがあります。

| ほねの役わり | 例 |
| --- | --- |
| 体を支える | せぼね(たくさんの小さなほねがつながってできている)、足のほね |
| 体の中の大切な部分を守る | 頭のほね(のうを守る)、むねのほね・ろっこつ(心ぞうやはいを守る) |
| きん肉とともに体を動かす | うでのほね、足のほね |

体を曲げられるところと曲げられないところがあります。うでは、ひじや手首で曲がりますが、ひじと手首の間では曲がりません。**ほねとほねのつなぎ目**で、体を曲げられるところを「**関節(かんせつ)**」といいます。ひじ、ひざ、手首、足首、指のつけ根など、体のいろいろな場所に関節があります。せぼねは小さなほねがたくさんつながっているので、体を前後や左右に曲げられるのです。

体を動かしているのは、ほねについている**きん肉**です。きん肉は、**ちぢんだり、ゆるんだり**することで、ほねを動かします。うでを曲げるときのしくみを見てみましょう。

| うでの動き | うでの内側のきん肉 | うでの外側のきん肉 |
| --- | --- | --- |
| うでを曲げるとき | ちぢむ(かたくふくらむ) | ゆるむ(やわらかい) |
| うでをのばすとき | ゆるむ(やわらかい) | ちぢむ(かたくなる) |

うでを曲げて力こぶを作ると、内側のきん肉がかたくふくらむのがわかります。このとき外側のきん肉はゆるんでいます。きん肉は「ちぢむ」ことしかできないので、**内側と外側の2つのきん肉がペアになって**、交代でちぢむことでうでを曲げたりのばしたりしているのです。きん肉は、じょうぶな「けん」でほねの関節をまたいだ先につながっています。かかとの上の「アキレスけん」は、さわってたしかめられる有名なけんです。

人だけでなく、ウサギやハト、魚など、ほかの動物にも、ほね・きん肉・関節があります。ウサギの後ろ足には大きなきん肉がついていて、力強くジャンプできます。ハトはむねのきん肉が大きく発達していて、つばさを力強く動かして飛びます。動物の体のつくりは、その動物の動き方と深く関係しているのです。

### やってみよう(観察・実験)

**うでのきん肉の動きを調べよう**

- 用意するもの:自分の体、うでのほねときん肉の模型(または厚紙とわゴムで作る模型)、にわとりの手羽先(観察用・あれば)
- 手順
  1. 左手で右うでの内側(力こぶの位置)を軽くにぎり、右うでをゆっくり曲げたりのばしたりして、きん肉のかたさとふくらみの変化を感じ取る。
  2. 外側のきん肉もにぎって、内側と反対の動きをしていることをたしかめる。
  3. 自分の体で、曲がるところ(関節)をさがして、体の図に印をつける。
  4. 厚紙2まいをわりピンでつなぎ(関節)、わゴム2本を内側と外側にかけて、きん肉がちぢむとうでが曲がる模型を作って動かしてみる。
- 結果の見方:「曲げると内側がかたくなり、のばすと外側がかたくなる」ことが感じ取れれば、きん肉がペアで働いていることが理解できています。
- 安全上の注意:関節は決まった向きにしか曲がりません。友だちのうでや指を無理な方向に曲げるようなことはぜったいにしない。手羽先を観察したあとは、必ず手をせっけんでよくあらう。

### 確認問題

1. 【穴埋め】ほねとほねのつなぎ目で、体を曲げられるところを(   )という。
2. 【選択】うでを曲げたとき、うでの内側のきん肉はどうなりますか。ア:ちぢむ イ:ゆるむ ウ:変わらない
3. 【選択】頭のほねの大切な役わりはどれですか。ア:体を曲げる イ:のうを守る ウ:血を作らない
4. 【記述】うでをのばすとき、内側と外側のきん肉はそれぞれどうなりますか。
5. 【選択】せぼねが前後左右に曲がるのはなぜですか。ア:せぼねがゴムのようにやわらかいから イ:小さなほねがたくさんつながっているから ウ:せぼねにはきん肉がないから
6. 【記述】ウサギが力強くジャンプできるのは、体のつくりとどんな関係がありますか。

### 解答と解説

1. **答え:関節** ひじ・ひざ・手首・足首・指のつけ根などが関節です。関節があるから体を曲げられます。
2. **答え:ア(ちぢむ)** うでを曲げると内側のきん肉がちぢんで、かたくふくらみます(力こぶ)。このとき外側のきん肉はゆるんでいます。
3. **答え:イ(のうを守る)** ほねには「体を支える」「大切な部分を守る」「体を動かす」役わりがあります。頭のほねはのうを、ろっこつは心ぞうやはいを守っています。
4. **答え:内側のきん肉はゆるみ、外側のきん肉はちぢむ。** 曲げるときと反対です。きん肉はちぢむことしかできないので、内側と外側がペアで交代して働きます。
5. **答え:イ** せぼねは1本の長いほねではなく、小さなほねがたくさんつながってできています。1つ1つのつなぎ目が少しずつ動くので、全体として大きく曲げられます。
6. **答え(例):ウサギの後ろ足には大きなきん肉がついていて、その大きなきん肉が力強くちぢむことで、強くジャンプできる。** 動物の体のつくり(きん肉のつき方)は、その動物の動き方と関係しています。

### つまずきやすいポイント

- 「きん肉は、のびてもちぢんでも力を出す」と考えがちですが、きん肉が力を出すのは「ちぢむ」ときだけです。だから、曲げる用とのばす用の2つのきん肉がペアで必要になります。
- 「うでを曲げると、内側も外側も両方かたくなる」と思う子がいます。力を入れすぎずにゆっくり動かして、内側と外側の交代をたしかめましょう。
- 「ほねが自分で動いて体が動く」という誤解があります。ほね自体は動く力を持たず、きん肉に引っぱられて動きます。
- 「せぼねは1本の長い曲がるほね」と思いこみやすいです。小さなほねがつながっている模型や図でたしかめましょう。

## 雨水の行方と地面のようす

### この単元で学ぶこと

- 水は、高い場所から低い場所へ向かって流れて集まることを理解する。
- 地面にしみこむ水のしみこみ方は、土のつぶの大きさによってちがうことを理解する。
- 雨水の行方(流れる・しみこむ・じょう発する)を、地面のようすと関係づけて考えられる。

### 解説

雨がふったあとの校庭を思いうかべてみましょう。水たまりができる場所と、できない場所があります。雨水はどこへ行ったのでしょうか。雨水の行方には、大きく分けて3つあります。

1. 地面を流れて、低いところへ集まる。
2. 地面にしみこむ。
3. じょう発して空気中へ出ていく(「水のすがた」の学習とつながります)。

**水は高いところから低いところへ流れる**

水たまりは、まわりより**低くなっている場所**にできます。地面はまっ平らに見えても、実は少しかたむいています。地面のかたむきは、ビー玉を置いてころがる向きを見たり、といにビー玉を入れて調べたりするとわかります。ビー玉がころがっていく方向が低い方で、水もそちらへ流れて集まります。校庭のはしにあるはい水こう(みぞ)は、流れてきた雨水を集めて流すために、低い場所に作られています。川の水が海へ向かって流れるのも、高いところから低いところへ流れる水のせいしつによるものです。

**土のつぶの大きさと水のしみこみ方**

校庭の土とすな場のすなでは、雨のあとのようすがちがいます。校庭には水たまりができやすいのに、すな場には水たまりがあまりできません。これは、土のつぶの大きさがちがうからです。

| 土の種類 | つぶの大きさ | 水のしみこみ方 | 水たまり |
| --- | --- | --- | --- |
| すな場のすな | 大きい | 速くしみこむ | できにくい |
| 校庭の土 | 小さい | ゆっくりしみこむ | できやすい |
| じゃり(小石) | とても大きい | とても速くしみこむ | ほとんどできない |

つぶが大きいと、つぶとつぶの間の**すき間が大きい**ので、水がすばやく通りぬけられます。つぶが小さいと、すき間も小さいので、水はゆっくりとしかしみこめません。だから、つぶの小さい土の校庭では、しみこみきれない水が地面にたまって、水たまりになるのです。

雨がやんだあと、水たまりがいつの間にか消えるのは、水が地面にしみこむのに加えて、水面からじょう発して水じょう気になり、空気中へ出ていくからです。しみこんだ水も、やがて地下を流れたり、じょう発したりして、自然の中をめぐっていきます。

### やってみよう(観察・実験)

**水の流れと、しみこみ方を調べよう**

- 用意するもの:ビー玉、とい(またはプラスチックのつつを半分に切ったもの)、ペットボトルを切って作ったそうち2つ、ガーゼ(またはネット)とわゴム、校庭の土、すな場のすな、水、計量カップ、時計
- 手順(地面のかたむき調べ)
  1. 雨上がりに校庭を観察し、水たまりの場所と、水が流れたあとを地図に記録する。
  2. 地面のいろいろな場所にビー玉をそっと置き、ころがる向きを調べて、地図に矢印で記録する。
  3. ビー玉のころがった向きと水たまりの場所をくらべる。
- 手順(しみこみ方くらべ)
  1. ペットボトルの上半分を逆さにし、口にガーゼをわゴムでとめて、ろうとのようなそうちを2つ作る。
  2. 1つに校庭の土、もう1つにすな場のすなを同じ量だけ入れる。
  3. 同じ量の水を同時に注ぎ、水が下に落ちきるまでの時間と、出てきた水の量をくらべる。
- 結果の見方:ビー玉のころがる先に水たまりがあれば「水は低い方へ流れる」ことが、すなの方が速く水が落ちれば「つぶが大きいほど水がしみこみやすい」ことがたしかめられます。
- 安全上の注意:雨上がりの校庭はすべりやすいので走らない。実験で使った土やすなは、決められた場所にもどす。実験のあとは手をよくあらう。

### 確認問題

1. 【穴埋め】水は、(   )い場所から(   )い場所へ向かって流れる。
2. 【選択】水たまりができやすいのはどんな場所ですか。ア:まわりより高い場所 イ:まわりより低い場所 ウ:かたむきと関係ない
3. 【選択】校庭の土とすな場のすなでは、どちらが水が速くしみこみますか。ア:校庭の土 イ:すな場のすな ウ:同じ
4. 【記述】すな場のすなの方が水が速くしみこむのはなぜですか。「つぶ」という言葉を使って説明しましょう。
5. 【選択】地面のかたむきを調べる方法として正しいものはどれですか。ア:ビー玉を置いてころがる向きを見る イ:温度計をさす ウ:虫めがねで見る
6. 【記述】雨がやんだあと、水たまりの水がなくなっていくわけを2つ書きましょう。

### 解答と解説

1. **答え:高、低** 水が高いところから低いところへ流れるのは、水たまりだけでなく、川が海へ流れることにも共通するせいしつです。
2. **答え:イ(まわりより低い場所)** 水は低い方へ流れて集まるので、水たまりはまわりより低くなっている場所にできます。
3. **答え:イ(すな場のすな)** すなは校庭の土よりつぶが大きいので、水が速くしみこみます。だからすな場には水たまりができにくいのです。
4. **答え(例):すなは土よりつぶが大きく、つぶとつぶの間のすき間が大きいので、水がすばやく通りぬけられるから。** 「つぶが大きい→すき間が大きい→しみこみやすい」というつながりで説明できることが大切です。
5. **答え:ア** ビー玉は低い方へころがるので、ころがる向きで地面のかたむきがわかります。といを使う方法もあります。
6. **答え(例):1つは、水が地面にしみこんでいくから。もう1つは、水面からじょう発して水じょう気になり、空気中へ出ていくから。** 「水のすがた」で学んだじょう発と結びつけて考えられるとよいです。

### つまずきやすいポイント

- 「校庭はまっ平らだから、水はどこにも流れない」と考えがちです。見た目には平らでも、地面にはわずかなかたむきがあり、水はそれを正直にたどって低い方へ集まります。ビー玉実験で実感しましょう。
- 「つぶが小さい方が、すき間にするする入って速くしみこみそう」という直感的な誤解が多いです。実さいはつぶが大きい方がすき間も大きく、速くしみこみます。実験で時間をはかってくらべることが大切です。
- 「水たまりの水は、ぜんぶ地面にしみこんで消えた」と、じょう発を忘れてしまいがちです。アスファルトのようにしみこみにくい場所の水たまりも消えることから、じょう発にも気づかせましょう。
- しみこみ方をくらべる実験で、土の量や水の量をそろえないと、正しくくらべられません。「くらべる実験では、調べたい条件以外は同じにする」という考え方をここでしっかり身につけましょう。

## おわりに:4年生の理科で身につけたいこと

4年生の理科では、どの単元でも「変化」と「関係づけ」がキーワードでした。

| 単元 | 何と何を関係づけたか |
| --- | --- |
| 季節と生き物 | 季節(気温)と生き物のようす |
| 天気と気温 | 天気と1日の気温の変化 |
| 電流のはたらき | かん電池のつなぎ方と電流の大きさ |
| 月と星 | 時間と月・星の位置 |
| 空気と水のせいしつ | おす力と体積・手ごたえ |
| 物の温度と体積 | 温度と体積・あたたまり方 |
| 水のすがた | 温度と水のすがた(氷・水・水じょう気) |
| 体のつくりと運動 | きん肉のちぢみと体の動き |
| 雨水の行方 | 地面のかたむき・土のつぶと水の動き |

観察や実験をするときは、「予想を立てる → 条件をそろえて調べる → 記録する → 結果から考える」という流れを大切にしましょう。この考え方は、5年生の「ふりこ」や「流れる水のはたらき」、6年生の学習にもずっとつながっていきます。
