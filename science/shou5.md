# 小学5年 理科

5年生の理科では、「条件をそろえて調べる」という考え方がとても大切になります。
何かを調べたいときは、**調べたい条件だけを変えて、それ以外の条件はすべて同じにする**——これが実験の基本です。
このページのすべての単元で、この考え方が出てきます。意識しながら読んでいきましょう。

---

## 天気の変化

### この単元で学ぶこと

- 雲の量や動きと、天気の変化には関係があることを知る
- 天気は、雲の動きにつれて、およそ西から東へ変わっていくことを理解する
- 気象情報（アメダス、気象衛星の雲画像）を使って、天気を予想できるようになる
- 台風の動きと、台風がもたらす天気の変化・災害について知る

### 解説

#### 雲と天気

空を見上げると、雲の形や量は毎日ちがいます。天気は、空全体を10としたときの**雲の量**で決めます。

| 雲の量（空全体を10として） | 天気 |
| --- | --- |
| 0〜8 | 晴れ（0〜1はとくに「快晴」） |
| 9〜10 | くもり |

雨がふっていれば、雲の量に関係なく「雨」です。

雲にはいろいろな種類があります。

| 雲の名前 | 見た目 | 天気との関係 |
| --- | --- | --- |
| 巻雲（すじ雲） | 高い空にすじのようにかかる白い雲 | 天気が変わり始める前ぶれのことがある |
| 積乱雲（入道雲・かみなり雲） | もくもくと高くもり上がる大きな雲 | 強い雨、かみなり、ひょうをもたらすことがある |
| 乱層雲（雨雲） | 空一面をおおう暗い灰色の雲 | 長い時間、雨をふらせる |

#### 天気は西から東へ変わる

気象衛星の雲画像を何日か続けて見ると、日本付近の雲は**およそ西から東へ**動いていることがわかります。
これは、日本の上空に西から東へ向かう風（偏西風といいます）がふいているためです。

そのため、天気も**およそ西から東へ**変わっていきます。

- 大阪で雨がふっているとき、その雨雲が東へ動いてくれば、東京もやがて雨になると予想できます。
- 夕焼けがきれいに見えると次の日は晴れることが多い、という言い伝えがあります。夕焼けが見えるのは西の空に雲がないということなので、西から晴れがやってくると考えられるからです。

#### 気象情報の使い方

| 情報 | わかること |
| --- | --- |
| 気象衛星の雲画像 | 雲がどこにあって、どちらへ動いているか |
| アメダス（地域気象観測システム） | 全国各地の雨の量、風、気温など |
| 天気図 | 高気圧・低気圧の位置など |

雲画像とアメダスの雨量情報を組み合わせると、「雨をふらせている雲が、これからどこへ動くか」を予想できます。

#### 台風と天気

台風は、日本のはるか**南の海上**で発生します。夏から秋にかけて日本に近づくことが多く、次のような特ちょうがあります。

- 台風が近づくと、**強い風**がふき、**短い時間にたくさんの雨**がふる
- 台風の動きは、ふつうの雲のように「西から東」だけではなく、南から北へ進んだり、日本付近で東寄りに曲がったりする
- 台風が通り過ぎると、青空が広がることが多い

台風は、こう水や土砂くずれなどの災害を引き起こすことがある一方、大切な水資源をもたらすという面もあります。

### やってみよう（観察・実験）

**雲の観察と天気の予想**

- 用意するもの：記録カード、方位磁針、（あれば）カメラ
- 手順
  1. 毎日決まった時刻（例：午前10時と午後2時）に、同じ場所で空を観察する。
  2. 方位磁針で方位を確かめ、雲の量（0〜10）、雲の形、雲が動く向きを記録する。
  3. 数日分の記録を並べて、雲のようすと天気の変化の関係を考える。
  4. テレビやインターネットの雲画像と、自分の観察記録を見比べる。
- 結果の見方：雲が増えて厚くなってくると雨に向かうことが多い。西の空のようすが、次の天気のヒントになる。
- 安全上の注意：**太陽を直接見ないこと**。目をいためます。雨や雷のときは屋内から観察しましょう。

### 確認問題

1. 空全体を10としたとき、雲の量が「7」の日の天気は何ですか。（晴れ・くもり・雨 から選ぶ）
2. 日本付近の天気は、およそどの方位からどの方位へ変わっていきますか。
3. 穴うめ：もくもくと高くもり上がり、強い雨やかみなりをもたらす雲を（　　）雲という。
4. 大阪が雨のとき、「東京もこのあと雨になりそうだ」と予想できるのはなぜですか。理由を書きましょう。
5. 台風について正しいものをすべて選びましょう。
   ア　台風は日本の北の海上で発生する。
   イ　台風が近づくと強い風がふき、大雨になることが多い。
   ウ　台風の進む向きは、いつも西から東と決まっている。
   エ　台風は災害を引き起こすこともあるが、水資源をもたらす面もある。

### 解答と解説

1. **晴れ**。雲の量が0〜8なら晴れ、9〜10ならくもりです。雨がふっていない限り、雲が7割あっても「晴れ」です。
2. **西から東**。日本の上空には西から東へふく風があり、雲がその風に流されて動くためです。
3. **積乱（せきらん）雲**。入道雲、かみなり雲ともよばれます。
4. （例）**天気は雲の動きとともに、およそ西から東へ変わっていくから。** 大阪は東京より西にあるので、大阪の雨雲が東へ動いてくると、東京も雨になると予想できます。
5. **イとエ**。アは誤り（台風は南の海上で発生します）。ウも誤り（台風の進路はさまざまで、西から東とは限りません）。

### つまずきやすいポイント

- 「雲が多い＝くもり」と思いこみやすいですが、**雲の量が8までは「晴れ」**です。数字の区切りを正確に覚えましょう。
- 「天気は西から東へ変わる」は、**雲が風に流されて動くから**です。理由とセットで覚えると忘れません。
- **台風の進路は「西から東」の決まりに当てはまらない**ことがあります。ふつうの天気の変化と台風を区別しましょう。
- 雨の日は「雲の量に関係なく雨」です。雲の量で判断するのは、雨がふっていないときだけです。

## 植物の発芽と成長

### この単元で学ぶこと

- 種子が発芽するために必要な条件（水・空気・適当な温度）を、条件を制御した実験で調べる
- 種子の中のつくりと、発芽に使われる養分（でんぷん）について知る
- 発芽した後の植物がよく成長するには、日光と肥料が関係することを理解する
- 「変える条件」と「変えない条件」を区別して実験を計画できるようになる

### 解説

#### 発芽の3つの条件

インゲンマメの種子を使って、発芽に何が必要かを調べます。
ここで大切なのが**条件の制御**です。調べたい条件だけを変え、ほかの条件はすべて同じにします。

| 調べたいこと | 変える条件 | 変えない条件 | 結果 |
| --- | --- | --- | --- |
| 水は必要か | 水をあたえる／あたえない | 空気、温度、種子の種類 | 水ありだけ発芽する |
| 空気は必要か | 空気にふれさせる／水にしずめて空気をなくす | 水、温度、種子の種類 | 空気ありだけ発芽する |
| 適当な温度は必要か | 室温（約20度）に置く／冷ぞう庫（約5度）に入れる | 水、空気、種子の種類 | 室温だけ発芽する |

この実験から、種子の発芽には次の**3つの条件**がすべて必要だとわかります。

1. **水**
2. **空気**
3. **適当な温度**

なお、温度を比べる実験では、冷ぞう庫の中は暗いので、室温のほうにも箱をかぶせて**暗くして条件をそろえる**くふうが必要です。ここがこの実験の大事なポイントです。
また、この実験から、**発芽に日光（光）は必要ない**こともわかります。

#### 種子の中の養分

インゲンマメの種子を水にひたしてやわらかくし、半分に割ってみると、中に小さな**根・くき・葉になる部分**と、大きな**子葉（しよう）**があります。

子葉に**ヨウ素液**をかけると、**青むらさき色**に変わります。ヨウ素液は**でんぷん**があると青むらさき色に変わる薬品なので、子葉にはでんぷんがふくまれているとわかります。

発芽してしばらくたったあとのしぼんだ子葉にヨウ素液をかけると、色はほとんど変わりません。つまり、**種子の中のでんぷんは、発芽と成長のための養分として使われた**のです。

#### 植物がよく成長する条件

発芽したインゲンマメのなえを使って、成長の条件を調べます。

| 比べるなえ | 日光 | 肥料 | 結果 |
| --- | --- | --- | --- |
| A | あてる | あたえる | 葉の色がこく、くきが太く、じょうぶに育つ |
| B | あてない（箱をかぶせる） | あたえる | 葉の色がうすく黄色っぽく、くきが細くひょろひょろになる |
| C | あてる | あたえない | Aより葉が小さく、育ちがおそい |

どのなえにも水はあたえます（水は「変えない条件」）。
この実験から、植物がよく成長するには、発芽の3条件に加えて**日光**と**肥料**が必要だとわかります。

### やってみよう（観察・実験）

**発芽に水が必要かを調べる実験**

- 用意するもの：インゲンマメの種子、カップ2個、だっし綿、水
- 手順
  1. カップ2個にだっし綿をしき、それぞれに同じくらいの大きさの種子をのせる。
  2. 一方（A）はだっし綿を水でしめらせ、もう一方（B）はかわいたままにする。
  3. 2つとも同じ場所（室温・同じ明るさ）に置き、毎日観察して記録する。
- 結果の見方：数日後、Aだけが発芽すれば、「発芽には水が必要」といえる。
- 安全上の注意：実験に使った種子は口に入れない。ヨウ素液を使うときは手や服につかないようにし、ついたらすぐ水で洗う。

### 確認問題

1. 種子の発芽に必要な3つの条件をすべて書きましょう。
2. 発芽に「日光」は必要ですか、必要ではないですか。
3. 穴うめ：でんぷんにヨウ素液をかけると（　①　）色に変わる。インゲンマメの種子ででんぷんが多くふくまれている部分を（　②　）という。
4. 「発芽に適当な温度が必要か」を調べるとき、室温に置く種子と冷ぞう庫に入れる種子を比べます。このとき室温の種子に箱をかぶせて暗くするのはなぜですか。
5. 日光に当てずに育てたインゲンマメのなえは、どのようなすがたになりますか。かんたんに書きましょう。
6. 次のうち、「植物がよく成長するための条件」として正しい組み合わせはどれですか。
   ア　水と空気だけ　　イ　日光と肥料　　ウ　肥料と暗さ

### 解答と解説

1. **水・空気・適当な温度**。この3つがすべてそろわないと発芽しません。
2. **必要ではない**。冷ぞう庫の実験からわかるように、暗くても温度などの条件がそろえば発芽します。日光が必要になるのは発芽したあとの成長です。
3. ①**青むらさき** ②**子葉**。子葉のでんぷんは発芽の養分として使われます。
4. （例）**冷ぞう庫の中は暗いので、明るさの条件をそろえるため。** そろえないと、発芽しなかった原因が温度なのか暗さなのか区別できなくなります。
5. （例）**葉の色がうすく（黄色っぽく）、くきが細長くひょろひょろした弱々しいすがたになる。**
6. **イ**。よく成長するには水に加えて日光と肥料が必要です。

### つまずきやすいポイント

- **「発芽の条件」と「成長の条件」を混同しやすい**です。発芽＝水・空気・適当な温度。成長＝それに加えて日光・肥料。日光と肥料は発芽には不要です。
- 「土がないと発芽しない」と思いがちですが、だっし綿の上でも発芽します。**土は発芽の条件ではありません**。
- 条件を調べる実験では、**変える条件は1つだけ**。2つ以上いっぺんに変えると、どの条件が原因かわからなくなります。
- 「適当な温度」の「適当」は「いいかげん」という意味ではなく、「**ちょうどよい**」という意味です。

## 魚のたんじょう（メダカの飼育と観察）

### この単元で学ぶこと

- メダカのおすとめすを、ひれの形で見分けられるようになる
- メダカのたまご（卵）が受精してから、中で少しずつ変化して子メダカがかえるまでを観察する
- 受精卵は約11日でふ化することを知る
- かえったばかりの子メダカが、はらのふくらみの養分で育つことを理解する

### 解説

#### メダカの飼い方

メダカを観察するには、まず元気に飼うことが大切です。

- 水そうは**直しゃ日光の当たらない明るい場所**に置く
- 水道水はそのまま使わず、**くみ置きした水**（1〜2日置いて塩素をぬいた水）を使う
- 水草（オオカナダモなど）を入れる（たまごを産みつける場所になる）
- えさは**食べ残しが出ない量**を毎日少しずつあたえる
- おすとめすを同じ水そうに入れる

#### おすとめすの見分け方

| 部分 | おす | めす |
| --- | --- | --- |
| 背びれ | **切れこみがある** | 切れこみがない |
| しりびれ | **平行四辺形に近い形で大きい** | 後ろが短く、三角形に近い |

#### 受精とたまごの変化

めすが産んだたまご（卵）と、おすが出す**精子**が結びつくことを**受精**といい、受精したたまごを**受精卵**といいます。受精しなければ、たまごは育ちません。

受精卵の中の変化を、かいぼうけんび鏡やそう眼実体けんび鏡で観察すると、次のように変化していきます（水温約25度のとき）。

| 受精からの日数（めやす） | たまごの中のようす |
| --- | --- |
| 直後 | あわのようなつぶが集まっている。まわりに毛のようなものがある |
| 2〜3日 | 体のもとになる形が見えてくる |
| 4〜5日 | **目**が大きく黒くはっきりしてくる |
| 6〜8日 | **心ぞう**が動き、血液が流れるのが見える。体がときどき動く |
| **約11日** | たまごのまくを破って**子メダカがかえる（ふ化する）** |

#### かえったばかりの子メダカ

ふ化したばかりの子メダカは、**はらがふくらんでいます**。この中に**養分**が入っていて、2〜3日はえさを食べなくても、この養分を使って育ちます。はらのふくらみが小さくなってきたら、細かくしたえさをあたえます。

たまごの中で育つ間の養分も、たまごの中にあるものが使われます。「養分をどこから得るか」という見方は、あとの「人のたんじょう」の学習にもつながります。

### やってみよう（観察・実験）

**メダカのたまごの観察**

- 用意するもの：メダカ（おす・めす）、水そう、水草、かいぼうけんび鏡（またはそう眼実体けんび鏡）、ペトリ皿、スポイト
- 手順
  1. 水草に産みつけられたたまごを、水草ごと別の入れ物に移す（親が食べてしまうのを防ぐため）。
  2. たまごを水草からそっとはずし、水を入れたペトリ皿に入れる。
  3. かいぼうけんび鏡で毎日観察し、日づけ・目や心ぞうのようす・体の動きをスケッチして記録する。
  4. ふ化するまで続け、記録を時間の順に並べて変化をまとめる。
- 結果の見方：目ができる→心ぞうが動く→ふ化、という順序に注目する。
- 安全上の注意：**けんび鏡に直しゃ日光を当てて観察しない**（目をいためる）。生き物は大切にあつかい、観察が終わったら元にもどす。

### 確認問題

1. メダカのおすは、背びれとしりびれにどんな特ちょうがありますか。
2. 穴うめ：めすの産んだたまごと、おすの出す（　①　）が結びつくことを（　②　）という。
3. メダカの受精卵は、水温約25度のとき、およそ何日でふ化しますか。
   ア　約3日　　イ　約11日　　ウ　約38日
4. かえったばかりの子メダカが、2〜3日えさを食べなくても育つのはなぜですか。
5. たまごの観察の記録として、変化の順番が正しいものはどれですか。
   ア　心ぞうが動く → 目がはっきりする → ふ化
   イ　目がはっきりする → 心ぞうが動く → ふ化
6. メダカを飼うとき、水そうを直しゃ日光の当たる窓ぎわに置いてはいけないのはなぜだと思いますか。考えて書きましょう。

### 解答と解説

1. **背びれに切れこみがあり、しりびれが平行四辺形に近い形で大きい。** めすは背びれに切れこみがなく、しりびれは後ろが短い形です。
2. ①**精子** ②**受精**。受精したたまご（受精卵）だけが育ちます。
3. **イ（約11日）**。約38日ではありません（人は約38「週」です。混同に注意）。
4. （例）**はらのふくらみの中に養分が入っていて、それを使って育つから。**
5. **イ**。目がはっきりしてくるのが先（4〜5日ごろ）、心ぞうの動きが見えるのはそのあと（6〜8日ごろ）です。
6. （例）**直しゃ日光が当たると水温が上がりすぎて、メダカが弱ってしまうから。**（水温が急に変わるのもよくありません）

### つまずきやすいポイント

- おす・めすの見分けは**「背びれの切れこみ」と「しりびれの形」の2つセット**で覚えましょう。体の大きさや色では見分けられません。
- 「受精しなくてもたまごは育つ」と思っている子がいますが、**受精卵でなければ育ちません**。
- メダカのふ化まで**約11日**、人のたんじょうまで**約38週**。「11日」と「38週」の単位（日と週）ごと正確に覚えましょう。
- たまごの中の子メダカの養分は「親がえさを運んでくる」のではなく、**はじめからたまごの中にある**ことに注意。ここが人（母親から養分をもらう）とのちがいです。

## 花から実へ（受粉と結実）

### この単元で学ぶこと

- 花のつくり（おしべ・めしべ・花びら・がく）と花粉のはたらきを知る
- アサガオのような「1つの花におしべとめしべがある花」と、ヘチマのような「おばなとめばながある花」を区別する
- めしべの先に花粉がつくこと（受粉）によって、実ができ、種子ができることを、条件を制御した実験で確かめる

### 解説

#### 花のつくり

アサガオの花を分解すると、外側から**がく、花びら、おしべ、めしべ**の順についています。

| 部分 | はたらき・特ちょう |
| --- | --- |
| めしべ | 花の中心に1本。先はしめっていて、花粉がつきやすい。もとのふくらんだ部分が実になる |
| おしべ | めしべのまわりに数本。先の「やく」から**花粉**が出る |
| 花びら | 虫を引きよせるなど |
| がく | 花を支える |

#### 2つのタイプの花

| タイプ | 例 | 特ちょう |
| --- | --- | --- |
| 1つの花におしべとめしべがそろっている | アサガオ | 花が開くときに自分の花粉で受粉しやすい |
| おばなとめばなに分かれている | ヘチマ、カボチャ、ツルレイシ（ゴーヤ） | おばなにはおしべ、めばなにはめしべがある。めばなのもとには実になるふくらみがある |

花粉をけんび鏡で見ると、植物の種類ごとに形がちがいます。ヘチマの花粉はラグビーボールのような形、アサガオの花粉はとげのある球のような形をしています。

#### 受粉と結実

おしべの花粉がめしべの先につくことを**受粉**といいます。
「受粉すると実ができるのか」を、ヘチマ（またはアサガオ）で確かめます。

**ヘチマでの実験**（変える条件：受粉させるかどうか。変えないようにする条件：それ以外すべて）

1. 次の日に開きそうなめばなのつぼみを2つ選び、どちらにも**ふくろをかぶせる**（虫などが勝手に花粉を運んでこないようにするため）。
2. 花が開いたら、
   - A：おばなの花粉をめしべの先につけて（受粉させて）、またふくろをかぶせる。
   - B：受粉させずに、そのままふくろをかぶせておく。
3. 数日後にようすを比べる。

| | A（受粉させた） | B（受粉させない） |
| --- | --- | --- |
| 結果 | めしべのもとがふくらみ、**実になる**。実の中に**種子**ができる | ふくらまず、やがて**かれて落ちる** |

この実験から、**受粉すると実ができ、実の中に種子ができる**ことがわかります。
つぼみのうちからふくろをかぶせるのは、実験する前に虫が花粉を運んで受粉してしまうのを防ぐためです。ここが条件制御のポイントです。

花粉は、虫（ミツバチなど）や風によってめしべまで運ばれます。植物は、種子によって生命をつないでいるのです。

### やってみよう（観察・実験）

**ヘチマの受粉実験**

- 用意するもの：ヘチマのめばなのつぼみ2つ、おばな、ポリエチレンのふくろ、モール（ふくろの口をとめる）、記録カード
- 手順
  1. 次の日に咲きそうなめばなのつぼみを2つ選び、それぞれふくろをかぶせて口を軽くとめる。
  2. 翌朝、花が開いたら、1つはふくろを外し、その日に咲いたおばなを摘んで花粉をめしべの先に軽くつけ、すぐにまたふくろをかぶせる。もう1つはふくろをかぶせたままにする。
  3. どちらの花か分かるように札をつけ、毎日観察して記録する。
- 結果の見方：受粉させた方だけ、めしべのもとがふくらんで実になっていくかを見る。
- 安全上の注意：花粉で手がかゆくなる人もいるので、さわったあとは手を洗う。屋外の観察では帽子をかぶり、ハチなどに注意する。

### 確認問題

1. おしべの花粉がめしべの先につくことを何といいますか。
2. ヘチマの花について正しいものはどれですか。
   ア　1つの花におしべとめしべの両方がある。
   イ　おばなとめばなに分かれていて、めばなのもとに実になるふくらみがある。
   ウ　おばなのもとがふくらんで実になる。
3. 受粉実験で、めばなのつぼみに前の日からふくろをかぶせておくのはなぜですか。
4. 穴うめ：受粉すると、めしべのもとがふくらんで（　①　）になり、その中に（　②　）ができる。
5. 花粉はどのようにしてめしべまで運ばれますか。2つ書きましょう。
6. 受粉させなかっためばなは、そのあとどうなりますか。

### 解答と解説

1. **受粉**。
2. **イ**。ヘチマはおばなとめばなに分かれています。実になるのは**めばな**のもとのふくらみです（ウは誤り）。
3. （例）**虫などが花粉を運んできて、実験の前に受粉してしまうのを防ぐため。** 受粉した・していないの条件を自分でコントロールするためのくふうです。
4. ①**実** ②**種子**。
5. **虫に運ばれる**、**風に運ばれる**。ヘチマやアサガオはおもに虫、トウモロコシやマツは風で運ばれます。
6. **実にならず、（黄色くなって）かれて落ちる。** 受粉しないと実も種子もできません。

### つまずきやすいポイント

- 「花がさけば必ず実ができる」わけではありません。**受粉があってはじめて実と種子ができます**。
- **実になるのはめしべのもと（めばなのもと）**です。「おばなが実になる」というまちがいが多いので注意。
- ふくろをかぶせる理由を「花を守るため」と答えてしまいがちですが、正しくは**よけいな受粉を防いで条件をそろえるため**です。
- アサガオ（1つの花に両方）とヘチマ（おばな・めばな）のタイプのちがいを、例とセットで覚えましょう。

## 流れる水のはたらきと土地の変化

### この単元で学ぶこと

- 流れる水には、地面をけずる「浸食」、けずった土や石を運ぶ「運搬」、積もらせる「堆積」の3つのはたらきがあることを知る
- 水の流れの速さや量によって、はたらきの大きさが変わることを、モデル実験で調べる
- 川の上流・中流・下流のようす（川はば、流れの速さ、石の形や大きさ）のちがいを理解する
- 大雨のときの川の変化と、災害を防ぐくふう（ダム、堤防、砂防ダムなど）を知る

### 解説

#### 流れる水の3つのはたらき

| はたらき | 内容 | よく起こる場所 |
| --- | --- | --- |
| **浸食（しん食）** | 地面をけずる | 流れが速いところ、かたむきが急なところ |
| **運搬（うんぱん）** | けずった土や石を運ぶ | 流れが速いところ |
| **堆積（たい積）** | 運ばれた土や石を積もらせる | 流れがゆるやかなところ |

土の山にみぞをつけて水を流すモデル実験をすると、次のことがわかります。

- **かたむきが急な所**：流れが速く、みぞが深くけずられる（浸食・運搬が大きい）
- **かたむきがゆるやかな所**：流れがおそく、土が積もる（堆積が大きい）
- **曲がっている所の外側**：流れが速く、けずられる（がけになりやすい）
- **曲がっている所の内側**：流れがおそく、積もる（川原になりやすい）
- **水の量を増やす**と、浸食も運搬も大きくなる

#### 川の上流・中流・下流

| | 上流（山の中） | 中流 | 下流（平地・海の近く） |
| --- | --- | --- | --- |
| 土地のかたむき | 急 | 中くらい | ゆるやか |
| 流れの速さ | 速い | 中くらい | ゆるやか |
| 川はば | せまい | 広くなる | 広い |
| 石のようす | **大きく、角ばっている** | やや小さく丸みがある | **小さく、丸い**。砂やどろも多い |
| 目立つはたらき | 浸食・運搬 | 運搬 | 堆積 |

下流の石が小さくて丸いのは、**運ばれる間に、石どうしがぶつかったり、川底や岸に当たったりして、角が取れ、われて小さくなっていく**からです。

#### 大雨のときの川と防災

台風や大雨のとき、川の水の量が急に増えます。すると流れが速くなり、浸食や運搬のはたらきが**とても大きく**なって、川岸がけずられたり、こう水が起きたりします。長い年月の間には、流れる水のはたらきによって、V字形の谷や扇状地（せんじょうち）など、土地のようすそのものが変えられていきます。

災害を防ぐためのくふうには、次のようなものがあります。

| くふう | はたらき |
| --- | --- |
| ダム | 水をためて、一度に流れる量を調節する |
| 堤防（ていぼう） | 水があふれるのを防ぐ |
| 砂防（さぼう）ダム | 土や石が一気に流れ下るのを防ぐ |
| ブロック（護岸） | 川岸がけずられるのを防ぐ |

### やってみよう（観察・実験）

**土の山に水を流すモデル実験**

- 用意するもの：土（砂まじり）、プランターや衣装ケースなどの入れ物（またはすな場）、ペットボトル（水を流す用）2本、じょうろ、旗になる小さな目印
- 手順
  1. 土で山を作り、上から下までみぞ（川の形）をつける。とちゅうにカーブを作る。
  2. ペットボトル1本分の水を上から静かに流し、けずられる所・積もる所を観察して目印を立てる。
  3. 次に**2本分の水を同時に**流し、けずられ方・運ばれ方がどう変わるかを比べる。
- 結果の見方
  - カーブの外側がけずられ、内側に土が積もるか。
  - 水の量を増やすと、けずれ方が大きくなるか。
  - 「変える条件は水の量だけ。みぞの形や土は同じにする」ことを意識する。
- 安全上の注意：ぬれた地面はすべりやすいので足元に注意。実際の川で観察するときは、**必ず大人と行き、増水時や雨の後は絶対に川に近づかない**。

### 確認問題

1. 流れる水の3つのはたらきの名前を書きましょう。
2. 川が曲がって流れている所で、流れが速く、岸がけずられてがけになりやすいのは、内側・外側のどちらですか。
3. 穴うめ：川の水の量が増えると、浸食と運搬のはたらきは（　　）なる。
4. 下流の川原の石が、上流の石に比べて小さく丸いのはなぜですか。理由を書きましょう。
5. 次のうち、堆積のはたらきが最も目立つ場所はどこですか。
   ア　山の中の急な流れ　　イ　カーブの外側　　ウ　海の近くのゆるやかな流れ
6. 大雨による川の災害を防ぐためのくふうを2つあげ、それぞれのはたらきをかんたんに書きましょう。

### 解答と解説

1. **浸食（けずる）・運搬（運ぶ）・堆積（積もらせる）**。はたらきと意味をセットで覚えましょう。
2. **外側**。外側は流れが速く浸食がさかん、内側は流れがおそく堆積がさかんで川原ができやすくなります。
3. **大きく**。水の量が増えると流れる水の力が強まり、けずる力・運ぶ力が大きくなります。
4. （例）**川を流れ下る間に、石どうしがぶつかり合ったり川底に当たったりして、角が取れたり、われて小さくなったりするから。**
5. **ウ**。流れがゆるやかな所では、運ばれてきた土や砂が積もります。アとイは浸食・運搬が目立つ場所です。
6. （例）**ダム＝水をためて一度に流れる水の量を調節する。堤防＝川の水があふれるのを防ぐ。**（砂防ダム、護岸ブロックなども正解）

### つまずきやすいポイント

- **「内側と外側」を逆に覚えるまちがい**がとても多いです。「外側＝速い＝けずられる」「内側＝おそい＝積もる」とセットで覚えましょう。
- 浸食・運搬・堆積は**3つ同時に起きている**こともあります。「上流は浸食だけ」ではなく、「上流では浸食が**目立つ**」という言い方が正確です。
- 下流の石が丸いのは「もともと丸い石が下流にある」のではなく、**運ばれる間に丸くなった**からです。
- 漢字は「浸食（しん食）」「運搬（運ぱん）」「堆積（たい積）」。教科書ではひらがなまじりで書かれることもあります。

## 物のとけ方

### この単元で学ぶこと

- 物が水にとけた液（水溶液）は、すき通っていて（透明で）、とけた物が液全体に広がっていることを知る
- 物が水にとけても、**全体の重さは変わらない**ことを確かめる
- 物が水にとける量には限りがあり、水の量や温度によって変わることを、食塩とミョウバンで比べて調べる
- ろ過のしかたと、とけた物を取り出す方法（冷やす・水を蒸発させる）を身につける

### 解説

#### 水溶液とは

食塩を水に入れてかき混ぜると、つぶが見えなくなり、すき通った液になります。このように、物が水にとけた液を**水溶液**といいます。

水溶液の特ちょう

- **透明**である（色がついていても、すき通っていれば水溶液）
- とけた物は、目に見えなくても**液全体に均一に広がっている**
- 時間がたっても、とけた物は下にたまらない

#### とけても重さは変わらない

水50gに食塩5gをとかして全体の重さをはかると、**55g**になります。
とけて見えなくなっても、食塩がなくなったわけではないのです。

> 水の重さ ＋ とかした物の重さ ＝ 水溶液の重さ

#### とける量には限りがある――食塩とミョウバンの比べっこ

水50mLに、食塩とミョウバンをすり切り1ぱいずつ加えてかき混ぜ、何ばいまでとけるかを調べます（変える条件は「とかす物の種類」や「水の温度」、変えない条件は「水の量」など）。

**水の温度ととける量（水50mLにとける量のめやす）**

| 水の温度 | 食塩 | ミョウバン |
| --- | --- | --- |
| 10度 | 約18g | 約4g |
| 30度 | 約18g | 約8g |
| 60度 | 約19g | 約29g |

この表から、大切なことが2つわかります。

1. **食塩は、水の温度を上げても、とける量がほとんど変わらない**
2. **ミョウバンは、水の温度を上げると、とける量が大きく増える**

また、**水の量を2倍にすると、とける物の量も2倍**になります。

#### ろ過のしかた

とけ残りがある液から、とけ残りだけを取り除くには**ろ過**をします。

- ろ紙を4つ折りにして開き、ろうとにぴったりつけ、水でしめらせる
- 液は**ガラスぼうに伝わらせて**静かに注ぐ
- ろうとの先の**とがった方をビーカーの内側のかべにつける**

ろ過した液（ろ液）は透明ですが、**とけている物はろ紙を通りぬけている**ので、ろ液の中にはまだ物がとけています。

#### とけた物を取り出す（再結晶）

ろ液から、とけている物を取り出すには2つの方法があります。

| 方法 | ミョウバン | 食塩 |
| --- | --- | --- |
| **液を冷やす** | とける量が大きく減るので、**つぶ（結晶）がたくさん出てくる** | とける量がほとんど変わらないので、**ほとんど出てこない** |
| **水を蒸発させる** | 出てくる | **出てくる**（食塩はこの方法が有効） |

出てきたつぶは、決まった形をしています。食塩の結晶は**さいころのような立方体**、ミョウバンの結晶は**正八面体（ピラミッドを2つ合わせたような形）**です。

### やってみよう（観察・実験）

**ミョウバンの水溶液を冷やして結晶を取り出す**

- 用意するもの：ミョウバン、食塩、ビーカー、メスシリンダー、ガラスぼう、ろうと・ろ紙、温度計、湯（60度くらい）、氷水、保護めがね
- 手順
  1. 60度くらいの湯50mLにミョウバンをとけるだけとかす。（食塩でも同じものを作る）
  2. とけ残りがあればろ過する。
  3. ろ液の入ったビーカーを氷水につけて冷やし、ようすを観察する。
  4. ミョウバンと食塩で、出てくるつぶの量を比べる。
- 結果の見方：ミョウバンはつぶがたくさん出てくるが、食塩はほとんど出てこない。→「温度によるとける量の変化のちがい」で説明できるか考える。
- 安全上の注意：**保護めがねをかける**。湯でやけどをしないように注意する。実験でとかした物は絶対に口に入れない。ガラス器具はていねいにあつかう。

### 確認問題

1. 水100gに食塩20gを完全にとかしました。できた食塩水の重さは何gですか。
2. 水溶液の特ちょうとして正しいものをすべて選びましょう。
   ア　すき通っている。
   イ　とけた物は液の下の方にたまっている。
   ウ　とけた物は液全体に広がっている。
3. 穴うめ：水の温度を上げると、ミョウバンのとける量は大きく（　①　）が、食塩のとける量は（　②　）。
4. 60度の水にミョウバンをとけるだけとかした液を冷やすと、つぶが出てくるのはなぜですか。
5. 食塩水から食塩を取り出すには、「冷やす」よりも「水を蒸発させる」方がよいのはなぜですか。
6. ろ過のしかたとして正しくないものはどれですか。
   ア　液はガラスぼうに伝わらせて注ぐ。
   イ　ろうとの先はビーカーの内側のかべにつける。
   ウ　ろ紙にはとけている物が残るので、ろ液は真水になる。

### 解答と解説

1. **120g**。とけて見えなくなっても物はなくなりません。水100g＋食塩20g＝120gです。
2. **アとウ**。水溶液は透明で、とけた物は液全体に均一に広がります。イのように下にたまることはありません。
3. ①**増える** ②**ほとんど変わらない**。この2つの物のちがいが、この単元のいちばん大事なポイントです。
4. （例）**水の温度が下がると、ミョウバンがとけることのできる量が減るので、とけきれなくなった分がつぶ（結晶）になって出てくるから。**
5. （例）**食塩は水の温度が変わってもとける量がほとんど変わらないため、冷やしてもほとんど出てこないから。** 水を蒸発させれば、とけていられる水が減るので食塩が出てきます。
6. **ウ**。ろ紙に残るのは「とけ残り（つぶ）」だけです。とけている物はろ紙を通りぬけるので、ろ液は真水ではなく、まだ物がとけた水溶液です。

### つまずきやすいポイント

- **「とけたら物はなくなった」という誤解**が最大のつまずきです。見えなくなっても物はあり、重さも残っています。
- **「ろ過すればとけている物も取り除ける」という誤解**。ろ過で取り除けるのは、とけていないつぶだけです。
- 食塩とミョウバンの性質を逆に覚えないこと。**「温度で大きく変わるのはミョウバン」**です。
- 「とける」（水溶液になる）と「とける」（氷が水になる＝融ける）は別のこと。食塩が水にとけるのは、氷がとけるのとはちがう現象です。
- メスシリンダーの目もりは、**液面のへこんだ下の面**を真横から読みます。

## ふりこの動き

### この単元で学ぶこと

- ふりこが1往復する時間（周期）が、何によって決まるのかを、条件を制御した実験で調べる
- ふりこの1往復する時間は、**ふりこの長さ**によって変わり、**おもりの重さ**や**ふれはば**では変わらないことを理解する
- 実験の結果を平均を使って処理する方法を身につける

### 解説

#### ふりことは

糸におもりをつるして、左右にふれるようにしたものを**ふりこ**といいます。

- **ふりこの長さ**：糸をつるした点から、**おもりの中心**までの長さ
- **1往復**：ふれ始めた位置から、反対側までいって、また元の位置にもどるまで
- **ふれはば**：ふりこがふれる角度の大きさ

#### 何が1往復の時間を決めるのか

「1往復する時間」に関係しそうな条件は3つ考えられます。**1つずつ変えて**調べます。

| 実験 | 変える条件 | 変えない条件 | 結果 |
| --- | --- | --- | --- |
| 1 | ふりこの長さ（25cm・50cm・100cm） | おもりの重さ、ふれはば | **長いほど1往復の時間が長くなる** |
| 2 | おもりの重さ（10g・20g・30g） | ふりこの長さ、ふれはば | **変わらない** |
| 3 | ふれはば（10度・20度・30度） | ふりこの長さ、おもりの重さ | **変わらない** |

結論：**ふりこの1往復する時間は、ふりこの長さだけで決まる。**

- 長さのめやす：長さ25cmで1往復約1秒、長さ100cmで1往復約2秒
- おもりを重くしても、ふれはばを大きくしても、1往復の時間は変わりません

#### 正確にはかるくふう――10往復と平均

1往復は短くて、ストップウォッチではかるとどうしても誤差が出ます。そこで、

1. **10往復する時間**をはかり、10でわって1往復の時間を求める
2. 同じ測定を**3回**行い、**平均**を出す

> 平均 ＝ 合計 ÷ 回数

例：10往復の時間が 14.2秒、14.0秒、14.4秒 のとき
合計 42.6秒 ÷ 3回 ＝ 14.2秒（10往復の平均）→ 1往復は **約1.4秒**

#### おもりのつけ方の注意

おもりを増やすとき、おもりを**下にたてにつなげてつるすと、おもりの中心の位置が下がって「ふりこの長さ」が変わってしまいます**。おもりは1か所にまとめてつけ、長さが変わらないようにします。

ふりこの性質は、メトロノームやふりこ時計に利用されています。メトロノームはおもりの位置を上げ下げして（＝ふりこの長さを変えて）、テンポを変えています。

### やってみよう（観察・実験）

**ふりこの長さと1往復の時間の関係を調べる**

- 用意するもの：糸、おもり（同じ重さのもの）、スタンド、ストップウォッチ、分度器、記録用紙
- 手順
  1. 長さ25cmのふりこを作り、ふれはば（例：20度）を決めて手をはなす。
  2. 10往復の時間をはかる。これを3回くり返し、平均を出して10でわり、1往復の時間を求める。
  3. 長さを50cm、100cmに変えて同じことをする。**おもりの重さとふれはばは変えない。**
  4. 結果を表にまとめ、長さと1往復の時間の関係を考える。
- 結果の見方：長さが長いほど1往復の時間が長くなっていれば、「1往復の時間は長さによって変わる」といえる。
- 安全上の注意：ふれているおもりに顔を近づけない。スタンドがたおれないようにしっかり固定する。

### 確認問題

1. ふりこの1往復する時間を変えることができる条件はどれですか。
   ア　おもりの重さ　　イ　ふれはば　　ウ　ふりこの長さ
2. 穴うめ：ふりこの長さとは、糸をつるした点から（　　）までの長さのことである。
3. 10往復の時間を3回はかったら、15.0秒、14.7秒、15.3秒でした。1往復の時間は約何秒ですか。
4. 1往復の時間を、1往復ではなく10往復ではかって10でわるのはなぜですか。
5. ふりこ時計が少しおくれて（1往復の時間が長すぎて）います。正しく直すには、ふりこの長さをどうすればよいですか。理由も書きましょう。
6. ○か×か：ふれはばを大きくすると、おもりが動く道のりは長くなるが、1往復の時間は変わらない。

### 解答と解説

1. **ウ**。1往復の時間はふりこの長さだけで決まります。おもりの重さやふれはばを変えても、1往復の時間は変わりません。
2. **おもりの中心**。糸の長さではないことに注意しましょう。
3. 合計45.0秒÷3回＝15.0秒（10往復の平均）。15.0÷10＝**約1.5秒**。
4. （例）**1往復の時間は短く、はかり方のわずかなずれ（誤差）のえいきょうが大きいから。** 10往復ではかって10でわると、誤差のえいきょうを小さくできます。
5. **ふりこを短くする。** 1往復の時間はふりこの長さで決まり、短くすると1往復の時間が短くなるので、時計の進み方が正しくなります。
6. **○**。ふれはばが大きいと道のりは長くなりますが、そのぶん速く動くので、1往復にかかる時間は変わりません。

### つまずきやすいポイント

- **「おもりが重いほど速くふれる（または、ゆっくりふれる）」という誤解**がとても多いです。実験で確かめると、重さを変えても1往復の時間は変わりません。
- **「ふれはばが大きいと1往復の時間も長くなる」という誤解**も要注意。道のりは長くなりますが、速さも変わるので時間は同じです。
- ふりこの長さは**「おもりの中心まで」**。おもりをたてにつなげてつるすと、知らないうちに長さを変えてしまいます。
- 平均を出すとき、**明らかにおかしい値（はかりまちがい）が混ざっていないか**を先に確かめる習慣をつけましょう。

## 電磁石の性質

### この単元で学ぶこと

- コイルに鉄心を入れて電流を流すと、鉄心が磁石のはたらきをする（電磁石）ことを知る
- 電磁石は、電流が流れているときだけ磁石になり、電流の向きを変えると極が入れかわることを理解する
- 電磁石を強くする方法（電流を大きくする・コイルの巻数を増やす）を、条件を制御した実験で調べる
- 電磁石が身のまわりのどんな物に使われているかを知る

### 解説

#### 電磁石とは

導線を同じ向きに何回も巻いたものを**コイル**といいます。コイルの中に**鉄心**（鉄くぎなど）を入れて電流を流すと、鉄心が磁石のはたらきをするようになります。これが**電磁石**です。

#### 電磁石とふつうの磁石（永久磁石）のちがい

| | 電磁石 | ふつうの磁石 |
| --- | --- | --- |
| 磁石のはたらき | **電流が流れているときだけ** | いつも |
| N極とS極 | ある。**電流の向きを反対にすると極が入れかわる** | ある。極は変わらない |
| 強さ | **変えられる**（電流・巻数） | 変えられない |

電磁石にもN極とS極があることは、方位磁針を近づけると確かめられます。かん電池の＋と−を入れかえて電流の向きを反対にすると、方位磁針の針のふれる向きが反対になり、**極が入れかわった**ことがわかります。

#### 電磁石を強くするには

電磁石の強さは、引きつけられる**ゼムクリップ（鉄）の数**で比べます。ここでも条件の制御が大切です。

| 実験 | 変える条件 | 変えない条件 | 結果 |
| --- | --- | --- | --- |
| 1 | 電流の大きさ（かん電池1個→2個の直列つなぎ） | コイルの巻数、導線の長さ、鉄心 | **電流が大きいほど強くなる**（クリップが多くつく） |
| 2 | コイルの巻数（100回→200回） | 電流の大きさ、導線の**全体の長さ**、鉄心 | **巻数が多いほど強くなる** |

結論：**電磁石は、電流を大きくするほど、また、コイルの巻数を多くするほど強くなる。**

注意すべき条件制御のポイント

- 巻数を比べるとき、**導線の全体の長さは同じ**にします（あまった導線は切らずに束ねておく）。導線の長さがちがうと、流れる電流の大きさが変わってしまうからです。
- 電流の大きさは**電流計**（または簡易検流計）ではかります。電流の単位は**アンペア（A）**です。

#### 電流計の使い方

- 電流計は回路に**直列**につなぐ
- ＋たんしはかん電池の＋極側に、−たんしは**まず一番大きい5A**につなぎ、針のふれが小さければ500mA、50mAへつなぎかえる
- 電流計だけをかん電池に直接つないではいけない（大きすぎる電流が流れてこわれる）

#### 電磁石が使われているもの

モーター（せん風機、電気自動車）、スピーカー、鉄を運ぶクレーン（リサイクル工場）、ベルなど。クレーンは「電流を切ると鉄をはなせる」という電磁石ならではの性質を利用しています。

### やってみよう（観察・実験）

**電磁石の強さとコイルの巻数の関係を調べる**

- 用意するもの：エナメル線（同じ長さのもの）、鉄くぎ（またはボルト）、かん電池、電池ボックス、スイッチ、電流計（簡易検流計）、ゼムクリップ、紙やすり
- 手順
  1. エナメル線の両はしを紙やすりでけずってエナメルをはがす。
  2. 同じ長さのエナメル線で、100回巻きと200回巻きのコイルを作る（あまった線は切らずに束ねる）。
  3. 回路にスイッチと電流計を直列につなぎ、電流の大きさが同じことを確かめる。
  4. それぞれの電磁石でゼムクリップを引きつけ、ついた数を3回ずつ数えて平均を出す。
- 結果の見方：200回巻きの方がクリップが多くつけば、「巻数が多いほど電磁石は強い」といえる。
- 安全上の注意：**電流を流したままにしない**（導線やかん電池が熱くなる）。スイッチを入れるのは調べるときだけ。かん電池をショート（＋極と−極を導線だけで直接つなぐこと）させない。

### 確認問題

1. 電磁石がふつうの磁石とちがう点を2つ書きましょう。
2. 電磁石を強くする方法を2つ書きましょう。
3. かん電池の＋極と−極を入れかえて電流の向きを反対にすると、電磁石はどうなりますか。
   ア　磁石のはたらきがなくなる　　イ　N極とS極が入れかわる　　ウ　変わらない
4. 穴うめ：電流の大きさは（　①　）計ではかり、単位は（　②　）である。
5. コイルの巻数と電磁石の強さの関係を調べるとき、100回巻きと200回巻きのコイルで「導線全体の長さ」を同じにするのはなぜですか。
6. 鉄を運ぶクレーンに、ふつうの磁石ではなく電磁石が使われているのはなぜですか。

### 解答と解説

1. （例）**電流が流れているときだけ磁石になる。電流の向きを変えると極が入れかわる。**（強さを変えられる、も正解）
2. **電流を大きくする**（かん電池を直列に増やすなど）、**コイルの巻数を多くする**。
3. **イ**。電流の向きが反対になると、N極とS極が入れかわります。磁石のはたらき自体はなくなりません（なくなるのは電流を切ったとき）。
4. ①**電流** ②**アンペア（A）**。電流計は回路に直列につなぎます。
5. （例）**導線の長さがちがうと、回路に流れる電流の大きさが変わってしまい、強さのちがいが「巻数のせい」なのか「電流のせい」なのかわからなくなるから。** 変える条件を巻数だけにするためです。
6. （例）**電流を流すと鉄を引きつけ、電流を切ると鉄をはなすことができるから。** ふつうの磁石では、くっついた鉄を自由にはなせません。

### つまずきやすいポイント

- **エナメル線のはしのエナメルをはがし忘れる**と電流が流れません。実験がうまくいかないときは、まずここを確かめましょう。
- 「電磁石には極がない」という誤解に注意。**電磁石にもN極・S極があり**、電流の向きで入れかわります。
- 巻数の実験で導線を切って長さを変えてしまうミスが多発します。**あまった導線は束ねて、全体の長さをそろえる**こと。
- かん電池2個を**並列**につないでも、電流の大きさはかん電池1個のときとほとんど変わらないので、電磁石は強くなりません。強くするのは**直列**つなぎです。
- 電流計の−たんしは**大きい5Aたんしから**。小さいたんしから使うと針がふり切れてこわれることがあります。

## 人のたんじょう

### この単元で学ぶこと

- 人の生命が、女性の体内でつくられる卵（卵子）と男性の体内でつくられる精子が結びつく「受精」から始まることを知る
- 受精卵が母親の子宮の中で、約38週かけて育ってから生まれることを理解する
- 子宮の中のつくり（羊水・たいばん・へそのお）と、それぞれのはたらきを知る
- メダカのたんじょうと比べて、共通点とちがいを考える

### 解説

#### 生命のはじまり

人の生命は、女性の体内でつくられる**卵（卵子）**と、男性の体内でつくられる**精子**が結びつくこと、すなわち**受精**から始まります。受精した卵を**受精卵**といいます。

- 受精卵の大きさは約**0.1mm**。針の先でついたあなくらいの大きさです。
- 受精卵は、母親の**子宮**の中で育ちます。

#### 子宮の中のつくりとはたらき

| 名前 | はたらき |
| --- | --- |
| **子宮** | 赤ちゃん（胎児）が育つ部屋 |
| **羊水** | 子宮の中を満たす液体。外からのしょうげきから胎児を守る |
| **たいばん** | 母親の血液から届いた**養分など**を胎児にわたし、胎児からの**いらなくなった物**を母親側に返す、受けわたしの場所 |
| **へそのお** | 胎児とたいばんをつなぐ管。養分などの通り道 |

生まれたあと、へそのおのあとが「おへそ」になります。

#### 子宮の中での育ち（めやす）

| 受精からの週数 | 身長・体重のめやす | ようす |
| --- | --- | --- |
| 約4週 | 約0.5cm | 心ぞうが動き始める |
| 約8週 | 約3cm | 目や耳ができ、手足の形がはっきりしてくる |
| 約16週 | 約25cm・約300g | 男女の区別がつく。体を動かし始める |
| 約24週 | 約35cm・約1000g | 体を回転させ、よく動く |
| 約32週 | 約40cm・約1800g | かみの毛やつめがのびてくる |
| **約38週** | **約50cm・約3000g** | **子宮の中で回れないくらいに育ち、生まれる** |

受精から誕生まで、およそ**38週（約266日）**。生まれるときの身長は約50cm、体重は約3000gがめやすです。

#### メダカと人を比べてみると

| | メダカ | 人 |
| --- | --- | --- |
| 生命のはじまり | **受精**（共通） | **受精**（共通） |
| 育つ場所 | たまごの中（親の体の外） | **母親の子宮の中** |
| 養分 | **たまごの中の養分** | **母親からたいばん・へそのおを通してもらう** |
| 受精から誕生まで | 約11日 | 約38週 |

「受精から生命が始まる」ことは共通ですが、**育つ場所**と**養分のもらい方**が大きくちがいます。

### やってみよう（観察・調べ学習）

**赤ちゃんの育ちを調べてモデルで実感する**

- 用意するもの：図書室の本や信頼できる資料、記録カード、（モデル体験用に）約3kgのおもり（2Lペットボトル1本半など）とリュック
- 手順
  1. 受精卵の大きさ（約0.1mm）を、紙に針であけたあなと見比べてみる。
  2. 週数ごとの身長・体重を調べて表やグラフにまとめる。
  3. 約3kgのおもりを入れたリュックを体の前にかかえて、生まれる直前の赤ちゃんの重さを体感してみる。
  4. たいばん・へそのお・羊水のはたらきを図にまとめ、メダカとのちがいを表で整理する。
  5. 家の人に、自分が生まれたときの話を聞いてみるのもよい。
- 結果の見方：約38週の間に、0.1mmの受精卵が約50cm・約3000gまで育つという変化の大きさに注目する。
- 注意：人のたんじょうは一人ひとりちがいがあるもの。表の数字はあくまで「めやす」であり、ちがいがあるのは当たり前だということも大切な学びです。

### 確認問題

1. 人の生命は、何と何が結びつくことから始まりますか。また、そのことを何といいますか。
2. 受精卵はどこで育ちますか。
3. 穴うめ：胎児は（　①　）という管を通して、（　②　）で母親から養分などを受け取っている。
4. 羊水にはどんなはたらきがありますか。
5. 人の受精から誕生までの期間はおよそどれですか。
   ア　約11日　　イ　約38日　　ウ　約38週
6. メダカと人のたんじょうについて、「共通していること」と「ちがうこと」を1つずつ書きましょう。

### 解答と解説

1. **卵（卵子）と精子**が結びつくことから始まり、これを**受精**といいます。
2. **母親の子宮の中**。約38週かけて育ちます。
3. ①**へそのお** ②**たいばん**。たいばんで養分などを受け取り、いらなくなった物を返します。
4. （例）**外からのしょうげきをやわらげて、胎児を守るはたらき。**
5. **ウ（約38週）**。アの約11日はメダカのふ化までの日数です。
6. （例）共通していること：**どちらも受精から生命が始まる。** ちがうこと：**メダカはたまごの中の養分で育つが、人は母親からたいばんとへそのおを通して養分をもらって育つ。**（育つ場所のちがいでも正解）

### つまずきやすいポイント

- **「約38週」と「約38日」のまちがい**が非常に多いです。メダカ（約11日）と単位ごとセットで区別しましょう。
- **たいばんとへそのおの役わりの混同**に注意。へそのおは「通り道（管）」、たいばんは「受けわたしの場所」です。
- 「胎児は子宮の中でお乳を飲んでいる」「口から養分をとっている」という誤解があります。養分は**へそのおを通して**受け取ります。
- 胎児は羊水の中にいて、**肺で呼吸はしていません**。酸素もたいばん・へそのおを通して受け取っています。
- メダカとの比かくでは、「たまご（卵）から始まる」という共通点と、「養分のもらい方・育つ場所」というちがいを、両方セットで説明できるようにしましょう。

## 5年生のまとめ――「条件をそろえて調べる」ということ

この1年でみなさんは、たくさんの実験をしてきました。ふり返ってみると、どの実験にも共通するやり方がありました。

| 単元 | 変えた条件の例 | そろえた条件の例 |
| --- | --- | --- |
| 発芽 | 水のあるなし | 空気、温度、種子の種類 |
| 受粉 | 受粉させるかどうか | ふくろをかぶせること、花の種類 |
| 流れる水 | 水の量 | みぞの形、土の種類 |
| 物のとけ方 | 水の温度 | 水の量、とかす物 |
| ふりこ | ふりこの長さ | おもりの重さ、ふれはば |
| 電磁石 | コイルの巻数 | 電流の大きさ、導線の長さ |

**調べたい条件を1つだけ変えて、ほかはすべてそろえる。**
この考え方は、6年生の理科でも、中学校でも、そして大人になってからも使う、科学のいちばん大切な道具です。
