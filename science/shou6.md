# 小学6年 理科

小学6年生の理科では、これまでに学んできた「観察」や「実験」の力をいかして、目に見えないもののしくみを考えていきます。空気の中の気体、体の中を流れる血液、月と太陽の位置、大地の中のようす、水よう液の正体、電気の変身など、身近だけれどふしぎな世界をくわしく調べていきましょう。

実験には火や薬品を使うものもあります。安全に気をつけて、「予想 → 実験 → 結果 → 考察（考えること）」の順に進めるのが理科のコツです。

## 物の燃え方と空気

### この単元で学ぶこと

- 物が燃え続けるためには、新しい空気（酸素）が必要であることを理解する
- 空気は「ちっ素」「酸素」「二酸化炭素」などの気体が混ざってできていることを知る
- 物が燃えると空気中の酸素の一部が使われ、二酸化炭素ができることを実験で確かめる
- 石灰水や気体検知管を使って、気体の変化を調べる方法を身につける

### 解説

ろうそくや木、紙が燃えるとき、じつは空気の力を借りています。びんの中でろうそくを燃やすと、しばらくして火が消えてしまいます。これは、燃えるのに必要な**酸素**が使われて少なくなるからです。ふたをせずに、下から新しい空気が入るようにしておくと、ろうそくは燃え続けます。つまり、**物が燃え続けるには、たえず新しい空気（酸素）が入れかわる必要がある**のです。

空気は1種類の気体ではありません。次のような気体が混ざってできています。

| 気体の名前 | 空気中の体積のおよその割合 | とくちょう |
|---|---|---|
| ちっ素 | 約78% | 物を燃やすはたらきはない |
| 酸素 | 約21% | 物を燃やすはたらきがある |
| 二酸化炭素 | 約0.04% | 物を燃やすはたらきはない。石灰水を白くにごらせる |

ここで大切なのは、**「物を燃やすはたらき」があるのは酸素だけ**だということです。ちっ素や二酸化炭素の中では、ろうそくの火はすぐに消えてしまいます。

では、物が燃えると空気はどう変わるのでしょうか。ろうそくを燃やした後の空気を**気体検知管**で調べると、次のような変化がわかります。

| | 燃やす前 | 燃やした後 |
|---|---|---|
| 酸素 | 約21% | 約17%くらいに減る |
| 二酸化炭素 | 約0.04% | 増える（数%） |

ここで間ちがえやすいのは、「酸素は全部なくなる」と思ってしまうことです。実際には、**酸素は約21%から約17%くらいまで減るだけで、全部はなくなりません**。酸素がある程度より少なくなると、火は消えてしまうのです。

また、物が燃えると**二酸化炭素ができます**。これを確かめるには**石灰水**を使います。石灰水は二酸化炭素にふれると白くにごる性質があるので、燃やした後のびんに石灰水を入れてふると白くにごり、二酸化炭素ができたことがわかります。

### やってみよう（観察・実験）

**実験1：びんの中でろうそくを燃やす**

用意するもの
- ガラスのびん（広口びん）
- ろうそく、ねんど（ろうそくを立てる）
- ふた、ふたのないびん
- マッチまたはライター（先生と一緒に）

手順
1. ろうそくに火をつけ、びんをかぶせる。
2. ふたをしたびんと、下にすき間をあけたびんで、燃え方をくらべる。

結果の見方
- ふたをして密ぺいすると、しばらくして火が消える。
- 下から空気が入るようにすると、燃え続ける。

**実験2：燃やした後の空気を調べる**

用意するもの
- 集気びん、石灰水、気体検知管、ろうそく

手順
1. 集気びんの中でろうそくを燃やし、火が消えたら石灰水を少し入れてふる。
2. 燃やす前と後の空気を気体検知管で調べる。

結果の見方
- 石灰水が白くにごる → 二酸化炭素ができた。
- 酸素は21%から17%くらいに減っている。

安全上の注意
- 火を使うので、燃えやすい物を近くに置かない。
- 気体検知管の先はガラスで切れやすいので、ゴムのカバーを使い、素手でおらない。
- 換気（かんき）をよくする。

### 確認問題

1. 空気の中でいちばん多くふくまれている気体は何ですか。次から選びなさい。
   ア ちっ素　イ 酸素　ウ 二酸化炭素

2. 物を燃やすはたらきがある気体は（　　）である。

3. びんの中でろうそくを燃やすと、やがて火が消えます。その理由を書きなさい。

4. 物が燃えた後にできて、石灰水を白くにごらせる気体は何ですか。

5. 物が燃えると、酸素はどうなりますか。正しいものを選びなさい。
   ア 全部なくなる　イ 一部が使われて減る　ウ 増える

6. ろうそくを燃やし続けるには、どうすればよいですか。かんたんに書きなさい。

### 解答と解説

1. **ア ちっ素**。空気の約78%はちっ素です。酸素は約21%です。
2. **酸素**。ちっ素や二酸化炭素には物を燃やすはたらきはありません。
3. **例：燃えるのに必要な酸素が使われて少なくなり、新しい空気が入ってこないから。** 酸素がなくなるのではなく、「減って」火が消えます。
4. **二酸化炭素**。石灰水を白くにごらせる性質があります。
5. **イ 一部が使われて減る**。約21%から約17%くらいまで減りますが、全部はなくなりません。
6. **例：たえず新しい空気（酸素）が入れかわるようにする。** 下にすき間を作るなど、空気が流れる工夫をします。

### つまずきやすいポイント

- 「酸素が全部なくなるから火が消える」と考えがちですが、実際は**一部が減るだけ**です。酸素の割合が下がると燃えられなくなります。
- 「二酸化炭素が物を燃やす」と思う人がいますが、逆です。二酸化炭素の中では火は消えます。
- 石灰水が「白くにごる」のは二酸化炭素の合図。「にごらない=二酸化炭素がない」と読み取ります。

## 人や動物の体（呼吸・消化・血液の循環）

### この単元で学ぶこと

- 呼吸によって、酸素を取り入れ二酸化炭素を出していることを理解する
- 食べ物が消化・吸収されるしくみと、消化にかかわる臓器のはたらきを知る
- 心臓のはたらきと、血液が体をめぐって養分や酸素を運ぶことを理解する
- 各臓器が協力して生命を保っていることに気づく

### 解説

**1. 呼吸（こきゅう）**

わたしたちは息を吸ったり吐いたりしています。吸う空気（すう空気）と吐く空気（はく空気）を石灰水や気体検知管でくらべると、次のようにちがいます。

| | 酸素 | 二酸化炭素 |
|---|---|---|
| 吸う空気 | 多い（約21%） | 少ない（約0.04%） |
| 吐く空気 | 減る（約17%くらい） | 増える（数%） |

吐いた空気を石灰水に通すと白くにごります。これは、体の中で**酸素を取り入れ、二酸化炭素を出している**からです。空気を取り入れる臓器を**肺（はい）**といいます。

**2. 消化（しょうか）**

食べ物は、口・胃・小腸・大腸などを通るあいだに、体に取り入れやすい養分に変えられます。これを**消化**といいます。口から始まって食べ物が通る1本の長い管を**消化管**といいます。

| 臓器 | おもなはたらき |
|---|---|
| 口 | かんで細かくし、だ液とまぜる |
| 胃 | 胃液で食べ物をさらに消化する |
| 小腸 | 養分と水分を体に吸収する |
| 大腸 | おもに水分を吸収する |
| かん臓 | 養分をたくわえたり作りかえたりする |

だ液には、でんぷんを別のもの（糖など）に変えるはたらきがあります。ごはんをよくかむとあまく感じるのは、だ液がでんぷんを変えているからです。これは**ヨウ素液**を使って確かめられます（でんぷんがあると青むらさき色になる）。

**3. 血液の循環（じゅんかん）**

**心臓**は、ちぢんだりゆるんだりして血液を全身に送り出すポンプのはたらきをしています。この動きが**はく動**で、手首などで感じるのが**みゃく**です。

血液は、肺で取り入れた**酸素**や、小腸で吸収した**養分**を全身の体のすみずみまで運びます。そして、体でできた**二酸化炭素**などの不要なものを受け取って運びます。不要なものは、じん臓でこしとられて**尿（にょう）**になったり、肺から二酸化炭素として出されたりします。

### やってみよう（観察・実験）

**実験1：だ液のはたらきを調べる**

用意するもの
- でんぷんのりを溶かした水、だ液、試験管、ヨウ素液、40℃くらいのお湯

手順
1. でんぷん液を2本の試験管に入れる。
2. 一方にはだ液を、もう一方には水を入れる。
3. 40℃くらいのお湯につけてしばらく待ち、ヨウ素液を数てき入れる。

結果の見方
- 水を入れたほう → 青むらさき色になる（でんぷんが残っている）。
- だ液を入れたほう → 色があまり変わらない（でんぷんが変えられた）。

**観察2：はく動を数える**

手順
- 手首や首にそっと指をあて、1分間のみゃくの回数を数える。運動の前後でくらべる。

結果の見方
- 運動後はみゃくが速くなる（体が多くの酸素を必要とするため）。

安全上の注意
- お湯はやけどに注意。ヨウ素液は服や手につくと色がつくので気をつける。

### 確認問題

1. 空気を取り入れて、酸素と二酸化炭素を交かんする臓器を何といいますか。

2. 吐く空気には、吸う空気にくらべて（　　）が多くふくまれている。

3. でんぷんがあることを調べる薬品は（　　）で、でんぷんがあると（　　）色になる。

4. 血液を全身に送り出すポンプのはたらきをする臓器を何といいますか。

5. 食べ物の養分をおもに吸収する臓器を、次から選びなさい。
   ア 胃　イ 小腸　ウ 大腸

6. 血液は体の中で何を運んでいますか。2つ書きなさい。

### 解答と解説

1. **肺（はい）**。息を吸うと肺で酸素を取り入れます。
2. **二酸化炭素**（および水蒸気）。体で酸素が使われて二酸化炭素ができるためです。
3. **ヨウ素液**、**青むらさき**色。でんぷんの目印になります。
4. **心臓**。ちぢんだりゆるんだりして血液を送り出します。
5. **イ 小腸**。小腸で養分と水分を吸収します。
6. **例：酸素、養分（二酸化炭素や不要なものでも正解）**。行きは酸素や養分、帰りは二酸化炭素などを運びます。

### つまずきやすいポイント

- 「吐く空気には酸素がまったくない」と思いがちですが、吐く空気にも酸素はまだ残っています（約17%くらい）。二酸化炭素が「増える」のがポイントです。
- 消化は「食べ物をとかす」だけでなく、「体に取り入れやすい養分に変える」ことです。
- 心臓が動かす血液は、行き（動脈血）と帰り（静脈血）で運ぶものがちがいます。片道だけでなく、体をぐるぐる回っています。

## 植物の養分と水の通り道（光合成・蒸散）

### この単元で学ぶこと

- 植物が日光を受けて、葉ででんぷんを作っていることを理解する
- 根から吸い上げた水が、くき・葉を通って全体に運ばれることを知る
- 葉から水が水蒸気となって出ていく「蒸散（じょうさん）」を理解する
- ヨウ素液を使ってでんぷんができたことを確かめる

### 解説

**1. 葉ででんぷんを作る（光合成）**

植物は、葉に日光が当たると**でんぷん**（養分）を作ります。これを確かめるには、**ヨウ素液**を使います。日光に当てた葉と当てていない葉をくらべると、日光に当てた葉のほうがヨウ素液で**青むらさき色**に変わり、でんぷんができていることがわかります。

日光に当てないようにアルミニウムはくで葉をおおっておくと、その部分にはでんぷんができません。このことから、**でんぷんを作るには日光が必要**だとわかります。

**2. 水の通り道**

植物は、**根**から水を吸い上げます。吸い上げた水は、**くき**の中の細い管を通って、**葉**まで運ばれます。ホウセンカのくきを、色をつけた水（食紅など）にさしておくと、水が通った管が赤く染まって見えます。

- 根 → 水を吸い上げる
- くき → 水の通り道（細い管）
- 葉 → 水を使う・水を出す

**3. 蒸散（じょうさん）**

葉まで運ばれた水は、葉から**水蒸気**となって空気中に出ていきます。これを**蒸散**といいます。葉にはとても小さな穴（気こう）があり、そこから水蒸気が出ます。ふくろを葉にかぶせておくと、内側に水てきがついて、水が出ていることがわかります。蒸散のおかげで、根から新しい水がどんどん吸い上げられます。

### やってみよう（観察・実験）

**実験1：葉にでんぷんができるか調べる（ヨウ素液）**

用意するもの
- ジャガイモなどの葉、アルミニウムはく、湯、ヨウ素液、エタノール（先生と）

手順
1. 前日の夕方、葉の一部をアルミニウムはくでおおう。
2. 翌日よく日光に当てたあと、葉をつみとる。
3. お湯でやわらかくし、あたためたエタノールで緑色をぬく。
4. 水で洗い、ヨウ素液につけて色を見る。

結果の見方
- 日光に当てた部分 → 青むらさき色（でんぷんあり）。
- おおった部分 → 色があまり変わらない（でんぷんなし）。

**実験2：蒸散を調べる**

手順
- 葉のついた枝と、葉を取った枝にそれぞれふくろをかぶせ、しばらく置く。

結果の見方
- 葉のあるほうのふくろの内側に、より多くの水てきがつく。

安全上の注意
- エタノールは燃えやすいので直接火にかけず、お湯であたためる。ヨウ素液は色がつくので注意。

### 確認問題

1. 植物が葉で養分（でんぷん）を作るために必要なものは何ですか。

2. でんぷんができたかどうかを調べるには（　　）を使い、青むらさき色になれば（　　）ができている。

3. 根から吸い上げた水は、おもにどこを通って葉まで運ばれますか。

4. 葉から水が水蒸気となって出ていくことを何といいますか。

5. 次のうち正しいものを選びなさい。
   ア 日光が当たらなくても葉ででんぷんは作られる。
   イ 日光が当たった葉ででんぷんが作られる。

6. 蒸散には植物にとってどんなよいことがありますか。かんたんに書きなさい。

### 解答と解説

1. **日光（と水、二酸化炭素）**。とくに日光が大切です。
2. **ヨウ素液**、**でんぷん**。ヨウ素液はでんぷんの目印です。
3. **くき（の中の細い管）**。色水で染めると通り道が見えます。
4. **蒸散**。葉の小さな穴（気こう）から水蒸気が出ます。
5. **イ**。日光が当たった葉ででんぷんが作られます。
6. **例：水が水蒸気となって出ることで、根から新しい水を吸い上げやすくなる。**

### つまずきやすいポイント

- ヨウ素液の実験では、前もって葉の中のでんぷんの条件をそろえるため、前日から準備することがあります。「日光の有無だけ」をくらべるのがねらいです。
- 「水はくきの外側を流れる」と思いがちですが、くきの**中の細い管**を通ります。
- 蒸散で出ていくのは「水」ではなく**水蒸気**（気体）です。

## 生き物と環境（食物連鎖、水・空気とのかかわり）

### この単元で学ぶこと

- 生き物どうしが「食べる・食べられる」の関係でつながっていることを理解する
- 食べ物のもとをたどると、日光を受けて養分を作る植物にいきつくことを知る
- 生き物が水や空気（酸素・二酸化炭素）とかかわって生きていることを理解する
- 自然環境を守ることの大切さに気づく

### 解説

**1. 食物連鎖（しょくもつれんさ）**

生き物は、ほかの生き物を食べたり食べられたりしながら生きています。この「食べる・食べられる」のつながりを**食物連鎖**といいます。

例：　植物 → バッタ → カエル → ヘビ → ワシ

このつながりのもとをたどっていくと、いつも**植物**にいきつきます。植物は日光を受けて自分で養分（でんぷん）を作ることができます。動物は自分で養分を作れないので、植物やほかの動物を食べて養分を得ています。**すべての動物の食べ物のもとは、植物が作った養分**なのです。

**2. 水とのかかわり**

生き物の体の多くは水でできています。人も動物も植物も、水がなければ生きていけません。水は、飲んだり吸ったりして体に取り入れられ、体の中を通って、また外に出ていきます。地球上の水は、海・川・雨・水蒸気などとして、ぐるぐるとめぐっています。

**3. 空気とのかかわり**

生き物は空気ともかかわっています。

| はたらき | 取り入れる | 出す |
|---|---|---|
| 動物・植物の呼吸 | 酸素 | 二酸化炭素 |
| 植物が日光を受けて養分を作るとき | 二酸化炭素 | 酸素 |

植物は、日光が当たっているときは養分を作りながら**酸素を出しています**。動物も植物も呼吸で酸素を使い二酸化炭素を出しますが、植物のはたらきによって空気中の酸素が保たれています。このように、生き物と空気はおたがいに深くかかわり合っています。

### やってみよう（観察・実験）

**観察1：食べ物のつながりを調べる**

手順
- 校庭や池の生き物を観察し、「何を食べているか」「何に食べられるか」を調べて、矢印でつなぐ図を作る。

結果の見方
- 矢印は「食べられるもの → 食べるもの」の向きに書く。もとをたどると植物にいきつく。

**観察2：植物が出す気体を調べる（発てん的）**

手順
- 水草（オオカナダモなど）を水に入れ、日光に当てて出るあわを観察する。

結果の見方
- 日光に当てると水草から小さなあわ（酸素をふくむ）が出る。

安全上の注意
- 池や川で観察するときは、水辺に近づきすぎない。生き物はやさしくあつかい、できるだけもとにもどす。

### 確認問題

1. 生き物どうしの「食べる・食べられる」のつながりを何といいますか。

2. 食物連鎖のもとをたどると、必ず何にいきつきますか。

3. 動物が自分で養分を作れないのに対して、植物は（　　）を受けて自分で養分を作ることができる。

4. 植物が日光を受けて養分を作るとき、空気中に出す気体は何ですか。

5. 次の食物連鎖の（　　）にあてはまる生き物を1つ考えて書きなさい。
   植物 →（　　）→ カエル → ヘビ

6. 生き物にとって水はどんな点で大切ですか。かんたんに書きなさい。

### 解答と解説

1. **食物連鎖**。矢印で表すことが多いです。
2. **植物**。植物だけが日光を受けて自分で養分を作れます。
3. **日光**。植物は光合成で養分を作ります。
4. **酸素**。呼吸では二酸化炭素を出しますが、養分を作るときは酸素を出します。
5. **例：バッタ（コオロギ、イナゴなど植物を食べる小さな動物ならよい）**。
6. **例：体の多くが水でできており、水がないと生きられないから。**

### つまずきやすいポイント

- 食物連鎖の矢印の向きをまちがえやすいです。矢印は「**食べられるもの → 食べるもの**」の向き（養分が移る向き）に書きます。
- 「植物は酸素だけ出す」と思いがちですが、植物も呼吸をしていて二酸化炭素も出します。日光が当たっているときは、養分作りで出す酸素のほうが多くなります。

## 月と太陽（月の形の見え方）

### この単元で学ぶこと

- 月の形が日によって変わって見えることを理解する
- 月の形の見え方は、太陽と月の位置関係によって決まることを知る
- 月は太陽の光を反射して光っていることを理解する
- 新月から満月まで、約15日でうつり変わることを知る

### 解説

月は毎日少しずつ形が変わって見えます。三日月、半月、満月……などとよびます。じつは、**月そのものの形が変わっているのではありません**。月は**太陽の光を反射して光っている**ので、太陽の光が当たっている側だけが明るく見えます。そのため、地球から見ると、明るい部分の見え方が日によって変わるのです。

月の形の見え方は、**太陽と月と地球の位置関係**で決まります。

| 月の見え方 | 太陽と月の位置の関係 |
|---|---|
| 新月（見えない） | 月が太陽と同じ方向にあり、光る面が地球から見えない |
| 三日月・半月 | 太陽と月が少しずつはなれていく |
| 満月 | 月が太陽と反対の方向にあり、光る面全体が見える |

新月（月が見えない日）から始まって、少しずつ満ちていき、**約15日で満月**になります。その後は少しずつ欠けていき、また新月にもどります。この満ち欠けは、月が地球のまわりを回っている（公転している）ために起こります。

太陽と月をくらべると、次のようなちがいがあります。

| | 太陽 | 月 |
|---|---|---|
| 光り方 | 自分で光っている | 太陽の光を反射して光る |
| 表面 | とても高温でまぶしい | 岩石でできていて、クレーターがある |

### やってみよう（観察・実験）

**観察1：月の形の変化を記録する**

用意するもの
- 記録用紙、方位じしん、時計

手順
1. 数日おきに同じ時こくに月を観察し、形と見える方位を記録する。
2. 太陽がしずむころの月の位置もあわせて記録する。

結果の見方
- 日がたつと、月の形が三日月 → 半月 → 満月と変わっていく。
- 満月は太陽がしずむころに東の空からのぼる。

**実験2：ボールで月の見え方を再現する**

用意するもの
- 白いボール（月に見立てる）、電灯（太陽に見立てる）

手順
- 電灯を太陽、自分を地球、ボールを月として、ボールを持って回りながら、明るい部分の見え方を観察する。

結果の見方
- ボールと電灯の位置関係で、明るく見える部分が変わり、月の満ち欠けと同じになる。

安全上の注意
- 太陽を直接、目で見てはいけない（目を痛める）。月の観察は暗いので、足もとや車に注意する。

### 確認問題

1. 月が光って見えるのはなぜですか。

2. 月の形の見え方は、何と何の位置関係で決まりますか。

3. 太陽と反対の方向に月があるとき、月はどんな形に見えますか。

4. 新月から満月になるまで、およそ何日かかりますか。

5. 次のうち正しいものを選びなさい。
   ア 月は自分で光を出している。
   イ 月は太陽の光を反射して光っている。

6. 太陽と月のちがいを1つ書きなさい。

### 解答と解説

1. **例：太陽の光を反射しているから。** 月は自分では光りません。
2. **太陽と月（と地球）の位置関係**。位置関係で明るい部分の見え方が変わります。
3. **満月**。光る面全体が地球のほうを向くためです。
4. **約15日**。新月から半月をへて満月になります。
5. **イ**。月は太陽の光を反射して光っています。
6. **例：太陽は自分で光っているが、月は太陽の光を反射して光っている。**

### つまずきやすいポイント

- 「月の形そのものが変わる」と思いがちですが、変わるのは**見える明るい部分**です。月自体はボールのような形のままです。
- 「雲で月がかくれて欠ける」わけではありません。満ち欠けは太陽と月の位置関係で起こります。
- 太陽を直接見ると目を痛めます。太陽の観察は絶対に肉眼で行いません。

## 大地のつくりと変化（地層、火山、地震）

### この単元で学ぶこと

- 地層は、れき・砂・どろ・火山灰などが積み重なってできることを理解する
- 地層は、流れる水のはたらきや火山のふん火によってできることを知る
- 地層の中から化石が見つかることや、地層からその土地の過去がわかることを理解する
- 火山のふん火や地震によって、大地が変化することを知る

### 解説

**1. 地層（ちそう）のつくり**

がけなどで、しま模様に見える土や石の重なりを**地層**といいます。地層は、**れき（小石）・砂・どろ・火山灰**などが層になって積み重なってできています。層のつぶは、下の層ほど古く、上の層ほど新しいのがふつうです。

**2. 地層のでき方**

地層のできかたには、大きく2つあります。

| でき方 | ようす | つぶのとくちょう |
|---|---|---|
| 流れる水のはたらき | 川が運んだれき・砂・どろが海や湖の底に積もる | つぶが丸みをおびている |
| 火山のはたらき | ふん火で出た火山灰などが積もる | つぶが角ばっている、ガラスのようなものがある |

流れる水で運ばれたつぶは、水の中で転がるうちに角がとれて**丸く**なります。れき（重い）は河口の近くに、どろ（軽い）は遠くまで運ばれて積もるので、つぶの大きさで層が分かれます。

一方、火山灰の層のつぶは**角ばって**いて、丸くありません。これで、その層が水のはたらきか火山のはたらきかを見分けられます。

**3. 化石（かせき）**

地層の中からは、大昔の生き物の**化石**が見つかることがあります。化石からは、その地層ができた当時のようす（海だったか、あたたかかったかなど）がわかります。

**4. 大地の変化：火山と地震**

- **火山のふん火**…地下のマグマがふき出し、よう岩や火山灰を出す。新しい土地ができたり、地形が変わったりする。
- **地震**…大地が急にずれて動くこと。土地が持ち上がったり、われめ（断層）ができたりする。

火山も地震も、わたしたちの生活にめぐみ（温泉など）ももたらしますが、災害を起こすこともあります。ふだんから備えることが大切です。

### やってみよう（観察・実験）

**実験1：流れる水で地層を作る**

用意するもの
- れき・砂・どろを混ぜた土、とい（水を流す道）、水、とうめいな入れ物

手順
1. 混ぜた土を、水とともに何回かにわけて水そうに流しこむ。
2. 積もったようすを横から観察する。

結果の見方
- 重いれきや砂が下に、細かいどろが上に積もり、層に分かれる。

**観察2：がけや地層の見学**

手順
- がけの地層を安全な場所から観察し、層の色・つぶの大きさ・角ばり方を記録する。ルーペでつぶを見る。

結果の見方
- 丸いつぶ → 流れる水のはたらき。角ばったつぶ・ガラス質 → 火山灰の層。

安全上の注意
- がけの下や上は落石・くずれの危険があるので近づかない。かならず大人といっしょに、安全な距離から観察する。

### 確認問題

1. れき・砂・どろ・火山灰などが積み重なって、しま模様になっているものを何といいますか。

2. 地層のでき方を2つ書きなさい。

3. 流れる水で運ばれたつぶと、火山灰のつぶでは、形にどんなちがいがありますか。

4. 地層の中から見つかる、大昔の生き物のあとを何といいますか。

5. 次のうち、火山のはたらきでできた層のつぶの特ちょうを選びなさい。
   ア 丸みをおびている　イ 角ばっている

6. 地震が起こると、大地にはどんな変化が起こることがありますか。1つ書きなさい。

### 解答と解説

1. **地層**。がけなどでしま模様に見えます。
2. **流れる水のはたらき**と**火山のはたらき（火山灰が積もる）**。
3. **例：流れる水で運ばれたつぶは丸みをおびていて、火山灰のつぶは角ばっている。** 運ばれる間に角がとれるかどうかのちがいです。
4. **化石**。当時のようすを知る手がかりになります。
5. **イ 角ばっている**。火山灰は運ばれずに積もるので角ばっています。
6. **例：土地が持ち上がったり、われめ（断層）ができたりする。**

### つまずきやすいポイント

- 「地層はいつも水の中でできる」と思いがちですが、**火山灰が積もってできる地層**もあります。つぶの形で見分けます。
- 「下の層ほど新しい」は逆です。ふつう**下の層ほど古い**です。
- れき・砂・どろのちがいは色ではなく**つぶの大きさ**（れき>砂>どろ）で決まります。

## てこのはたらき

### この単元で学ぶこと

- てこには「支点・力点・作用点」の3つの点があることを理解する
- 支点から力点までのきょりを長くすると、小さな力で物を動かせることを知る
- てこがつり合うときのきまり（力の大きさ×支点からのきょり）を理解する
- てこのはたらきを利用した身近な道具に気づく

### 解説

**1. てこの3つの点**

棒を1点で支えて、物を持ち上げたり動かしたりする道具を**てこ**といいます。てこには次の3つの点があります。

| 点の名前 | 意味 |
|---|---|
| 支点（してん） | てこを支える動かない点 |
| 力点（りきてん） | 力を加える点 |
| 作用点（さようてん） | 物に力がはたらく点 |

**2. 小さい力で動かすには**

てこを使って、できるだけ小さい力で物を持ち上げるには、次のようにします。

- **支点から力点までのきょりを長くする**（力点を支点から遠くする）
- **支点から作用点までのきょりを短くする**（作用点を支点に近づける）

こうすると、小さな力で重い物を持ち上げられます。反対に、力点を支点に近づけると、大きな力が必要になります。

**3. 実験用てこのつり合い**

**実験用てこ**（左右に目もりのある棒に、おもりをつるせる装置）を使うと、つり合いのきまりがわかります。てこが水平につり合うとき、次の関係が成り立ちます。

> **（左のうでの）力の大きさ × 支点からのきょり ＝ （右のうでの）力の大きさ × 支点からのきょり**

つまり「おもりの重さ × 支点からのきょり」が左右で等しくなると、てこは水平につり合います。

例：左のうでの目もり2に、おもり3個（3の重さ）をつるすと、左は 3 × 2 ＝ 6。右のうでで6にするには、目もり3におもり2個（2 × 3 ＝ 6）や、目もり6におもり1個（1 × 6 ＝ 6）などにすればつり合います。

**4. 身近な道具**

てこのはたらきを使った道具には、はさみ、ペンチ、くぎぬき（バール）、せんぬき、ピンセットなどがあります。それぞれ支点・力点・作用点の位置がちがいます。

### やってみよう（観察・実験）

**実験1：実験用てこでつり合いを調べる**

用意するもの
- 実験用てこ、同じ重さのおもり

手順
1. 左のうでのある目もりに、おもりを何個かつるす。
2. 右のうでで、てこが水平につり合う場所とおもりの数を見つける。
3. 「おもりの数 × 目もり」を左右で計算し、くらべる。

結果の見方
- 左右で「おもりの数 × 目もり」が等しいとき、水平につり合う。

**実験2：小さい力で持ち上げる**

手順
- 棒と支点を用意し、力点の位置を変えて、同じおもりを持ち上げる手ごたえをくらべる。

結果の見方
- 力点が支点から遠いほど、軽い手ごたえで持ち上がる。

安全上の注意
- おもりが落ちて足に当たらないよう注意する。てこの棒で人をつつかない。

### 確認問題

1. てこの、力を加える点を何といいますか。

2. できるだけ小さい力で物を持ち上げるには、支点から力点までのきょりを（長く・短く）すればよい。

3. 実験用てこがつり合うときのきまりを、式のことばで書きなさい。

4. 左のうでの目もり4に、おもり3個をつるしました。右のうでの目もり6につるすおもりは何個でつり合いますか。

5. 次のうち、てこを利用した道具を1つ選びなさい。
   ア はさみ　イ 温度計　ウ ものさし

6. 力点を支点に近づけると、必要な力はどうなりますか。

### 解答と解説

1. **力点**。支える点は支点、物にはたらく点は作用点です。
2. **長く**。力点を支点から遠くすると小さな力ですみます。
3. **例：力の大きさ（おもりの重さ）×支点からのきょり が、左右で等しくなる。**
4. **2個**。左は 3個 × 4 ＝ 12。右は 目もり6なので、12 ÷ 6 ＝ 2個。
5. **ア はさみ**。2枚の刃が支点で回るてこの道具です。
6. **例：大きくなる（大きな力が必要になる）。**

### つまずきやすいポイント

- 支点・力点・作用点の位置は道具によってちがいます。「いつも真ん中が支点」ではありません。
- つり合いの計算は「重さ＋きょり」ではなく「重さ**×**きょり」です。かけ算だと覚えましょう。
- 「力点を遠くすると力が大きくなる」と逆に覚えがちです。**遠いほど楽（小さい力）**です。

## 水よう液の性質（酸性・中性・アルカリ性、金属を溶かす水溶液）

### この単元で学ぶこと

- 水よう液には酸性・中性・アルカリ性の3つのなかまがあることを理解する
- リトマス紙を使って、水よう液のなかまを見分ける方法を身につける
- 水よう液には、気体が溶けているものや、金属を溶かすものがあることを知る
- 実験を安全に行う方法を身につける

### 解説

**1. 水よう液のなかま分け**

水にいろいろなものが溶けたものを**水よう液**といいます。水よう液は、性質によって次の3つになかま分けできます。

| なかま | 例 | 味・とくちょう（味は確かめない） |
|---|---|---|
| 酸性 | 塩酸、炭酸水、酢、レモン水 | すっぱいなかま |
| 中性 | 食塩水、さとう水、水 | どちらでもない |
| アルカリ性 | 石灰水、アンモニア水、水酸化ナトリウム水よう液 | にがい・ぬるぬるするなかま |

※ 味やにおいで確かめるのは危険なので、薬品では絶対にしません。

**2. リトマス紙による見分け方**

**リトマス紙**を使うと、水よう液のなかまがわかります。赤色リトマス紙と青色リトマス紙の2種類があります。

| なかま | 青色リトマス紙 | 赤色リトマス紙 |
|---|---|---|
| 酸性 | 赤色に変わる | 変わらない |
| 中性 | 変わらない | 変わらない |
| アルカリ性 | 変わらない | 青色に変わる |

覚え方：**酸性は青→赤**（青いリトマス紙が赤くなる）、**アルカリ性は赤→青**（赤いリトマス紙が青くなる）、**中性はどちらも変わらない**。

**3. 気体が溶けている水よう液**

水よう液の中には、**気体が溶けている**ものがあります。

- **炭酸水**…二酸化炭素が溶けている。あわが出て、石灰水を白くにごらせる。
- **塩酸**…塩化水素という気体が溶けている。つんとしたにおいがする。

炭酸水を熱すると、溶けていた二酸化炭素が出ていって、あとに何も残りません。一方、食塩水を熱して水を蒸発させると、あとに**食塩（固体）**が残ります。溶けているものが気体か固体かで、蒸発させたときのようすがちがいます。

**4. 金属を溶かす水よう液**

**塩酸**には、金属を溶かすはたらきがあります。

- 塩酸に**アルミニウム**を入れると、あわ（気体）を出しながら溶ける。
- 塩酸に**鉄（スチールウールなど）**を入れると、あわを出しながら溶ける。

金属が溶けた塩酸の液を蒸発させると、もとの金属とはちがう別のもの（固体）が出てきます。これは、金属が塩酸と反応して、別のものに変わったからです。つまり、**溶けてなくなったのではなく、別のものに変化した**のです。

### やってみよう（観察・実験）

**実験1：リトマス紙で見分ける**

用意するもの
- 塩酸、食塩水、石灰水などの水よう液、赤色・青色リトマス紙、ガラス棒

手順
1. ガラス棒に水よう液をつけ、リトマス紙にちょんとつける（じか手でさわらない）。
2. 色の変化を記録し、なかま分けする。

結果の見方
- 青→赤なら酸性、赤→青ならアルカリ性、どちらも変わらなければ中性。

**実験2：塩酸に金属を入れる**

用意するもの
- うすい塩酸、アルミニウムはく、鉄（スチールウール）、試験管

手順
1. 試験管にうすい塩酸を入れ、アルミニウムや鉄を少し入れる。
2. あわの出方と溶け方を観察する。

結果の見方
- あわ（気体）を出しながら溶ける。液を蒸発させると別の固体が残る。

安全上の注意
- 塩酸などの薬品は目や皮ふにつけない。**保護めがね**をかける。
- 直接手でさわらない、においを強くかがない、換気をよくする。
- こぼしたらすぐ大人に知らせる。

### 確認問題

1. 水よう液は、性質によって酸性・中性・（　　）の3つになかま分けできる。

2. 青色リトマス紙を赤色に変える水よう液は、何性ですか。

3. 赤色リトマス紙を青色に変える水よう液は、何性ですか。

4. 炭酸水に溶けている気体は何ですか。

5. うすい塩酸にアルミニウムを入れると、どうなりますか。
   ア 何も起こらない　イ あわを出しながら溶ける　ウ かたくなる

6. 食塩水を熱して水を蒸発させると、あとに何が残りますか。

### 解答と解説

1. **アルカリ性**。石灰水やアンモニア水などが仲間です。
2. **酸性**。「酸性は青→赤」と覚えます。
3. **アルカリ性**。「アルカリ性は赤→青」と覚えます。
4. **二酸化炭素**。石灰水を白くにごらせます。
5. **イ あわを出しながら溶ける**。塩酸は金属を溶かします。
6. **食塩（固体）**。食塩は固体なので、水が蒸発しても残ります（炭酸水なら何も残りません）。

### つまずきやすいポイント

- リトマス紙の色変化はまぎらわしいので、**「酸性=青が赤」「アルカリ性=赤が青」「中性=変わらない」**とセットで覚えます。「変わらない=中性」も大事な情報です。
- 金属が塩酸に溶けると「消えてなくなった」と思いがちですが、**別のものに変化した**だけです。蒸発させると別の固体が出ます。
- 薬品は味やにおいで確かめてはいけません。安全第一です。

## 電気の利用（発電・蓄電、電気の変換、プログラミング）

### この単元で学ぶこと

- 電気は作り出す（発電する）ことも、ためる（蓄電する）こともできることを理解する
- 電気は、光・音・熱・運動などにすがたを変えて利用できることを知る
- 手回し発電機やコンデンサー、光電池のはたらきを理解する
- 電気を効率よく使う工夫や、プログラミングによる制御を知る

### 解説

**1. 電気を作る（発電）**

電気は、いろいろな方法で作り出すことができます。

- **手回し発電機**…ハンドルを回すと電気ができる。速く回すと強い電気ができ、豆電球が明るくなる。回す向きを変えると、電流の向きも変わる。
- **光電池（太陽光発電）**…光を当てると電気ができる。明るいほど強い電気ができる。

**2. 電気をためる（蓄電）**

**コンデンサー**は、電気をためておく部品です。手回し発電機でコンデンサーに電気をためてから、発電機をはずして豆電球やLED（発光ダイオード）につなぐと、ためた電気で明かりがつきます。ためた電気を使い切ると、明かりは消えます。

**3. 電気のすがたを変える（変換）**

電気は、いろいろなすがたに変えて利用できます。

| 変わるすがた | 使う道具の例 |
|---|---|
| 光 | 豆電球、LED（発光ダイオード） |
| 音 | ブザー、スピーカー |
| 熱 | 電熱線、電気ストーブ、アイロン |
| 運動 | モーター、せん風機 |

**LED**は、豆電球よりも少ない電気で光ります。同じ量の電気をためたコンデンサーにつなぐと、LEDのほうが豆電球より**長く光り続けます**。だから、少ない電気で明るくできるLEDは、省エネにやくだちます。

**4. 電気を効率よく使う工夫とプログラミング**

必要なときだけ電気を使うようにすると、むだが減ります。たとえば、人がいるときだけ明かりをつける照明があります。これは、**人を感知するセンサー**と、**コンピューターへの命令（プログラム）**によって動いています。

「暗くなったら」「人が近づいたら」といった条件のときだけ電気を流すように、コンピューターに順番に命令を出すことを**プログラミング**といいます。プログラミングを使うと、電気をむだなく、便利に使うことができます。

例：「もし 暗く なったら → 明かりをつける」「もし 人が いなくなったら → 明かりを消す」

### やってみよう（観察・実験）

**実験1：手回し発電機で豆電球をつける**

用意するもの
- 手回し発電機、豆電球、LED、モーター、ブザー

手順
1. 発電機に豆電球をつなぎ、ハンドルを回して明かりをつける。
2. 回す速さを変えて明るさをくらべる。回す向きも変えてみる。

結果の見方
- 速く回すほど明るい。向きを変えると電流の向きが変わる（モーターは逆回転）。

**実験2：コンデンサーに電気をためてくらべる**

手順
1. 手回し発電機でコンデンサーに電気をためる（同じ回数だけ回す）。
2. 豆電球とLEDにそれぞれつなぎ、光っている時間をくらべる。

結果の見方
- 同じ電気の量でも、LEDのほうが長く光る（少ない電気で光るため）。

安全上の注意
- コンデンサーには+と−の向きがある。まちがえてつながない。
- 発電機を必要以上に速く回しすぎない。電熱線は熱くなるのでさわらない。

### 確認問題

1. ハンドルを回すと電気を作れる道具を何といいますか。

2. 電気をためておくことができる部品を何といいますか。

3. 電気を「光」に変えて利用する道具を1つ書きなさい。

4. 同じ量の電気で、豆電球とLEDでは、どちらが長く光りますか。

5. 光を当てると電気を作ることができる装置を何といいますか。

6. 「暗くなったら明かりをつける」というように、条件を決めてコンピューターに命令することを何といいますか。

### 解答と解説

1. **手回し発電機**。回す速さで電気の強さが変わります。
2. **コンデンサー**。ためた電気であとから明かりをつけられます。
3. **例：豆電球（LEDでも正解）**。ほかに音ならブザー、熱なら電熱線です。
4. **LED**。少ない電気で光るので長く光ります。省エネにやくだちます。
5. **光電池（太陽電池）**。明るいほど強い電気ができます。
6. **プログラミング**。電気をむだなく使う工夫に使われます。

### つまずきやすいポイント

- コンデンサーは電気を「作る」のではなく「**ためる**」部品です。作るのは発電機や光電池です。
- LEDは向き（+と−）を正しくつながないと光りません。豆電球は向きが関係ありませんが、LEDは向きが大切です。
- 「電気は使うと消えてなくなる」だけでなく、光・音・熱・運動など**別のすがたに変わっている**と考えると理解しやすいです。

## まとめ

小学6年の理科では、目に見えない気体や体の中のはたらき、遠くの月、大地の中、水よう液の正体、電気の変身など、はば広いテーマを学びました。どの単元にも共通するのは、「**予想を立てて、実験や観察で確かめ、結果から考える**」という理科の進め方です。

- 物の燃え方…酸素が使われ、二酸化炭素ができる（酸素は全部はなくならない）
- 体のはたらき…呼吸・消化・血液の循環が協力して命を保つ
- 植物…日光で養分（でんぷん）を作り、水は根→くき→葉と通って蒸散する
- 生き物と環境…食べる・食べられるでつながり、水や空気とかかわり合う
- 月と太陽…月は太陽の光を反射し、位置関係で満ち欠けする
- 大地…地層は水や火山のはたらきででき、地震や火山で変化する
- てこ…力の大きさ×支点からのきょりでつり合う
- 水よう液…酸性・中性・アルカリ性に分けられ、塩酸は金属を溶かす
- 電気…作る・ためる・変える・かしこく使う（プログラミング）

安全に気をつけながら、身のまわりのふしぎをこれからも楽しく調べていきましょう。
