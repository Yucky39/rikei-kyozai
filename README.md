# 理数教材ライブラリ

小学校1年生から高校3年生までの**算数・数学**と**理科**を体系的に学べる教材集です。

📖 **Webサイト版（数式がきれいに表示されます）**: https://yucky39.github.io/rikei-kyozai/

## 教材の構成

すべての教材は次の構成で書かれています。

- **この単元で学ぶこと** — 学習目標
- **解説** — 具体例を使った丁寧な説明
- **例題** — 途中式・考え方つきの模範解答
- **練習問題** — 自分で解く問題（基本〜入試レベル）
- **解答と解説** — 全問題の詳しい解説
- **つまずきやすいポイント** — よくある間違いと対策
